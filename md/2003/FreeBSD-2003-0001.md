# Buffer overflows in libmcrypt
libmcrypt does incomplete input validation, leading to several buffer
overflows. Additionally, a memory leak is present. Both of these
problems may be exploited in a denial-of-service attack.


## Affected packages
 - **libmcrypt**
    - **Fixed:** 2.5.6
