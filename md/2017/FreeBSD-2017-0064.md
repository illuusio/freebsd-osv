# kdepimlibs -- directory traversal on KTNEF
Albert Aastals Cid reports:

> A directory traversal issue was found in KTNEF which can be exploited
> by tricking a user into opening a malicious winmail.dat file. The
> issue allows to write files with the permission of the user opening
> the winmail.dat file during extraction.


## Affected packages
 - **kdepimlibs**
    - **Fixed:** 4.14.10_7
