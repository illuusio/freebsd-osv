# libraw -- buffer overflow
libraw developers report:

> LibRaw before 0.18.4 has a heap-based Buffer Overflow in the
> processCanonCameraInfo function via a crafted file.


## Affected packages
 - **libraw**
    - **Fixed:** 0.18.4
