# PHP -- denial of service attack
The PHP project reports:

> The PHP development team announces the immediate availability of PHP
> 5.6.32. This is a security release. Several security bugs were fixed
> in this release. All PHP 5.6 users are encouraged to upgrade to this
> version.

> The PHP development team announces the immediate availability of PHP
> 7.0.25. This is a security release. Several security bugs were fixed
> in this release. All PHP 7.0 users are encouraged to upgrade to this
> version.

> The PHP development team announces the immediate availability of PHP
> 7.1.11. This is a bugfix release, with several bug fixes included. All
> PHP 7.1 users are encouraged to upgrade to this version.
