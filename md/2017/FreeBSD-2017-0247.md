# FFmpeg -- multiple vulnerabilities
FFmpeg security reports:

> Multiple vulnerabilities have been fixed in FFmpeg 3.3.4. Please refer
> to the CVE list for details.


## Affected packages
 - **ffmpeg**
    - **Fixed:** 3.3.4
 - **mythtv**
    - **Fixed:** 29.1,1
 - **mythtv-frontend**
    - **Fixed:** 29.1,1
