# xorg-server -- multiple vulnerabilities
Adam Jackson reports:

> One regression fix since 1.19.4 (mea culpa), and fixes for CVEs
> 2017-12176 through 2017-12187.


## Affected packages
 - **xephyr**
    - **Fixed:** 1.18.4_5,1
 - **xorg-dmx**
    - **Fixed:** 1.18.4_5,1
 - **xorg-nestserver**
    - **Fixed:** 1.19.1_2,2
 - **xorg-server**
    - **Fixed:** 1.18.4_5,1
 - **xorg-vfbserver**
    - **Fixed:** 1.19.1_2,1
 - **xwayland**
    - **Fixed:** 1.19.1_2
