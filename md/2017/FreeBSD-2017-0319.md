# rubygem-passenger -- arbitrary file read vulnerability
Phusion reports:

> The cPanel Security Team discovered a vulnerability in Passenger that
> allows users to list the contents of arbitrary files on the system.
> CVE-2017-16355 has been assigned to this issue.


## Affected packages
 - **rubygem-passenger**
    - **Introduced:** 5.0.10
    - **Fixed:** 5.1.11
