# rubygems -- deserialization vulnerability
oss-security mailing list:

> There is a possible unsafe object desrialization vulnerability in
> RubyGems. It is possible for YAML deserialization of gem
> specifications to bypass class white lists. Specially crafted
> serialized objects can possibly be used to escalate to remote code
> execution.


## Affected packages
 - **ruby22-gems**
    - **Fixed:** 2.6.14
 - **ruby23-gems**
    - **Fixed:** 2.6.14
 - **ruby24-gems**
    - **Fixed:** 2.6.14
