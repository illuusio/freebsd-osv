# fbsdmon -- information disclosure vulnerability
Alan Somers reports:

> The web site used by this port, http://fbsdmon.org, has been taken
> over by cybersquatters. That means that users are sending their system
> info to an unknown party.


## Affected packages
 - **fbsdmon**
