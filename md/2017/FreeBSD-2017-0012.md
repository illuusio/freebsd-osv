# lynx -- multiple vulnerabilities
Oracle reports:

> Lynx is vulnerable to POODLE by still supporting vulnerable version of
> SSL. Lynx is also vulnerable to URL attacks by incorrectly parsing
> hostnames ending with an \'?\'.


## Affected packages
 - **lynx**
    - **Fixed:** 2.8.8.2_5,1
