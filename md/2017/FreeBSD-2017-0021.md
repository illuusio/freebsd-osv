# Ansible -- Command execution on Ansible controller from host
Computest reports:

> Computest found and exploited several issues that allow a compromised
> host to execute commands on the Ansible controller and thus gain
> access to other hosts controlled by that controller.


## Affected packages
 - **ansible**
    - **Introduced:** 1.9.6_1,1
    - **Fixed:** 2.2.0.0_2
