# konversation -- crash in IRC message parsing
KDE reports:

> Konversation has support for colors in IRC messages. Any malicious
> user connected to the same IRC network can send a carefully crafted
> message that will crash the Konversation user client.


## Affected packages
 - **konversation**
    - **Fixed:** 1.7.3
