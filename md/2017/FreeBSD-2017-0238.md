# node -- access to unintended files
node developers report:

> Node.js 8.5.0 before 8.6.0 allows remote attackers to access
> unintended files, because a change to \"..\" handling was incompatible
> with the pathname validation used by unspecified community modules.


## Affected packages
 - **node**
    - **Introduced:** 8.5.0
    - **Fixed:** 8.6.0
