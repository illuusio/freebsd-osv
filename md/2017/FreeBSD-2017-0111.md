# freetype2 -- buffer overflows
Werner Lemberg reports:

> CVE-2017-8105, CVE-2017-8287: Older FreeType versions have
> out-of-bounds writes caused by heap-based buffer overflows related to
> Type 1 fonts.


## Affected packages
 - **freetype2**
    - **Fixed:** 2.8
