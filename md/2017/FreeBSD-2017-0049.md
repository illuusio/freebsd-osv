# diffoscope -- arbitrary file write
Ximin Luo reports:

> \[v67\] introduced a security hole where diffoscope may write to
> arbitrary locations on disk depending on the contents of an untrusted
> archive.


## Affected packages
 - **py34-diffoscope**
    - **Introduced:** 67
    - **Fixed:** 76
 - **py35-diffoscope**
    - **Introduced:** 67
    - **Fixed:** 76
 - **py36-diffoscope**
    - **Introduced:** 67
    - **Fixed:** 76
