# libraw -- denial of service and remote code execution
libraw developers report:

> A Stack-based Buffer Overflow was discovered in xtrans_interpolate in
> internal/dcraw_common.cpp in LibRaw before 0.18.3. It could allow a
> remote denial of service or code execution attack.


## Affected packages
 - **libraw**
    - **Fixed:** 0.18.3
