# Flash Player -- multiple vulnerabilities
Adobe reports:

> - These updates resolve a buffer overflow vulnerability that could
>   lead to code execution (CVE-2017-2997).
> - These updates resolve memory corruption vulnerabilities that could
>   lead to code execution (CVE-2017-2998, CVE-2017-2999).
> - These updates resolve a random number generator vulnerability used
>   for constant blinding that could lead to information disclosure
>   (CVE-2017-3000).
> - These updates resolve use-after-free vulnerabilities that could lead
>   to code execution (CVE-2017-3001, CVE-2017-3002, CVE-2017-3003).


## Affected packages
 - **linux-flashplayer**
    - **Fixed:** 25.0.0.127
