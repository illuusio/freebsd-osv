# jabberd -- authentication bypass vulnerability
SecurityFocus reports:

> JabberD is prone to an authentication-bypass vulnerability. An
> attacker can exploit this issue to bypass the authentication mechanism
> and perform unauthorized actions. This may lead to further attacks.


## Affected packages
 - **jabberd**
    - **Fixed:** 2.6.1
