# evince and atril -- command injection vulnerability in CBT handler
GNOME reports:

> The comic book backend in evince 3.24.0 (and earlier) is vulnerable to
> a command injection bug that can be used to execute arbitrary commands
> when a CBT file is opened.
>
> The same vulnerability affects atril, the Evince fork.


## Affected packages
 - **evince**
    - **Fixed:** 3.24.0
 - **evince-lite**
    - **Fixed:** 3.24.0
 - **atril**
    - **Fixed:** 1.18.1
    - **Introduced:** 1.19.0
    - **Fixed:** 1.19.1
 - **atril-lite**
    - **Fixed:** 1.18.1
    - **Introduced:** 1.19.0
    - **Fixed:** 1.19.1
