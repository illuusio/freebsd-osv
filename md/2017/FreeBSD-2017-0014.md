# libvncserver -- multiple buffer overflows
libvnc server reports:

> Two unrelated buffer overflows can be used by a malicious server to
> overwrite parts of the heap and crash the client (or possibly execute
> arbitrary code).


## Affected packages
 - **libvncserver**
    - **Fixed:** 0.9.11
