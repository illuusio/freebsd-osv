# rubygem-geminabox -- XSS vulnerabilities
NVD reports:

> Stored cross-site scripting (XSS) vulnerability in \"geminabox\" (Gem
> in a Box) before 0.13.10 allows attackers to inject arbitrary web
> script via the \"homepage\" value of a \".gemspec\" file, related to
> views/gem.erb and views/index.erb.


## Affected packages
 - **rubygem-geminabox**
    - **Fixed:** 0.13.10
