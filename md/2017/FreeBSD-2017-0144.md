# tor -- security regression
The Tor Project reports:

> Tor 0.3.0.9 fixes a path selection bug that would allow a client to
> use a guard that was in the same network family as a chosen exit
> relay. This is a security regression; all clients running earlier
> versions of 0.3.0.x or 0.3.1.x should upgrade to 0.3.0.9 or
> 0.3.1.4-alpha.


## Affected packages
 - **tor**
    - **Fixed:** 0.3.0.9
 - **tor-devel**
    - **Fixed:** 0.3.1.4.a
