# cURL -- multiple vulnerabilities
The cURL project reports:

> - FILE buffer read out of bounds
> - TFTP sends more than buffer size
> - URL globbing out of bounds read


## Affected packages
 - **curl**
    - **Fixed:** 7.55.0
