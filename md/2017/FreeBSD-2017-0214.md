# libgd -- Denial of servica via double free
libgd developers report:

> Double free vulnerability in the gdImagePngPtr function in libgd2
> before 2.2.5 allows remote attackers to cause a denial of service via
> vectors related to a palette with no colors.


## Affected packages
 - **libgd**
    - **Fixed:** 2.2.5
