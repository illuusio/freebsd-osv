# kde-runtime -- kdesu: displayed command truncated by unicode string terminator
Albert Aastals Cid reports:

> A maliciously crafted command line for kdesu can result in the user
> only seeing part of the commands that will actually get executed as
> super user.


## Affected packages
 - **kde-runtime**
    - **Fixed:** 4.14.3_5
