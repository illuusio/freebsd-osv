# miniupnpc -- integer signedness error
Tintinweb reports:

> An integer signedness error was found in miniupnp\'s miniwget allowing
> an unauthenticated remote entity typically located on the local
> network segment to trigger a heap corruption or an access violation in
> miniupnp\'s http response parser when processing a specially crafted
> chunked-encoded response to a request for the xml root description
> url.


## Affected packages
 - **miniupnpc**
    - **Fixed:** 2.0.20170509
