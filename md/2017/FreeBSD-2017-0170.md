# sqlite3 -- heap-buffer overflow
Google reports:

> A heap-buffer overflow (sometimes a crash) can arise when running a
> SQL request on malformed sqlite3 databases.


## Affected packages
 - **sqlite3**
    - **Fixed:** 3.20.0
