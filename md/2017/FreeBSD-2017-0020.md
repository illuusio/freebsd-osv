# phpmailer -- Remote Code Execution
SecurityFocus reports:

> PHPMailer is prone to an local information-disclosure vulnerability.
> Attackers can exploit this issue to obtain sensitive information that
> may aid in launching further attacks.


## Affected packages
 - **phpmailer**
    - **Fixed:** 5.2.22
 - **tt-rss**
