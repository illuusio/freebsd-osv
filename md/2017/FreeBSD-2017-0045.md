# tiff -- multiple vulnerabilities
libtiff project reports:

> Multiple flaws have been discovered in libtiff library and utilities.

 - **tiff**
    - **Fixed:** 4.0.7
 - **linux-c6-libtiff**
    - **Fixed:** 3.9.4_5
 - **linux-c6-tiff**
    - **Fixed:** 3.9.4_5
 - **linux-c7-libtiff**
    - **Fixed:** 4.0.3_3
 - **linux-c7-tiff**
    - **Fixed:** 4.0.3_3
