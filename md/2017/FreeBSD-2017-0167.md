# Cacti -- Cross-site scripting (XSS) vulnerability in auth_profile.php
kimiizhang reports:

> Cross-site scripting (XSS) vulnerability in auth_profile.php in Cacti
> 1.1.13 allows remote authenticated users to inject arbitrary web
> script or HTML via specially crafted HTTP Referer headers.
