# mysql -- denial of service vulnerability
Openwall reports:

> C client library for MySQL (libmysqlclient.so) has use-after-free
> defect which can cause crash of applications using that MySQL client.


## Affected packages
 - **mariadb55-client**
    - **Fixed:** 5.5.54
 - **mariadb100-client**
    - **Fixed:** 10.0.30
 - **mariadb101-client**
    - **Fixed:** 10.1.22
 - **mysql55-client**
    - **Fixed:** 5.5.54
 - **mysql56-client**
    - **Fixed:** 5.6.21
 - **mysql57-client**
    - **Fixed:** 5.7.5
