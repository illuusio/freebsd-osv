# libsoup -- stack based buffer overflow
Tobias Mueller reports:

> libsoup is susceptible to a stack based buffer overflow attack when
> using chunked encoding. Regardless of libsoup being used as a server
> or client.


## Affected packages
 - **libsoup**
    - **Fixed:** 2.52.2_1
