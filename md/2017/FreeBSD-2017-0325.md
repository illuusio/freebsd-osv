# mozilla -- multiple vulnerabilities
Mozilla Foundation reports:

> CVE-2017-7845: Buffer overflow when drawing and validating elements
> with ANGLE library using Direct 3D 9
>
> CVE-2017-7846: JavaScript Execution via RSS in mailbox:// origin
>
> CVE-2017-7847: Local path string can be leaked from RSS feed
>
> CVE-2017-7848: RSS Feed vulnerable to new line Injection
>
> CVE-2017-7829: Mailsploit part 1: From address with encoded null
> character is cut off in message header display
