# The Bouncy Castle Crypto APIs: CVE-2017-13098 ("ROBOT")
The Legion of the Bouncy Castle reports:

> Release: 1.59
>
> CVE-2017-13098 (\"ROBOT\"), a Bleichenbacher oracle in TLS when RSA
> key exchange is negotiated. This potentially affected BCJSSE servers
> and any other TLS servers configured to use JCE for the underlying
> crypto - note the two TLS implementations using the BC lightweight
> APIs are not affected by this.


## Affected packages
 - **bouncycastle**
    - **Fixed:** 1.59
 - **bouncycastle15**
    - **Fixed:** 1.59
