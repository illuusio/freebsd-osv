# libsamplerate -- multiple vulnerabilities
NVD reports:

> In libsamplerate before 0.1.9, a buffer over-read occurs in the
> calc_output_single function in src_sinc.c via a crafted audio file.


## Affected packages
 - **libsamplerate**
    - **Fixed:** 0.1.9
 - **linux-c6-libsamplerate**
    - **Fixed:** 0.1.9
 - **linux-c7-libsamplerate**
    - **Fixed:** 0.1.9
