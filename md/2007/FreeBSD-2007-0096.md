# flyspray -- authentication bypass
The Flyspray Project reports:

> Flyspray authentication system can be bypassed by sending a carefully
> crafted post request.
>
> To be vulnerable, PHP configuration directive output_buffering has to
> be disabled or set to a low value.


## Affected packages
 - **flyspray**
    - **Fixed:** 0.9.9.2
