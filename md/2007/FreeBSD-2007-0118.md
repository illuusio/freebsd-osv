# ldapscripts -- Command Line User Credentials Disclosure
Ganael Laplanche reports:

> Up to now, each ldap\* command was called with the -w parameter, which
> allows to specify the bind password on the command line.
> Unfortunately, this could make the password appear to anybody
> performing a \`ps\` during the call. This is now avoided by using the
> -y parameter and a password file.


## Affected packages
 - **ldapscripts**
    - **Fixed:** 1.7.1
