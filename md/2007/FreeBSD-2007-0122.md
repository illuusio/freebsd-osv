# openldap -- multiple remote denial of service vulnerabilities
BugTraq reports:

> OpenLDAP is prone to multiple remote denial-of-service vulnerabilities
> because of an incorrect NULL-termination issue and a double-free
> issue.


## Affected packages
 - **openldap-server**
    - **Fixed:** 2.3.39
    - **Introduced:** 2.4.0,1
    - **Fixed:** 2.4.6
