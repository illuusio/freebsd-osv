# dircproxy -- remote denial of service
Securiweb reports:

> dircproxy allows remote attackers to cause a denial of service
> (segmentation fault) via an ACTION command without a parameter, which
> triggers a NULL pointer dereference, as demonstrated using a blank /me
> message from irssi.


## Affected packages
 - **dircproxy**
    - **Fixed:** 1.0.5_1
 - **dircproxy-devel**
    - **Fixed:** 1.2.0.b2_1
