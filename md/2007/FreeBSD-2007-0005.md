# fetchmail -- crashes when refusing a message bound for an MDA
Matthias Andree reports:

> When delivering messages to a message delivery agent by means of the
> \"mda\" option, fetchmail can crash (by passing a NULL pointer to
> ferror() and fflush()) when refusing a message. SMTP and LMTP delivery
> modes aren\'t affected.


## Affected packages
 - **fetchmail**
    - **Introduced:** 6.3.5
    - **Fixed:** 6.3.6
