# liveMedia -- DoS vulnerability
The live555 development team reports:

> Fixed a bounds-checking error in \"parseRTSPRequestString()\" caused
> by an int vs. unsigned problem.

> The function which handles the incoming queries from the clients is
> affected by a vulnerability which allows an attacker to crash the
> server remotely using the smallest RTSP query possible to use.


## Affected packages
 - **liveMedia**
    - **Fixed:** 2007.11.18,1
