# wordpress -- unmoderated comments disclosure
Blogsecurity reports:

> An attacker can read comments on posts that have not been moderated.
> This can be a real security risk if blog admins are using unmoderated
> comments (comments that have not been made public) to hide sensitive
> notes regarding posts, future work, passwords etc. So please be
> careful if you are one of these blog admins.


## Affected packages
 - **wordpress**
    - **Fixed:** 2.2.2
 - **de-wordpress**
    - **Fixed:** 2.2.2
 - **zh-wordpress**
    - **Fixed:** 2.2.2
