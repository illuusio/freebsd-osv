# flac123 -- stack overflow in comment parsing
isecpartners reports:

> flac123, also known as flac-tools, is vulnerable to a buffer overflow
> in vorbis comment parsing. This allows for the execution of arbitrary
> code.


## Affected packages
 - **flac123**
    - **Fixed:** 0.0.10
