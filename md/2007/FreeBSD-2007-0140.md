# phpmyadmin -- Cross Site Scripting
phpMyAdmin security announcement:

> The login page auth_type cookie was vulnerable to XSS via the
> convcharset parameter. An attacker could use this to execute malicious
> code on the visitors computer


## Affected packages
 - **phpmyadmin**
    - **Fixed:** 2.11.2.2
