# libvorbis -- Multiple memory corruption flaws
isecpartners reports:

> libvorbis contains several vulnerabilities allowing heap overwrite,
> read violations and a function pointer overwrite. These bugs cause a
> at least a denial of service, and potentially code execution.


## Affected packages
 - **libvorbis**
    - **Fixed:** 1.2.0,3
