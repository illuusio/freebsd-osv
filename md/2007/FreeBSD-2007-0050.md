# mod_jk -- information disclosure
Kazu Nambo reports:

> URL decoding the the Apache webserver prior to decoding in the Tomcat
> server could pypass access control rules and give access to pages on a
> different AJP by sending a crafted URL.


## Affected packages
 - **mod_jk**
    - **Fixed:** 1.2.23,1
 - **mod_jk-ap2**
    - **Fixed:** 1.2.23
