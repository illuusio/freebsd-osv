# claws-mail -- APOP vulnerability
CVE reports:

> The APOP protocol allows remote attackers to guess the first 3
> characters of a password via man-in-the-middle (MITM) attacks that use
> crafted message IDs and MD5 collisions.


## Affected packages
 - **claws-mail**
    - **Fixed:** 2.9.0
