# png -- DoS crash vulnerability
A Libpng Security Advisory reports:

> A grayscale PNG image with a malformed (bad CRC) tRNS chunk will crash
> some libpng applications.
>
> This vulnerability could be used to crash a browser when a user tries
> to view such a malformed PNG file. It is not known whether the
> vulnerability could be exploited otherwise.


## Affected packages
 - **png**
    - **Fixed:** 1.2.17
