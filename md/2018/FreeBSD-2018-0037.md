# python -- possible integer overflow vulnerability
Python issue:

> There is a possible integer overflow in PyString_DecodeEscape function
> of the file stringobject.c, which can be abused to gain a heap
> overflow, possibly leading to arbitrary code execution.


## Affected packages
 - **python34**
    - **Fixed:** 3.4.8
 - **python35**
    - **Fixed:** 3.5.5
