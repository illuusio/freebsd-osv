# GraphicsMagick -- SVG/Rendering vulnerability
GraphicsMagick News:

> Fix heap write overflow of PrimitiveInfo and PointInfo arrays. This is
> another manefestation of CVE-2016-2317, which should finally be fixed
> correctly due to active detection/correction of pending overflow
> rather than using estimation.
