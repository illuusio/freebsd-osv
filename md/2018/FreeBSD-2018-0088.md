# SQLite -- Corrupt DB can cause a NULL pointer dereference
MITRE reports:

> SQLite databases whose schema is corrupted using a CREATE TABLE AS
> statement could cause a NULL pointer dereference, related to build.c
> and prepare.c.


## Affected packages
 - **sqlite3**
    - **Fixed:** 3.22.0_1
