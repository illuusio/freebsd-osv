# OpenSSL -- Multiple vulnerabilities in 1.1 branch
The OpenSSL project reports:

> Timing vulnerability in ECDSA signature generation (CVE-2018-0735):
> The OpenSSL ECDSA signature algorithm has been shown to be vulnerable
> to a timing side channel attack. An attacker could use variations in
> the signing algorithm to recover the private key (Low).
>
> Timing vulnerability in DSA signature generation (CVE-2018-0734):
> Avoid a timing attack that leaks information via a side channel that
> triggers when a BN is resized. Increasing the size of the BNs prior to
> doing anything with them suppresses the attack (Low).


## Affected packages
 - **openssl-devel**
    - **Fixed:** 1.1.0i_1
 - **openssl111**
    - **Fixed:** 1.1.1_2
 - **libressl**
    - **Introduced:** 2.8.0
    - **Fixed:** 2.8.3
 - **libressl-devel**
    - **Introduced:** 2.8.0
    - **Fixed:** 2.8.3
