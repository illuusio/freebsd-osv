# mozilla -- multiple vulnerabilities
The Mozilla Foundation reports:

> # CVE-2018-5146: Out of bounds memory write in libvorbis
>
> An out of bounds memory write while processing Vorbis audio data was
> reported through the Pwn2Own contest.
>
> # CVE-2018-5147: Out of bounds memory write in libtremor
>
> The libtremor library has the same flaw as CVE-2018-5146. This library
> is used by Firefox in place of libvorbis on Android and ARM platforms.
