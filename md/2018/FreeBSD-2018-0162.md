# clamav -- multiple vulnerabilities
Joel Esler reports:

> 3 security fixes in this release:
>
> - CVE-2017-16932: Vulnerability in libxml2 dependency (affects ClamAV
>   on Windows only).
> - CVE-2018-0360: HWP integer overflow, infinite loop vulnerability.
>   Reported by Secunia Research at Flexera.
> - CVE-2018-0361: ClamAV PDF object length check, unreasonably long
>   time to parse relatively small file. Report ed by aCaB.


## Affected packages
 - **clamav**
    - **Fixed:** 0.100.1
