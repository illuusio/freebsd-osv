# Containous Traefik -- exposes the configuration and secret
MITRE reports:

> Containous Traefik 1.6.x before 1.6.6, when \--api is used, exposes
> the configuration and secret if authentication is missing and the
> API\'s port is publicly reachable.


## Affected packages
 - **traefik**
    - **Fixed:** 1.6.6
