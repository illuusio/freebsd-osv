# tinc -- Buffer overflow
tinc-vpn.org reports:

> The authentication protocol allows an oracle attack that could
> potentially be exploited.
>
> If a man-in-the-middle has intercepted the TCP connection it might be
> able to force plaintext UDP packets between two nodes for up to a
> PingInterval period.


## Affected packages
 - **tinc**
    - **Fixed:** 1.0.35
 - **tinc-devel**
    - **Fixed:** 1.1pre17
