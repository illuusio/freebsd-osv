# qutebrowser -- Remote code execution due to CSRF
qutebrowser team reports:

> Due to a CSRF vulnerability affecting the qute://settings page, it was
> possible for websites to modify qutebrowser settings. Via settings
> like editor.command, this possibly allowed websites to execute
> arbitrary code.


## Affected packages
 - **qutebrowser**
    - **Introduced:** 1.4.0
    - **Fixed:** 1.4.1
    - **Introduced:** 1.0.0
    - **Fixed:** 1.3.3_1
