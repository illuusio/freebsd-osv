# firefox -- Arbitrary code execution through unsanitized browser UI
The Mozilla Foundation reports:

> Mozilla developer **Johann Hofmann** reported that unsanitized output
> in the browser UI can lead to arbitrary code execution.


## Affected packages
 - **firefox**
    - **Fixed:** 58.0.1,1
 - **waterfox**
    - **Fixed:** 56.0.3.65
