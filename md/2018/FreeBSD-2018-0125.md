# wget -- cookie injection vulnerability
Harry Sintonen of F-Secure Corporation reports:

> GNU Wget is susceptible to a malicious web server injecting arbitrary
> cookies to the cookie jar file.


## Affected packages
 - **wget**
    - **Fixed:** 1.19.5
