# p7zip -- heap-based buffer overflow
MITRE reports:

> Heap-based buffer overflow in the
> NCompress::NShrink::CDecoder::CodeReal method in 7-Zip before 18.00
> and p7zip allows remote attackers to cause a denial of service
> (out-of-bounds write) or potentially execute arbitrary code via a
> crafted ZIP archive.


## Affected packages
 - **p7zip**
    - **Fixed:** 16.02_1
