# curl -- SMTP send heap buffer overflow
Peter Wu reports:

> curl might overflow a heap based memory buffer when sending data over
> SMTP and using a reduced read buffer.


## Affected packages
 - **curl**
    - **Introduced:** 7.54.1
    - **Fixed:** 7.61.0
