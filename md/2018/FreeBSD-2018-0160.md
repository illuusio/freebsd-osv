# wordpress -- multiple issues
wordpressdevelopers reports:

> Taxonomy: Improve cache handling for term queries.
>
> Posts, Post Types: Clear post password cookie when logging out.
>
> Widgets: Allow basic HTML tags in sidebar descriptions on Widgets
> admin screen.
>
> Community Events Dashboard: Always show the nearest WordCamp if one is
> coming up, even if there are multiple Meetups happening first.
>
> Privacy: Make sure default privacy policy content does not cause a
> fatal error when flushing rewrite rules outside of the admin context.
