# Flash Player -- multiple vulnerabilities
Adobe reports:

> - This update resolves out-of-bounds read vulnerabilities that could
>   lead to information disclosure (CVE-2018-12824, CVE-2018-12826,
>   CVE-2018-12827).
> - This update resolves a security bypass vulnerability that could lead
>   to security mitigation bypass (CVE-2018-12825).
> - This update resolves a component vulnerability that could lead to
>   privilege escalation (CVE-2018-12828).


## Affected packages
 - **linux-flashplayer**
    - **Fixed:** 30.0.0.154
