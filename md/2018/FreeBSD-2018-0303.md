# FreeBSD -- bootpd buffer overflow
# Problem Description:

Due to insufficient validation of network-provided data it may be
possible for a malicious attacker to craft a bootp packet which could
cause a stack buffer overflow.

# Impact:

It is possible that the buffer overflow could lead to a Denial of
Service or remote code execution.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 12.0
    - **Fixed:** 12.0_1
    - **Introduced:** 11.2
    - **Fixed:** 11.2_7
