# consul -- vulnerability in embedded DNS library
Consul developers report:

> A flaw was found in the embedded DNS library used in consul which may
> allow a denial of service attack. Consul was updated to include the
> fixed version.


## Affected packages
 - **consul**
    - **Fixed:** 1.0.5
