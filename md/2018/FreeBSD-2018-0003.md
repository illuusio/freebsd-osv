# awstats -- remote code execution
Mitre reports:

> Awstats version 7.6 and earlier is vulnerable to a path traversal flaw
> in the handling of the \"config\" and \"migrate\" parameters resulting
> in unauthenticated remote code execution.


## Affected packages
 - **awstats**
    - **Fixed:** 7.7,1
