# kamailio - buffer overflow
A specially crafted REGISTER message with a malformed branch or From tag
triggers an off-by-one heap-based buffer overflow in the
tmx_check_pretran function in modules/tmx/tmx_pretran.c


## Affected packages
 - **kamailio**
    - **Fixed:** 5.1.2
