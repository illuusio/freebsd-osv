# Mbed TLS -- Local timing attack on RSA decryption
Janos Follath reports:

> An attacker who can run code on the same machine that is performing an
> RSA decryption can potentially recover the plaintext through a
> Bleichenbacher-like oracle.


## Affected packages
 - **mbedtls**
    - **Fixed:** 2.14.1
