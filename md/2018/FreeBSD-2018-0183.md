# OpenJPEG -- multiple vulnerabilities
OpenJPEG reports:

> Multiple vulnerabilities have been found in OpenJPEG, the opensource
> JPEG 2000 codec. Please consult the CVE list for further details.
>
> CVE-2017-17479 and CVE-2017-17480 were fixed in r477112.
>
> CVE-2018-5785 was fixed in r480624.
>
> CVE-2018-6616 was fixed in r489415.


## Affected packages
 - **openjpeg**
    - **Fixed:** 2.3.0_3
