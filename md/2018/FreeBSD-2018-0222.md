# X11 Session -- SDDM allows unauthorised unlocking
MITRE reports:

> An issue was discovered in SDDM through 0.17.0. If configured with
> ReuseSession=true, the password is not checked for users with an
> already existing session. Any user with access to the system D-Bus can
> therefore unlock any graphical session.

The default configuration of SDDM on FreeBSD is not affected, since it
has ReuseSession=false.


## Affected packages
 - **sddm**
    - **Fixed:** 0.17.0_1
