# drupal -- Drupal core - Moderately critical
The Drupal security team reports:

> CKEditor, a third-party JavaScript library included in Drupal core,
> has fixed a cross-site scripting (XSS) vulnerability. The
> vulnerability stemmed from the fact that it was possible to execute
> XSS inside CKEditor when using the image2 plugin (which Drupal 8 core
> also uses).


## Affected packages
 - **drupal8**
    - **Fixed:** 8.5.2
