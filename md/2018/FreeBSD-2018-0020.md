# gcab -- stack overflow
Upstream reports:

> A stack-based buffer overflow within GNOME gcab through 0.7.4 can be
> exploited by malicious attackers to cause a crash or, potentially,
> execute arbitrary code via a crafted .cab file.


## Affected packages
 - **gcab**
    - **Fixed:** 0.8
