# bitmessage -- remote code execution vulnerability
Bitmessage developers report:

> A remote code execution vulnerability has been spotted in use against
> some users running PyBitmessage v0.6.2. The cause was identified and a
> fix has been added and released as 0.6.3.2. (Will be updated if/when
> CVE will be available.)


## Affected packages
 - **bitmessage**
    - **Fixed:** 0.6.2
