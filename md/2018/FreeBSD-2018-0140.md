# firefox -- Heap buffer overflow rasterizing paths in SVG with Skia
The Mozilla Foundation reports:

> A heap buffer overflow can occur in the Skia library when rasterizing
> paths using a maliciously crafted SVG file with anti-aliasing turned
> off. This results in a potentially exploitable crash.


## Affected packages
 - **firefox**
    - **Fixed:** 60.0.2,1
 - **waterfox**
    - **Fixed:** 56.2.0.13_5
 - **firefox-esr**
    - **Fixed:** 52.8.1,1
 - **seamonkey**
    - **Fixed:** 2.49.4
 - **linux-seamonkey**
    - **Fixed:** 2.49.4
