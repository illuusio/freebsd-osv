# links -- denial of service
NIST reports:

> The put_chars function in html_r.c in Twibright Links 2.14 allows
> remote attackers to cause a denial of service (buffer over-read) via a
> crafted HTML file.


## Affected packages
 - **links**
    - **Fixed:** 2.16,1
