# Gitlab -- multiple vulnerabilities
Gitlab reports:

> Wiki XSS
>
> Sanitize gem updates
>
> XSS in url_for(params)
>
> Content injection via username
>
> Activity feed publicly displaying internal project names
>
> Persistent XSS in charts


## Affected packages
 - **gitlab**
    - **Introduced:** 11.0.0
    - **Fixed:** 11.0.1
    - **Introduced:** 10.8.0
    - **Fixed:** 10.8.5
    - **Introduced:** 4.1
    - **Fixed:** 10.7.6
