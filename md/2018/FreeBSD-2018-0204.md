# botan2 -- ECDSA side channel
botan2 developers report:

> A side channel in the ECDSA signature operation could allow a local
> attacker to recover the secret key. Found by Keegan Ryan of NCC Group.
>
> Bug introduced in 2.5.0, fixed in 2.7.0. The 1.10 branch is not
> affected.


## Affected packages
 - **botan2**
    - **Introduced:** 2.5.0
    - **Fixed:** 2.7.0
