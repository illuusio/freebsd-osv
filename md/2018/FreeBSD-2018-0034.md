# exim -- a buffer overflow vulnerability, remote code execution
Exim developers report:

> There is a buffer overflow in base64d(), if some pre-conditions are
> met. Using a handcrafted message, remote code execution seems to be
> possible.


## Affected packages
 - **exim**
    - **Fixed:** 4.90.1
