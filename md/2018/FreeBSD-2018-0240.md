# pango -- remote DoS vulnerability
> libpango in Pango 1.40.8 through 1.42.3, as used in hexchat and other
> products, allows remote attackers to cause a denial of service
> (application crash) or possibly have unspecified other impact via
> crafted text with invalid Unicode sequences.


## Affected packages
 - **pango**
    - **Fixed:** 1.42.4
