# PostgreSQL vulnerabilities
The PostgreSQL project reports:

> - CVE-2018-1058: Uncontrolled search path element in pg_dump and other
>   client applications


## Affected packages
 - **postgresql93-server**
    - **Introduced:** 9.3.0
    - **Fixed:** 9.3.22
 - **postgresql94-server**
    - **Introduced:** 9.4.0
    - **Fixed:** 9.4.17
 - **postgresql95-server**
    - **Introduced:** 9.5.0
    - **Fixed:** 9.5.12
 - **postgresql96-server**
    - **Introduced:** 9.6.0
    - **Fixed:** 9.6.8
 - **postgresql10-server**
    - **Introduced:** 10.0
    - **Fixed:** 10.3
