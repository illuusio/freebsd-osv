# libraw -- multiple DoS vulnerabilities
Secunia Research reports:

> CVE-2017-16909: An error related to the
> \"LibRaw::panasonic_load_raw()\" function (dcraw_common.cpp) can be
> exploited to cause a heap-based buffer overflow and subsequently cause
> a crash via a specially crafted TIFF image.
>
> CVE-2017-16910: An error within the \"LibRaw::xtrans_interpolate()\"
> function (internal/dcraw_common.cpp) can be exploited to cause an
> invalid read memory access.


## Affected packages
 - **libraw**
    - **Fixed:** 0.18.6
