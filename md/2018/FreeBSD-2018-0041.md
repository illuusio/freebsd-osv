# bchunk -- access violation near NULL on destination operand and crash
Mitre reports:

> bchunk 1.2.0 and 1.2.1 is vulnerable to an \"Access violation near
> NULL on destination operand\" and crash when processing a malformed
> CUE (.cue) file.


## Affected packages
 - **bchunk**
    - **Introduced:** 1.2.0
    - **Fixed:** 1.2.1
