# h2o -- heap buffer overflow during logging
Marlies Ruck reports:

> Fix heap buffer overflow while trying to emit access log - see
> references for full details.
>
> CVE-2018-0608: Buffer overflow in H2O version 2.2.4 and earlier allows
> remote attackers to execute arbitrary code or cause a denial of
> service (DoS) via unspecified vectors.


## Affected packages
 - **h2o**
    - **Fixed:** 2.2.5
