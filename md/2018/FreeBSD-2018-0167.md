# Several Security Defects in the Bouncy Castle Crypto APIs
The Legion of the Bouncy Castle reports:

> Release 1.60 is now available for download.
>
> CVE-2018-1000180: issue around primality tests for RSA key pair
> generation if done using only the low-level API.
>
> CVE-2018-1000613: lack of class checking in deserialization of
> XMSS/XMSS\^MT private keys with BDS state information.


## Affected packages
 - **bouncycastle**
    - **Fixed:** 1.60
 - **bouncycastle15**
    - **Fixed:** 1.60
 - **puppetserver**
 - **puppetserver5**
    - **Fixed:** 5.3.8
 - **puppetserver6**
    - **Fixed:** 6.2.1
