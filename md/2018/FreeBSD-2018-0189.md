# sinatra -- XSS vulnerability
Sinatra blog:

> Sinatra had a critical vulnerability since v2.0.0. The purpose of this
> release is to fix CVE-2018-11627.
>
> The vulnerability is that XSS can be executed by using illegal
> parameters.


## Affected packages
 - **rubygem-sinatra**
    - **Introduced:** 2.0.0
    - **Fixed:** 2.0.2
