# FreeBSD -- Improper ELF header parsing
# Problem Description:

Insufficient validation was performed in the ELF header parser, and
malformed or otherwise invalid ELF binaries were not rejected as they
should be.

# Impact:

Execution of a malicious ELF binary may result in a kernel crash or may
disclose kernel memory.


## Affected packages
 - **FreeBSD-kernel**
    - **Introduced:** 11.2
    - **Fixed:** 11.2_3
    - **Introduced:** 11.1
    - **Fixed:** 11.1_14
    - **Introduced:** 10.4
