# lshell -- Multiple security issues
lshell reports:

> It is possible to escape lshell if an allowed command can execute an
> arbitrary non allowed one (issue #122).
>
> Inappropriate parsing of commands can lead to arbitrary command
> execution (issue #147, #149, #151).


## Affected packages
 - **lshell**
    - **Fixed:** 0.9.18
