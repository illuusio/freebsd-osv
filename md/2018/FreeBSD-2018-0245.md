# Gitlab -- multiple vulnerabilities

> Merge request information disclosure
>
> Private project namespace information disclosure
>
> Gitlab Flavored Markdown API information disclosure

    - **Introduced:** 11.2.0
    - **Fixed:** 11.2.5
    - **Introduced:** 10.2.0
