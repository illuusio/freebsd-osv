# asterisk -- Infinite loop when reading iostreams
The Asterisk project reports:

> When connected to Asterisk via TCP/TLS if the client abruptly
> disconnects, or sends a specially crafted message then Asterisk gets
> caught in an infinite loop while trying to read the data stream. Thus
> rendering the system as unusable.


## Affected packages
 - **asterisk15**
    - **Fixed:** 15.4.1
