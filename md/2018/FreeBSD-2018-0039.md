# bchunk -- heap-based buffer overflow and crash
Mitre reports:

> bchunk 1.2.0 and 1.2.1 vulnerable to a heap-based buffer overflow and
> crash when processing a malformed CUE (.cue) file.


## Affected packages
 - **bchunk**
    - **Introduced:** 1.2.0
    - **Fixed:** 1.2.1
