# ZendFramework1 -- SQL injection vulnerability
Zend Framework developers report:

> The PDO adapters of Zend Framework 1 do not filter null bytes values
> in SQL statements. A PDO adapter can treat null bytes in a query as a
> string terminator, allowing an attacker to add arbitrary SQL following
> a null byte, and thus create a SQL injection.


## Affected packages
 - **ZendFramework1**
    - **Fixed:** 1.12.16
