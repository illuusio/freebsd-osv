# libraw -- index overflow in smal_decode_segment
ChenQin reports:

> The LibRaw raw image decoder has multiple vulnerabilities that can
> cause memory errors which may lead to code execution or other
> problems.
>
> In CVE-2015-8366, LibRaw\'s smal_decode_segment function does not
> handle indexes carefully, which can cause an index overflow.


## Affected packages
 - **libraw**
    - **Fixed:** 0.17.1
