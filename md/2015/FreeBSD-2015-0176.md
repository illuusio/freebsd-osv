# Adobe Flash Player -- critical vulnerabilities
Adobe reports:

> Adobe has released security updates for Adobe Flash Player. These
> updates address critical vulnerabilities that could potentially allow
> an attacker to take control of the affected system. Adobe is aware of
> a report that an exploit targeting CVE-2015-5119 has been publicly
> published.


## Affected packages
 - **linux-c6-flashplugin**
    - **Fixed:** 11.2r202.481
 - **linux-f10-flashplugin**
    - **Fixed:** 11.2r202.481
