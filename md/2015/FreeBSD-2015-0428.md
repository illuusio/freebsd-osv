# mono -- DoS and code execution
NCC Group reports:

> An attacker who can cause a carefully-chosen string to be converted to
> a floating-point number can cause a crash and potentially induce
> arbitrary code execution.


## Affected packages
 - **mono**
    - **Fixed:** 4.2
