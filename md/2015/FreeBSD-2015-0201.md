# mysql -- SSL Downgrade
Duo Security reports:

> Researchers have identified a serious vulnerability in some versions
> of Oracle's MySQL database product that allows an attacker to strip
> SSL/TLS connections of their security wrapping transparently.


## Affected packages
 - **php56-mysql**
    - **Fixed:** 5.6.11
 - **php56-mysqli**
    - **Fixed:** 5.6.11
 - **php55-mysql**
    - **Fixed:** 5.5.27
 - **php55-mysqli**
    - **Fixed:** 5.5.27
 - **php5-mysql**
    - **Fixed:** 5.4.43
 - **php5-mysqli**
    - **Fixed:** 5.4.43
 - **mariadb55-client**
    - **Fixed:** 5.5.44
 - **mariadb100-client**
    - **Fixed:** 10.0.20
