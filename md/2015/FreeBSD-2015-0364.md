# jenkins -- remote code execution via unsafe deserialization
Jenkins Developers report:

> Unsafe deserialization allows unauthenticated remote attackers to run
> arbitrary code on the Jenkins master.


## Affected packages
 - **jenkins**
    - **Fixed:** 1.638
 - **jenkins-lts**
    - **Fixed:** 1.625.2
