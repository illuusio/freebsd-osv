# e2fsprogs -- potential buffer overflow in closefs()
Theodore Ts\'o reports:

> On a carefully crafted filesystem that gets modified through tune2fs
> or debugfs, it is possible to trigger a buffer overrun when the file
> system is closed via closefs().


## Affected packages
 - **e2fsprogs**
    - **Fixed:** 1.42.12_2
