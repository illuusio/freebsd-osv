# Joomla! -- Core - ACL Violation vulnerabilities
The JSST and the Joomla! Security Center report:

> ## \[20151003\] - Core - ACL Violations
>
> Inadequate ACL checks in com_content provide potential read access to
> data which should be access restricted.


## Affected packages
 - **joomla3**
    - **Introduced:** 3.0.0
    - **Fixed:** 3.4.5
