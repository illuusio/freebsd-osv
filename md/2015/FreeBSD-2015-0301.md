# libssh2 -- denial of service vulnerability
Mariusz Ziulek reports:

> A malicious attacker could man in the middle a real server and cause
> libssh2 using clients to crash (denial of service) or otherwise read
> and use completely unintended memory areas in this process.


## Affected packages
 - **libssh2**
    - **Fixed:** 1.5.0,2
