# ansible -- enable host key checking in paramiko connection type
Ansible changelog reports:

> Host key checking is on by default. Disable it if you like by adding
> host_key_checking=False in the \[default\] section of
> /etc/ansible/ansible.cfg or \~/ansible.cfg or by exporting
> ANSIBLE_HOST_KEY_CHECKING=False.


## Affected packages
 - **ansible**
    - **Fixed:** 1.2.1
