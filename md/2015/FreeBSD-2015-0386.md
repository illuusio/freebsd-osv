# libraw -- memory objects not properly initialized
ChenQin reports:

> The LibRaw raw image decoder has multiple vulnerabilities that can
> cause memory errors which may lead to code execution or other
> problems.
>
> In CVE-2015-8367, LibRaw\'s phase_one_correct function does not handle
> memory initialization correctly, which may cause other problems.


## Affected packages
 - **libraw**
    - **Fixed:** 0.17.1
