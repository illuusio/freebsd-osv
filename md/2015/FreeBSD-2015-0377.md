# kibana4 -- CSRF vulnerability
Elastic reports:

> Vulnerability Summary: Kibana versions prior to 4.1.3 and 4.2.1 are
> vulnerable to a CSRF attack.
>
> Remediation Summary: Users should upgrade to 4.1.3 or 4.2.1.


## Affected packages
 - **kibana4**
    - **Introduced:** 4.0.0
    - **Fixed:** 4.1.3
 - **kibana41**
    - **Introduced:** 4.0.0
    - **Fixed:** 4.1.3
 - **kibana42**
    - **Introduced:** 4.2.0
    - **Fixed:** 4.2.1
