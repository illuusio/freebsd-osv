# pivotx -- Multiple unrestricted file upload vulnerabilities
Pivotx reports:

> Multiple unrestricted file upload vulnerabilities in fileupload.php in
> PivotX before 2.3.9 allow remote authenticated users to execute
> arbitrary PHP code by uploading a file with a (1) .php or (2) .php#
> extension, and then accessing it via unspecified vectors.


## Affected packages
 - **pivotx**
    - **Fixed:** 2.3.9
