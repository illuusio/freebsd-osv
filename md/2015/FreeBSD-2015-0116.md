# rest-client -- session fixation vulnerability
Andy Brody reports:

> When Ruby rest-client processes an HTTP redirection response, it
> blindly passes along the values from any Set-Cookie headers to the
> redirection target, regardless of domain, path, or expiration.


## Affected packages
 - **rubygem-rest-client**
    - **Fixed:** 1.6.7_1
