# Joomla! -- Core - XSS Vulnerability
The JSST and the Joomla! Security Center report:

> ## \[20150908\] - Core - XSS Vulnerability
>
> Inadequate escaping leads to XSS vulnerability in login module.


## Affected packages
 - **joomla3**
    - **Introduced:** 3.4.0
    - **Fixed:** 3.4.4
