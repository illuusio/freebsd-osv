# rubygem-rails -- multiple vulnerabilities
Ruby on Rails blog:

> Rails 3.2.22, 4.1.11 and 4.2.2 have been released, along with web
> console and jquery-rails plugins and Rack 1.5.4 and 1.6.2.


## Affected packages
 - **rubygem-activesupport**
    - **Fixed:** 3.2.22
 - **rubygem-activesupport4**
    - **Fixed:** 4.2.2
 - **rubygem-jquery-rails**
    - **Fixed:** 3.1.3
 - **rubygem-jquery-rails4**
    - **Fixed:** 4.0.4
 - **rubygem-rack**
    - **Fixed:** 1.4.6
 - **rubygem-rack15**
    - **Fixed:** 1.5.4
 - **rubygem-rack16**
    - **Fixed:** 1.6.2
 - **rubygem-rails**
    - **Fixed:** 3.2.22
 - **rubygem-rails4**
    - **Fixed:** 4.2.2
 - **rubygem-web-console**
    - **Fixed:** 2.1.3
