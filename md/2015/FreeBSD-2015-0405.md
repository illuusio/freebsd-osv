# java -- multiple vulnerabilities
Oracle reports:

> This Critical Patch Update contains 25 new security fixes for Oracle
> Java SE. 24 of these vulnerabilities may be remotely exploitable
> without authentication, i.e., may be exploited over a network without
> the need for a username and password.


## Affected packages
 - **openjdk8**
    - **Fixed:** 8.66.17
 - **openjdk8-jre**
    - **Fixed:** 8.66.17
 - **openjdk7**
    - **Fixed:** 7.91.02,1
 - **openjdk7-jre**
    - **Fixed:** 7.91.02,1
