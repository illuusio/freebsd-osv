# elasticsearch -- directory traversal attack via snapshot API
Elastic reports:

> Vulnerability Summary: Elasticsearch versions from 1.0.0 to 1.6.0 are
> vulnerable to a directory traversal attack.
>
> Remediation Summary: Users should upgrade to 1.6.1 or later, or
> constrain access to the snapshot API to trusted sources.


## Affected packages
 - **elasticsearch**
    - **Introduced:** 1.0.0
    - **Fixed:** 1.6.1
