# Quassel IRC -- SQL injection vulnerability
Quassel IRC developers report:

> Restarting a PostgreSQL database while Quassel Core is running would
> not properly re-initialize the database session inside Quassel,
> bringing back an old security issue (CVE-2013-4422).


## Affected packages
 - **quassel**
    - **Fixed:** 0.11.1
