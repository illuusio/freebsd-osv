# Xymon -- buffer overrun
Debian reports:

> web/acknowledge.c uses a string twice in a format string, but only
> allocates memory for one copy.


## Affected packages
 - **xymon-server**
    - **Introduced:** 4.3.4
    - **Fixed:** 4.3.18
