# xscreensaver - lock bypass
RedHat bugzilla reports:

> In dual screen configurations, unplugging one screen will cause
> xscreensaver to crash, leaving the screen unlocked.


## Affected packages
 - **xscreensaver**
    - **Fixed:** 5.34
 - **xscreensaver-gnome**
    - **Fixed:** 5.34
 - **xscreensaver-gnome-hacks**
    - **Fixed:** 5.34
