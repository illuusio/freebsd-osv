# miniupnpc -- buffer overflow
Talos reports:

> An exploitable buffer overflow vulnerability exists in the XML parser
> functionality of the MiniUPnP library. A specially crafted XML
> response can lead to a buffer overflow on the stack resulting in
> remote code execution. An attacker can set up a server on the local
> network to trigger this vulnerability.


## Affected packages
 - **miniupnpc**
    - **Introduced:** 1.9.1
    - **Fixed:** 1.9.20150917
    - **Fixed:** 1.9_1
