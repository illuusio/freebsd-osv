# Wesnoth -- Remote information disclosure
US-CERT/NIST reports:

> The WML/Lua API in Battle for Wesnoth 1.7.x through 1.11.x and 1.12.x
> before 1.12.2 allows remote attackers to read arbitrary files via a
> crafted (1) campaign or (2) map file.


## Affected packages
 - **wesnoth**
    - **Introduced:** 1.7.0
    - **Fixed:** 1.12.2
 - **wesnoth-devel**
    - **Introduced:** 1.7.0
    - **Fixed:** 1.12.2
