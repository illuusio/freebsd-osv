# WebKit-gtk -- Multiple vulnerabilities

> This release fixes the following security issues: CVE-2014-1344,
> CVE-2014-1384, CVE-2014-1385, CVE-2014-1386, CVE-2014-1387,
> CVE-2014-1388, CVE-2014-1389, CVE-2014-1390.

 - **webkit-gtk2**

 - **webkit-gtk3**
    - **Fixed:** 1.4.8
