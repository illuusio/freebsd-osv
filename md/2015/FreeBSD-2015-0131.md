# security/ossec-hids-* -- root escalation via syscheck feature
OSSEC reports:

> The CVE-2015-3222 vulnerability, which allows for root escalation via
> sys check has been fixed in OSSEC 2.8.2. This issue does not affect
> agents.


## Affected packages
 - **ossec-hids-server**
    - **Introduced:** 2.7
    - **Fixed:** 2.8.2
 - **ossec-hids-client**
    - **Introduced:** 2.7
    - **Fixed:** 2.8.2
 - **ossec-hids-local**
    - **Introduced:** 2.7
    - **Fixed:** 2.8.2
