# pear-twig -- remote code execution
Fabien Potencier reports:

> End users can craft valid Twig code that allows them to execute
> arbitrary code (RCEs) via the \_self variable, which is always
> available, even in sandboxed templates.


## Affected packages
 - **pear-twig-twig**
    - **Fixed:** 1.20.0
