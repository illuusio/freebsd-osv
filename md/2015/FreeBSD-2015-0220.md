# sox -- memory corruption vulnerabilities
Michele Spagnuolo, Google Security Team, reports:

> The write heap buffer overflows are related to ADPCM handling in WAV
> files, while the read heap buffer overflow is while opening a .VOC.


## Affected packages
 - **sox**
    - **Fixed:** 14.4.2
