# remind -- buffer overflow with malicious reminder file input
Dianne Skoll reports:

> BUG FIX: Fix a buffer overflow found by Alexander Keller.

The bug can be manifested by an extended DUMP command using a system
variable (that is a special variable whose name begins with \'\$\')


## Affected packages
 - **remind**
    - **Fixed:** 3.1.15
