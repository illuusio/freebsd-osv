# unzip -- out of boundary access issues in test_compr_eb
Ubuntu Security Notice USN-2489-1 reports:

> Michal Zalewski discovered that unzip incorrectly handled certain
> malformed zip archives. If a user or automated system were tricked
> into processing a specially crafted zip archive, an attacker could
> possibly execute arbitrary code.


## Affected packages
 - **unzip**
    - **Fixed:** 6.0_4
