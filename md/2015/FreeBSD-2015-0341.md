# Joomla! -- Core - Open Redirect vulnerability
The JSST and the Joomla! Security Center report:

> ## \[20150601\] - Core - Open Redirect
>
> Inadequate checking of the return value allowed to redirect to an
> external page.


## Affected packages
 - **joomla3**
    - **Introduced:** 3.0.0
    - **Fixed:** 3.4.2
