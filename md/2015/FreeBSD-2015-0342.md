# Joomla! -- Core - CSRF Protection vulnerabilities
The JSST and the Joomla! Security Center report:

> ## \[20150602\] - Core - CSRF Protection
>
> Lack of CSRF checks potentially enabled uploading malicious code.


## Affected packages
 - **joomla3**
    - **Introduced:** 3.2.0
    - **Fixed:** 3.4.2
