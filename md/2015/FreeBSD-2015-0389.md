# KeePassX -- information disclosure
Yves-Alexis Perez reports:

> Starting an export (using File / Export to / KeepassX XML file) and
> cancelling it leads to KeepassX saving a cleartext XML file in \~/.xml
> without any warning.


## Affected packages
 - **KeePassX**
    - **Fixed:** 0.4.4
