# OpenSSH -- PermitRootLogin may allow password connections with 'without-password'
> OpenSSH 7.0 contained a logic error in PermitRootLogin=
> prohibit-password/without-password that could, depending on
> compile-time configuration, permit password authentication to root
> while preventing other forms of authentication. This problem was
> reported by Mantas Mikulenas.


## Affected packages
 - **openssh-portable**
    - **Introduced:** 7.0.p1,1
    - **Fixed:** 7.0.p1,1
