# Joomla! -- Core - XSS Vulnerability
The JSST and the Joomla! Security Center report:

> ## \[20140901\] - Core - XSS Vulnerability
>
> Inadequate escaping leads to XSS vulnerability in com_media.


## Affected packages
 - **joomla3**
    - **Introduced:** 3.2.0
    - **Fixed:** 3.2.5
    - **Introduced:** 3.3.0
    - **Fixed:** 3.3.4
