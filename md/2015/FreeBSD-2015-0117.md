# rest-client -- plaintext password disclosure
The open sourced vulnerability database reports:

> REST Client for Ruby contains a flaw that is due to the application
> logging password information in plaintext. This may allow a local
> attacker to gain access to password information.


## Affected packages
 - **rubygem-rest-client**
    - **Fixed:** 1.6.7_1
