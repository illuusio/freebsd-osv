# django -- multiple vulnerabilities
The Django project reports:

> Today the Django team is issuing multiple releases \-- Django 1.4.18,
> Django 1.6.10, and Django 1.7.3 \-- as part of our security process.
> These releases are now available on PyPI and our download page.
>
> These releases address several security issues. We encourage all users
> of Django to upgrade as soon as possible.
