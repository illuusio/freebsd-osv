# Joomla! -- Core - Unauthorized Login vulnerability
The JSST and the Joomla! Security Center report:

> ## \[20140902\] - Core - Unauthorized Logins
>
> Inadequate checking allowed unauthorized logins via LDAP
> authentication.


## Affected packages
 - **joomla3**
    - **Fixed:** 3.2.5
    - **Introduced:** 3.3.0
    - **Fixed:** 3.3.4
 - **joomla2**
    - **Fixed:** 2.5.25
