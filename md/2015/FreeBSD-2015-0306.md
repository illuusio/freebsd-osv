# codeigniter -- mysql database driver vulnerability
The CodeIgniter changelog reports:

> Security: Removed a fallback to mysql_escape_string() in the mysql
> database driver (escape_str() method) when there\'s no active database
> connection.


## Affected packages
 - **codeigniter**
    - **Fixed:** 2.2.3
