# samba -- Elevation of privilege to Active Directory Domain Controller
Samba team reports:

> In Samba\'s AD DC we neglected to ensure that attempted modifications
> of the userAccountControl attribute did not allow the
> UF_SERVER_TRUST_ACCOUNT bit to be set.


## Affected packages
 - **samba4**
    - **Introduced:** 4.0.0
    - **Fixed:** 4.0.23
 - **samba41**
    - **Introduced:** 4.1.0
    - **Fixed:** 4.1.15
