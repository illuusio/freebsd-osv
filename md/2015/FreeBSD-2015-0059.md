# osc -- shell command injection via crafted _service files
SUSE Security Update reports:

> osc before 0.151.0 allows remote attackers to execute arbitrary
> commands via shell metacharacters in a \_service file.


## Affected packages
 - **osc**
    - **Fixed:** 0.151.0
