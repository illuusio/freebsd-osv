# phpMyAdmin -- Content spoofing vulnerability
The phpMyAdmin development team reports:

> This vulnerability allows an attacker to perform a content spoofing
> attack using the phpMyAdmin\'s redirection mechanism to external
> sites.
>
> We consider this vulnerability to be non critical since the spoofed
> content is escaped and no HTML injection is possible.


## Affected packages
 - **phpMyAdmin**
    - **Introduced:** 4.4.0
    - **Fixed:** 4.4.15.1
    - **Introduced:** 4.5.0
    - **Fixed:** 4.5.1
