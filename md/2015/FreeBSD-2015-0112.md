# proxychains-ng -- current path as the first directory for the library search path
Mamoru TASAKA reports:

> proxychains4 sets LD_PRELOAD to dlopen libproxychains4.so and execvp()
> the arbitrary command user has specified. proxychains4 sets the
> current directory as the first path to search libproxychains4.so


## Affected packages
 - **proxychains-ng**
    - **Fixed:** 4.9
