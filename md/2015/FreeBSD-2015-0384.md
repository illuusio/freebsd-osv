# openssl -- multiple vulnerabilities
OpenSSL project reports:

> 1.  BN_mod_exp may produce incorrect results on x86_64 (CVE-2015-3193)
> 2.  Certificate verify crash with missing PSS parameter
>     (CVE-2015-3194)
> 3.  X509_ATTRIBUTE memory leak (CVE-2015-3195)
> 4.  Race condition handling PSK identify hint (CVE-2015-3196)
> 5.  Anon DH ServerKeyExchange with 0 p parameter (CVE-2015-1794)


## Affected packages
 - **openssl**
    - **Fixed:** 1.0.2_5
 - **mingw32-openssl**
    - **Introduced:** 1.0.1
    - **Fixed:** 1.0.2e
 - **linux-c6-openssl**
    - **Fixed:** 1.0.1e_7
 - **FreeBSD**
    - **Introduced:** 10.2
    - **Fixed:** 10.2_8
    - **Introduced:** 10.1
    - **Fixed:** 10.1_25
    - **Introduced:** 9.3
    - **Fixed:** 9.3_31
