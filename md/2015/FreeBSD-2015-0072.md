# Dulwich -- Remote code execution
MITRE reports:

> Buffer overflow in the C implementation of the apply_delta function in
> \_pack.c in Dulwich before 0.9.9 allows remote attackers to execute
> arbitrary code via a crafted pack file.


## Affected packages
 - **py27-dulwich**
    - **Fixed:** 0.9.9
