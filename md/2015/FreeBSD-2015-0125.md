# strongswan -- Denial-of-service and potential remote code execution vulnerability
StrongSwan Project reports

> A denial-of-service and potential remote code execution vulnerability
> triggered by crafted IKE messages was discovered in strongSwan.
> Versions 5.2.2 and 5.3.0 are affected.


## Affected packages
 - **strongswan**
    - **Introduced:** 5.2.2
    - **Fixed:** 5.3.1
