# ansible -- multiple vulnerabilities
Ansible, Inc. reports:

> Ensure that hostnames match certificate names when using HTTPS -
> resolved in Ansible 1.9.2
>
> Improper symlink handling in zone, jail, and chroot connection plugins
> could lead to escape from confined environment - resolved in Ansible
> 1.9.2

 - **ansible**
    - **Fixed:** 1.9.2
