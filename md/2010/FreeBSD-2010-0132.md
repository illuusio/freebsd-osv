# phpMyAdmin -- XSS attack in database search
phpMyAdmin team reports:

> It was possible to conduct a XSS attack using spoofed request on the
> db search script.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 3.3.8.1
 - **phpMyAdmin211**
    - **Fixed:** 2.11.11.1
