# Zend Framework -- security issues in bundled Dojo library
The Zend Framework team reports:

> Several files in the bundled Dojo library were identified as having
> potential exploits, and the Dojo team also advised disabling or
> removing any PHP scripts in the Dojo library tree when deploying to
> production.


## Affected packages
 - **ZendFramework**
    - **Fixed:** 1.10.3
