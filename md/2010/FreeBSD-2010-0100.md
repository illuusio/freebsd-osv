# openx -- remote code execution vulnerability
The OpenX project reported:

> It has been brought to our attention that there is a vulnerability in
> the 2.8 downloadable version of OpenX that can result in a server
> running the downloaded version of OpenX being compromised.

This vulnerability exists in the file upload functionality and allows
attackers to upload and execute PHP code of their choice.


## Affected packages
 - **openx**
    - **Fixed:** 2.8.7
