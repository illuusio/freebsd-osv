# FreeBSD -- ZFS ZIL playback with insecure permissions
# Problem Description:

When replaying setattr transaction, the replay code would set the
attributes with certain insecure defaults, when the logged transaction
did not touch these attributes.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 7.1
    - **Fixed:** 7.1_10
    - **Introduced:** 7.2
    - **Fixed:** 7.2_6
    - **Introduced:** 8.0
    - **Fixed:** 8.0_2
