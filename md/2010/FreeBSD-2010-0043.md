# moodle -- multiple vulnerabilities
The Moodle release notes report multiple vulnerabilities which could
allow remote attackers to perform, amongst others, cross site scripting,
user enumeration and SQL injection attacks.


## Affected packages
 - **moodle**
    - **Fixed:** 1.9.8
