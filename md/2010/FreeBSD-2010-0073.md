# codeigniter -- file upload class vulnerability
Derek Jones reports:

> A fix has been implemented for a security flaw in CodeIgniter 1.7.2.
> All applications using the File Upload class should install the patch
> to ensure that their application is not subject to a vulnerability.


## Affected packages
 - **codeigniter**
    - **Fixed:** 1.7.2_1
