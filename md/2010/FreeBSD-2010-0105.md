# apr -- multiple vunerabilities
Secunia reports:

> Multiple vulnerabilities have been reported in APR-util, which can be
> exploited by malicious people to cause a DoS (Denial of Service).
>
> Two XML parsing vulnerabilities exist in the bundled version of expat.
>
> An error within the \"apr_brigade_split_line()\" function in
> buckets/apr_brigade.c can be exploited to cause high memory
> consumption.


## Affected packages
 - **apr1**
    - **Fixed:** 1.4.2.1.3.10
 - **apr0**
    - **Fixed:** 0.9.19.0.9.19
