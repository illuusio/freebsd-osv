# wireshark -- DOCSIS dissector denial of service
A vulnerability found in the DOCSIS dissector can cause Wireshark to
crash when a malformed packet trace file is opened. This means that an
attacker will have to trick a victim into opening such a trace file
before being able to crash the application


## Affected packages
 - **wireshark**
    - **Fixed:** 1.2.6_1
