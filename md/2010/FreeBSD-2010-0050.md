# piwik -- cross site scripting vulnerability
The Piwik security advisory reports:

> A non-persistent, cross-site scripting vulnerability (XSS) was found
> in Piwik\'s Login form that reflected the form_url parameter without
> being properly escaped or filtered.


## Affected packages
 - **piwik**
    - **Fixed:** 0.5.5
