# FreeBSD -- Insufficient environment sanitization in jail(8)
# Problem Description:

The jail(8) utility does not change the current working directory while
imprisoning. The current working directory can be accessed by its
descendants.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 8.0
    - **Fixed:** 8.0_3
