# wireshark -- LWRES vulnerability
Wireshark project reports:

> Babi discovered several buffer overflows in the LWRES dissector.
>
> It may be possible to make Wireshark crash remotely or by convincing
> someone to read a malformed packet trace file.


## Affected packages
 - **wireshark**
    - **Fixed:** 1.2.6
 - **wireshark-lite**
    - **Fixed:** 1.2.6
