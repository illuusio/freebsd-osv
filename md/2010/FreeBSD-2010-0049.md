# spamass-milter -- remote command execution vulnerability
The spamassassin milter plugin contains a vulnerability that can allow
remote attackers to execute commands on affected systems.

The vulnerability can be exploited trough a special-crafted email header
when the plugin was started with the \'-x\' (expand) flag.


## Affected packages
 - **spamass-milter**
    - **Fixed:** 0.3.1_8
