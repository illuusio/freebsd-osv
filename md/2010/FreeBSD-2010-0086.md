# corkscrew -- buffer overflow vulnerability
The affected corkscrew versions use sscanf calls without proper bounds
checking. In the authentication file parsing routine this can cause an
exploitable buffer overflow condition. A similar but issue exists in the
server response code but appears to be non-exploitable.


## Affected packages
 - **corkscrew**
    - **Fixed:** 2.0
