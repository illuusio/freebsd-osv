# FreeBSD -- Lost mbuf flag resulting in data corruption
# Problem Description:

The read-only flag is not correctly copied when a mbuf buffer reference
is duplicated. When the sendfile(2) system call is used to transmit data
over the loopback interface, this can result in the backing pages for
the transmitted file being modified, causing data corruption.

 - **FreeBSD**
