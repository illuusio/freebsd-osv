# FreeBSD -- Improper environment sanitization in rtld(1)
# Problem Description:

When running setuid programs rtld will normally remove potentially
dangerous environment variables. Due to recent changes in FreeBSD
environment variable handling code, a corrupt environment may result in
attempts to unset environment variables failing.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 7.1
    - **Fixed:** 7.1_9
    - **Introduced:** 7.2
    - **Fixed:** 7.2_5
    - **Introduced:** 8.0
    - **Fixed:** 8.0_1
