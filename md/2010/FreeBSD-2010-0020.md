# egroupware -- two vulnerabilities
Egroupware Team report:

> Nahuel Grisolia from CYBSEC S.A. Security Systems found two security
> problems in EGroupware:
>
> Serious remote command execution (allowing to run arbitrary command on
> the web server by simply issuing a HTTP request!).
>
> A reflected cross-site scripting (XSS).
>
> Both require NO valid EGroupware account and work without being logged
> in!


## Affected packages
 - **egroupware**
    - **Fixed:** 1.6.003
