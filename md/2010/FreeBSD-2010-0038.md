# pidgin -- multiple remote denial of service vulnerabilities
Three denial of service vulnerabilities where found in pidgin and allow
remote attackers to crash the application. The developers summarized
these problems as follows:

> Pidgin can become unresponsive when displaying large numbers of
> smileys

> Certain nicknames in group chat rooms can trigger a crash in Finch

> Failure to validate all fields of an incoming message can trigger a
> crash


## Affected packages
 - **pidgin**
    - **Fixed:** 2.6.6
 - **libpurple**
    - **Fixed:** 2.6.6
