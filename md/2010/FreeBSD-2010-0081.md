# vlc -- invalid id3v2 tags may lead to invalid memory dereferencing
VideoLAN project reports:

> VLC fails to perform sufficient input validation when trying to
> extract some meta-informations about input media through ID3v2 tags.
> In the failure case, VLC attempt dereference an invalid memory
> address, and a crash will ensure.


## Affected packages
 - **vlc**
    - **Introduced:** 0.9.0,3,1
    - **Fixed:** 1.1.2_1,3
