# isc-dhcp-server -- Empty link-address denial of service
ISC reports:

> If the server receives a DHCPv6 packet containing one or more
> Relay-Forward messages, and none of them supply an address in the
> Relay-Forward link-address field, then the server will crash. This can
> be used as a single packet crash attack vector.


## Affected packages
 - **isc-dhcp41-server**
    - **Introduced:** 4.1.0
    - **Fixed:** 4.1.2
