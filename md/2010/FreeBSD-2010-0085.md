# phpmyadmin -- Several XSS vulnerabilities
phpMyAdmin Team reports:

> It was possible to conduct a XSS attack using crafted URLs org POST
> parameters on several pages.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 3.3.5.1
 - **phpMyAdmin211**
    - **Fixed:** 2.11.10.1
