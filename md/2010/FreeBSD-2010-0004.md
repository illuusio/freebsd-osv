# dokuwiki -- multiple vulnerabilities
Dokuwiki reports:

> The plugin does no checks against cross-site request forgeries (CSRF)
> which can be exploited to e.g. change the access control rules by
> tricking a logged in administrator into visiting a malicious web site.

> The bug allows listing the names of arbitrary file on the webserver -
> not their contents. This could leak private information about wiki
> pages and server structure.


## Affected packages
 - **dokuwiki**
    - **Fixed:** 20091225_2
