# ziproxy -- atypical huge picture files vulnerability
Ziproxy 3.0.1 release fixes a security vulnerability related to atypical
huge picture files (\>4GB of size once expanded).


## Affected packages
 - **ziproxy**
    - **Fixed:** 3.0.1
