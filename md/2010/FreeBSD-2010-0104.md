# phpmyfaq -- cross site scripting vulnerabilities
The phpMyFAQ project reports:

> The phpMyFAQ Team has learned of a security issue that has been
> discovered in phpMyFAQ 2.6.x: phpMyFAQ doesn\'t sanitize some
> variables in different pages correctly. With a properly crafted URL it
> is e.g. possible to inject JavaScript code into the output of a page,
> which could result in the leakage of domain cookies (f.e. session
> identifiers)..


## Affected packages
 - **phpmyfaq**
    - **Fixed:** 2.6.9
