# bugzilla -- Cross Site Request Forgery
# A Bugzilla Security Advisory reports:

> Adobe does not properly restrict the SWF file format, which allows
> remote attackers to conduct cross-site request forgery (CSRF) attacks
> against Bugzilla\'s JSONP endpoint, possibly obtaining sensitive bug
> information, via a crafted OBJECT element with SWF content satisfying
> the character-set requirements of a callback API.
