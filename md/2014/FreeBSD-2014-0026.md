# file -- denial of service
The Fine Free file project reports:

> file before 5.17 allows context-dependent attackers to cause a denial
> of service (infinite recursion, CPU consumption, and crash) via a
> crafted indirect offset value in the magic of a file.


## Affected packages
 - **file**
    - **Fixed:** 5.17
