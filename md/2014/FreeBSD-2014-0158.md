# otrs -- Incomplete Access Control
The OTRS project reports:

> An attacker with valid OTRS credentials could access and manipulate
> ticket data of other users via the GenericInterface, if a ticket
> webservice is configured and not additionally secured.


## Affected packages
 - **otrs**
    - **Introduced:** 3.2.*,1
    - **Fixed:** 3.2.17
    - **Introduced:** 3.3.*,1
    - **Fixed:** 3.3.11
    - **Introduced:** 4.0.*,1
    - **Fixed:** 4.0.3
