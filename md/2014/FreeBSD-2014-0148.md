# OpenVPN -- denial of service security vulnerability
The OpenVPN project reports:

> In late November 2014 Dragana Damjanovic notified OpenVPN developers
> of a critical denial of service security vulnerability
> (CVE-2014-8104). The vulnerability allows an tls-authenticated client
> to crash the server by sending a too-short control channel packet to
> the server. In other words this vulnerability is denial of service
> only.


## Affected packages
 - **openvpn**
    - **Fixed:** 2.0.11
    - **Introduced:** 2.1.0
    - **Fixed:** 2.2.3
    - **Introduced:** 2.3.0
    - **Fixed:** 2.3.6
