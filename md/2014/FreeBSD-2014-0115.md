# krfb -- Multiple security issues in bundled libvncserver
Martin Sandsmark reports:

> krfb 4.14 \[and earlier\] embeds libvncserver which has had several
> security issues.
>
> Several remotely exploitable security issues have been uncovered in
> libvncserver, some of which might allow a remote authenticated user
> code execution or application crashes.


## Affected packages
 - **krfb**
    - **Fixed:** 4.12.5_4
