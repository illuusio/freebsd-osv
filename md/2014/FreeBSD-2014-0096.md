# gpgme -- heap-based buffer overflow in gpgsm status handler
Tomas Trnka reports:

> Gpgme contains a buffer overflow in the gpgsm status handler that
> could possibly be exploited using a specially crafted certificate.


## Affected packages
 - **gpgme**
    - **Fixed:** 1.5.0
