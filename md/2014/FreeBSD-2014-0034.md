# mutt -- denial of service, potential remote code execution
Beatrice Torracca and Evgeni Golov report:

> A buffer overflow has been discovered that could result in denial of
> service or potential execution of arbitrary code.
>
> This condition can be triggered by malformed RFC2047 header lines
