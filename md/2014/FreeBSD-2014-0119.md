# rsyslog -- remote syslog PRI vulnerability
The rsyslog project reports:

> potential abort when a message with PRI \> 191 was processed if the
> \"pri-text\" property was used in active templates, this could be
> abused to a remote denial of service from permitted senders
>
> The original fix for CVE-2014-3634 was not adequate.


## Affected packages
 - **rsyslog**
    - **Fixed:** 7.6.7
 - **rsyslog8**
    - **Fixed:** 8.4.2
