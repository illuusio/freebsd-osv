# phpMyAdmin -- Self-XSS due to unescaped HTML output in import.
The phpMyAdmin development team reports:

> When importing a file with crafted filename, it is possible to trigger
> an XSS. We consider this vulnerability to be non critical.


## Affected packages
 - **phpMyAdmin**
    - **Introduced:** 3.3.1
    - **Fixed:** 4.1.7
