# otrs -- Clickjacking issue
The OTRS Project reports:

> An attacker could embed OTRS in a hidden iframe tag of another page,
> tricking the user into clicking links in OTRS.


## Affected packages
 - **otrs**
    - **Fixed:** 3.1.21
    - **Introduced:** 3.2.*,1
    - **Fixed:** 3.2.16
    - **Introduced:** 3.3.*,1
    - **Fixed:** 3.3.6
