# strongswan -- multiple DoS vulnerabilities
strongSwan Project reports:

> A DoS vulnerability triggered by crafted IKEv1 fragmentation payloads
> was discovered in strongSwan\'s IKE daemon charon. All versions since
> 5.0.2 are affected.

> A DoS vulnerability and potential authorization bypass triggered by a
> crafted ID_DER_ASN1_DN ID payload was discovered in strongSwan. All
> versions since 4.3.3 are affected.

> A DoS vulnerability in strongSwan was discovered, which is triggered
> by XAuth usernames and EAP identities in versions 5.0.3 and 5.0.4.


## Affected packages
 - **strongswan**
    - **Fixed:** 5.1.1
