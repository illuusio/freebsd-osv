# postfixadmin -- SQL injection vulnerability
Thijs Kinkhorst reports:

> Postfixadmin has an SQL injection vulnerability. This vulnerability is
> only exploitable by authenticated users able to create new aliases.


## Affected packages
 - **postfixadmin**
    - **Fixed:** 2.3.7
