# subversion -- mod_dav_svn vulnerability
Subversion Project reports:

> Subversion\'s mod_dav_svn Apache HTTPD server module will crash when
> it receives an OPTIONS request against the server root and Subversion
> is configured to handle the server root and SVNListParentPath is on.
> This can lead to a DoS. There are no known instances of this problem
> being exploited in the wild, but the details of how to exploit it have
> been disclosed on the Subversion development mailing list.
