# bugzilla -- Social Engineering
# A Bugzilla Security Advisory reports:

> Dangerous control characters can be inserted into Bugzilla, notably
> into bug comments. If the text, which may look safe, is copied into a
> terminal such as xterm or gnome-terminal, then unexpected commands
> could be executed on the local machine.
