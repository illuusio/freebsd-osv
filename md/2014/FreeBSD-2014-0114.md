# bash -- remote code execution vulnerability
Chet Ramey reports:

> Under certain circumstances, bash will execute user code while
> processing the environment for exported function definitions.

The original fix released for CVE-2014-6271 was not adequate. A similar
vulnerability was discovered and tagged as CVE-2014-7169.
