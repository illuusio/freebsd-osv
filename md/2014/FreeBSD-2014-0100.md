# nginx -- inject commands into SSL session vulnerability
The nginx project reports:

> Security: pipelined commands were not discarded after STARTTLS command
> in SMTP proxy (CVE-2014-3556); the bug had appeared in 1.5.6.


## Affected packages
 - **nginx**
    - **Introduced:** 1.6.0,2
    - **Fixed:** 1.6.1,2
 - **nginx-devel**
    - **Introduced:** 1.5.6
    - **Fixed:** 1.7.4
