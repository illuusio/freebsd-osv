# OpenSSL -- Remote Data Injection / DoS
Applications that use SSL_MODE_RELEASE_BUFFERS, such as nginx, are prone
to a race condition which may allow a remote attacker to inject random
data into other connections.


## Affected packages
 - **openssl**
    - **Introduced:** 1.0.1
    - **Fixed:** 1.0.1_11
 - **mingw32-openssl**
    - **Introduced:** 1.0.1
    - **Fixed:** 1.0.1g
 - **FreeBSD**
    - **Introduced:** 10.0
    - **Fixed:** 10.0_2
