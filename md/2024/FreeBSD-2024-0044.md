# nginx-devel -- Multiple Vulnerabilities in HTTP/3
The nginx development team reports:

> When using HTTP/3 a segmentation fault might occur in a worker process
> while processing a specially crafted QUIC session.


## Affected packages
 - **nginx-devel**
    - **Introduced:** 1.25.0
    - **Fixed:** 1.25.4
