# element-web -- several vulnerabilities
Element team reports:

> Versions of Element Web and Desktop earlier than 1.11.85 do not check
> if thumbnails for attachments, stickers and images are coherent. It is
> possible to add thumbnails to events trigger a file download once
> clicked.

> A malicious homeserver can send invalid messages over federation which
> can prevent Element Web and Desktop from rendering single messages or
> the entire room containing them.


## Affected packages
 - **element-web**
    - **Fixed:** 1.11.85
