# minio -- unintentional information disclosure
Minio security advisory GHSA-95fr-cm4m-q5p9 reports:

> when used with anonymous requests by sending a random object name
> requests you can figure out if the object exists or not on the server
> on a specific bucket and also gain access to some amount of
> information.


## Affected packages
 - **minio**
    - **Fixed:** 2024.05.27.19.17.46
