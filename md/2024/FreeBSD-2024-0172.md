# Apache httpd -- Source code disclosure with handlers configured via AddType
The Apache httpd project reports:

> source code disclosure with handlers configured via AddType
> (CVE-2024-40725) (Important): A partial fix for CVE-2024-39884 in the
> core of Apache HTTP Server 2.4.61 ignores some use of the legacy
> content-type based configuration of handlers. \"AddType\" and similar
> configuration, under some circumstances where files are requested
> indirectly, result in source code disclosure of local content. For
> example, PHP scripts may be served instead of interpreted.


## Affected packages
 - **apache24**
    - **Introduced:** 2.4.60
    - **Fixed:** 2.4.62
