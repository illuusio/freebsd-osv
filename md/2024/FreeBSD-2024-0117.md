# PostgreSQL server -- Potentially allowing authenicated database users to see data that they shouldn't.
PostgreSQL project reports:

> A security vulnerability was found in the system views pg_stats_ext
> and pg_stats_ext_exprs, potentially allowing authenticated database
> users to see data they shouldn\'t. If this is of concern in your
> installation, run the SQL script
> /usr/local/share/postgresql/fix-CVE-2024-4317.sql for each of your
> databases. See the link for details.


## Affected packages
 - **postgresql-server**
    - **Fixed:** 16.3
    - **Fixed:** 15.7
    - **Fixed:** 14.12
