# librewolf -- Undefined behavior in selection node cache
security@mozilla.org reports:

> When manipulating the selection node cache, an attacker may have been
> able to cause unexpected behavior, potentially leading to an
> exploitable crash. This vulnerability affects Firefox \< 131.0.3.
