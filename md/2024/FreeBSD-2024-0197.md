# nginx -- Vulnerability in the ngx_http_mp4_module
The nginx development team reports:

> This update fixes the buffer overread vulnerability in the
> ngx_http_mp4_module.

 - **nginx-devel**
    - **Introduced:** 1.5.13,3
    - **Fixed:** 1.27.1,3
 - **nginx**
    - **Introduced:** 1.6.0,3
    - **Fixed:** 1.26.2,3
