# Gitlab -- vulnerabilities
Gitlab reports:

> Maintainer can leak Dependency Proxy password by changing Dependency
> Proxy URL via crafted POST request
>
> AI feature reads unsanitized content, allowing for attacker to hide
> prompt injection
>
> Project reference can be exposed in system notes


## Affected packages
 - **gitlab-ce**
    - **Introduced:** 17.4.0
    - **Fixed:** 17.4.1
    - **Introduced:** 17.3.0
    - **Fixed:** 17.3.4
    - **Introduced:** 15.6.0
    - **Fixed:** 17.2.8
 - **gitlab-ee**
    - **Introduced:** 17.4.0
    - **Fixed:** 17.4.1
    - **Introduced:** 17.3.0
