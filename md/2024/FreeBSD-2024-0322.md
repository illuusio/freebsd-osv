# py-matrix-synapse -- multiple vulnerabilities in versions prior to 1.120.1
element-hq/synapse developers report:

> \[The 1.120.1\] release fixes multiple security vulnerabilities, some
> affecting all prior versions of Synapse. Server administrators are
> encouraged to update Synapse as soon as possible. We are not aware of
> these vulnerabilities being exploited in the wild.
>
> Administrators who are unable to update Synapse may use the
> workarounds described in the linked GitHub Security Advisory below.


## Affected packages
 - **py38-matrix-synapse**
    - **Fixed:** 1.120.1
 - **py39-matrix-synapse**
    - **Fixed:** 1.120.1
 - **py310-matrix-synapse**
    - **Fixed:** 1.120.1
 - **py311-matrix-synapse**
    - **Fixed:** 1.120.1
