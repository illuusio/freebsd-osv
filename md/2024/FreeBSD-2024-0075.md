# phpmyfaq -- multiple vulnerabilities
phpMyFAQ team reports:

> The phpMyFAQ Team has learned of multiple security issues that\'d been
> discovered in phpMyFAQ 3.2.5 and earlier. phpMyFAQ contains cross-site
> scripting (XSS), SQL injection and bypass vulnerabilities.


## Affected packages
 - **phpmyfaq-php81**
    - **Fixed:** 3.2.6
 - **phpmyfaq-php82**
    - **Fixed:** 3.2.6
 - **phpmyfaq-php83**
    - **Fixed:** 3.2.6
