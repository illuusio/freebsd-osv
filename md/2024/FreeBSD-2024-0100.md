# ruby -- Arbitrary memory address read vulnerability with Regex search
sp2ip reports:

> If attacker-supplied data is provided to the Ruby regex compiler, it
> is possible to extract arbitrary heap data relative to the start of
> the text, including pointers and sensitive strings.
