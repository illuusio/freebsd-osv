# mozilla firefox -- protocol information guessing
security@mozilla.org reports:

> By monitoring the time certain operations take, an attacker could have
> guessed which external protocol handlers were functional on a user\'s
> system. This vulnerability affects Firefox \< 127, Firefox ESR \<
> 115.12, and Thunderbird \< 115.12.
