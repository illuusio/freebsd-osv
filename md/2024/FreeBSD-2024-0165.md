# traefik -- Bypassing IP allow-lists via HTTP/3 early data requests
The traefik authors report:

> There is a vulnerability in Traefik that allows bypassing IP
> allow-lists via HTTP/3 early data requests in QUIC 0-RTT handshakes
> sent with spoofed IP addresses.


## Affected packages
 - **traefik**
    - **Fixed:** 2.11.6
