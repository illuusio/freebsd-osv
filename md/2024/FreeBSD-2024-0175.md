# Mailpit -- Content Security Policy XSS
Mailpit developer reports:

> A vulnerability was discovered which allowed a bad actor with SMTP
> access to Mailpit to bypass the Content Security Policy headers using
> a series of crafted HTML messages which could result in a stored XSS
> attack via the web UI.


## Affected packages
 - **mailpit**
    - **Fixed:** 1.19.3
