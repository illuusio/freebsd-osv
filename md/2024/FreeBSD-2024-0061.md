# Gitlab -- Vulnerabilities
Gitlab reports:

> Bypassing CODEOWNERS approval allowing to steal protected variables
>
> Guest with manage group access tokens can rotate and see group access
> token with owner permissions


## Affected packages
 - **gitlab-ce**
    - **Introduced:** 16.9.0
    - **Fixed:** 16.9.2
    - **Introduced:** 16.8.0
    - **Fixed:** 16.8.4
    - **Introduced:** 11.3.0
    - **Fixed:** 16.7.7
