# libqb -- Buffer overflow
cve@mitre.org reports:

> log_blackbox.c in libqb before 2.0.8 allows a buffer overflow via long
> log messages because the header size is not considered.


## Affected packages
 - **libqb**
    - **Fixed:** 2.0.8
