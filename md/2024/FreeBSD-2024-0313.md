# gstreamer1-plugins-ogg -- Out-of-bounds write in Ogg demuxer
The GStreamer Security Center reports:

> An out-of-bounds write in the Ogg demuxer that can cause crashes for
> certain input files.


## Affected packages
 - **gstreamer1-plugins-ogg**
    - **Fixed:** 1.24.10
