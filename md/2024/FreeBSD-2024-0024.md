# OpenSSL -- Multiple vulnerabilities
The OpenSSL project reports:

> Excessive time spent checking invalid RSA public keys (CVE-2023-6237)
>
> PKCS12 Decoding crashes (CVE-2024-0727)


## Affected packages
 - **openssl**
    - **Fixed:** 3.0.13,1
 - **openssl-quictls**
    - **Fixed:** 3.0.13
 - **openssl31**
    - **Fixed:** 3.1.5
 - **openssl31-quictls**
    - **Fixed:** 3.1.5
 - **openssl32**
    - **Fixed:** 3.2.1
