# OpenSSL -- Use after free vulnerability
The OpenSSL project reports:

> Use After Free with SSL_free_buffers (low).
>
> Calling the OpenSSL API function SSL_free_buffers may cause memory to
> be accessed that was previously freed in some situations


## Affected packages
 - **openssl**
    - **Fixed:** 3.0.13_5,1
 - **openssl31**
    - **Fixed:** 3.1.5_5
 - **openssl32**
    - **Fixed:** 3.2.1_5
 - **openssl33**
    - **Fixed:** 3.3.0_2
 - **openssl-quictls**
    - **Fixed:** 3.0.13_5
 - **openssl31-quictls**
    - **Fixed:** 3.1.5_5
