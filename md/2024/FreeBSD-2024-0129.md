# Roundcube -- Cross-site scripting vulnerabilities
The Roundcube project reports:

> cross-site scripting (XSS) vulnerability in handling SVG animate
> attributes.
>
> cross-site scripting (XSS) vulnerability in handling list columns from
> user preferences.


## Affected packages
 - **roundcube**
    - **Fixed:** 1.6.7,1
