# OpenHAB CometVisu addon -- Multiple vulnerabilities
OpenHAB reports:

> This patch release addresses the following security advisories:
>
> - SSRF/XSS (CometVisu) -
>   [GHSA-v7gr-mqpj-wwh3](https://github.com/openhab/openhab-webui/security/advisories/GHSA-v7gr-mqpj-wwh3)
> - Sensitive information disclosure (CometVisu) -
>   [GHSA-3g4c-hjhr-73rj](https://github.com/openhab/openhab-webui/security/advisories/GHSA-3g4c-hjhr-73rj)
> - RCE through path traversal (CometVisu) -
>   [GHSA-f729-58x4-gqgf](https://github.com/openhab/openhab-webui/sec%20urity/advisories/GHSA-f729-58x4-gqgf)
> - Path traversal (CometVisu) -
>   [GHSA-pcwp-26pw-j98w](https://github.com/openhab/openhab-webui/security/advisories/GHSA-pcwp-26pw-j98w)
>
> All of these are related to the CometVisu add-on for openHAB - if you
> are a user of CometVisu, we strongly recommend to upgrade your system
> to openHAB 4.2.1 in order to fix those vulnerabilities.


## Affected packages
 - **openhab-addons**
    - **Fixed:** 4.2.1
