# OpenSSL -- Vector register corruption on PowerPC
The OpenSSL Team reports:

> The POLY1305 MAC (message authentication code) implementation contains
> a bug that might corrupt the internal state of applications running on
> PowerPC CPU based platforms if the CPU provides vector instructions.


## Affected packages
 - **openssl**
    - **Fixed:** 3.0.12_2,1
 - **openssl-quictls**
    - **Fixed:** 3.0.12_2
 - **openssl31**
    - **Fixed:** 3.1.4_2
 - **openssl31-quictls**
    - **Fixed:** 3.1.4_2
 - **openssl32**
    - **Fixed:** 3.2.0_1
