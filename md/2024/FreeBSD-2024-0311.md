# gstreamer1-plugins-gdkpixbuf -- NULL-pointer dereference
The GStreamer Security Center reports:

> A NULL-pointer dereference in the gdk-pixbuf decoder that can cause
> crashes for certain input files.


## Affected packages
 - **gstreamer1-plugins-gdkpixbuf**
    - **Fixed:** 1.24.10
