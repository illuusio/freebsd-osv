# RabbitMQ-C -- auth credentials visible in commandline tool options
hadmut reports:

> This C library includes 2 command-line tools that can take credentials
> as command-line options. The credentials are exposed as plain-text in
> the process list. This could allow an attacker with access to the
> process list to see the credentials.


## Affected packages
 - **rabbitmq-c**
    - **Fixed:** 0.14.0
