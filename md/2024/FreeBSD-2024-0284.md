# Matrix clients -- mxc uri validation in js sdk
matrix-js-sdk upstream reports:

> matrix-js-sdk before 34.11.0 is vulnerable to client-side path
> traversal via crafted MXC URIs. A malicious room member can trigger
> clients based on the matrix-js-sdk to issue arbitrary authenticated
> GET requests to the client\'s homeserver.


## Affected packages
 - **cinny**
    - **Fixed:** 4.2.3
 - **element-web**
    - **Fixed:** 1.11.85
