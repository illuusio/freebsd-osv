# vscode -- Visual Studio Code for Linux Remote Code Execution Vulnerability
VSCode developers report:

> Visual Studio Code for Linux Remote Code Execution Vulnerability
>
> A remote code execution vulnerability exists in VS Code 1.94.0 and
> earlier versions in the elevated save flow.


## Affected packages
 - **vscode**
    - **Fixed:** 1.94.1
