# OpenSSL -- Unbounded memory growth with session handling in TLSv1.3
The OpenSSL project reports:

> Some non-default TLS server configurations can cause unbounded memory
> growth when processing TLSv1.3 sessions


## Affected packages
 - **openssl**
    - **Fixed:** 3.0.13_3,1
 - **openssl31**
    - **Fixed:** 3.1.5_3
 - **openssl32**
    - **Fixed:** 3.2.1_2
 - **openssl-quictls**
    - **Fixed:** 3.0.13_3
 - **openssl31-quictls**
    - **Fixed:** 3.1.5_1
