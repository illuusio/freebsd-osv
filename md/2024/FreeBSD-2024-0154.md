# emacs -- Arbitrary shell code evaluation vulnerability
GNU Emacs developers report:

> Emacs 29.4 is an emergency bugfix release intended to fix a security
> vulnerability. Arbitrary shell commands are no longer run when turning
> on Org mode in order to avoid running malicious code.


## Affected packages
 - **emacs**
    - **Fixed:** 29.3_3,3
 - **emacs-canna**
    - **Fixed:** 29.3_3,3
 - **emacs-nox**
    - **Fixed:** 29.3_3,3
 - **emacs-wayland**
    - **Fixed:** 29.3_3,3
 - **emacs-devel**
    - **Fixed:** 30.0.50.20240615_1,3
 - **emacs-devel-nox**
    - **Fixed:** 30.0.50.20240615_1,3
