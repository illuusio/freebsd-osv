# rclone -- Multiple vulnerabilities
Multiple vulnerabilities in ssh and golang

> - CVE-2023-45286: HTTP request body disclosure in go-resty disclosure
>   across requests.
> - CVE-2023-48795: The SSH transport protocol with certain OpenSSH
>   extensions, found in OpenSSH before 9.6 and other products, allows
>   remote attackers to bypass integrity checks.
