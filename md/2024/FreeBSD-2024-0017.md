# TinyMCE -- mXSS in multiple plugins
TinyMCE reports:

> Special characters in unescaped text nodes can trigger mXSS when using
> TinyMCE undo/redo, getContentAPI, resetContentAPI, and Autosave plugin


## Affected packages
 - **tinymce**
    - **Fixed:** 6.7.3
 - **roundcube**
    - **Fixed:** 1.6.6,1
