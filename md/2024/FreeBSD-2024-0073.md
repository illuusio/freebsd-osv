# quiche -- Multiple Vulnerabilities
Quiche Releases reports:

> This release includes 2 security fixes:
>
> - CVE-2024-1410: Unbounded storage of information related to
>   connection ID retirement, in quiche. Reported by Marten Seeman
>   (@marten-seeman)
> - CVE-2024-1765: Unlimited resource allocation by QUIC CRYPTO frames
>   flooding in quiche. Reported by Marten Seeman (@marten-seeman)
