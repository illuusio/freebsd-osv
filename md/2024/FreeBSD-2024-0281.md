# x11vnc -- access to shared memory segments
cve@mitre.org reports:

> scan.c in x11vnc 0.9.16 uses IPC_CREAT\|0777 in shmget calls, which
> allows access by actors other than the current user.


## Affected packages
 - **x11vnc**
    - **Fixed:** 0.9.16_8
