# jenkins -- XSS vulnerability
Jenkins Security Advisory:

> # Description
>
> ##### (High) SECURITY-2886 / CVE-2022-41224
>
> Jenkins 2.367 through 2.369 (both inclusive) does not escape tooltips
> of the l:helpIcon UI component used for some help icons on the Jenkins
> web UI.
>
> This results in a stored cross-site scripting (XSS) vulnerability
> exploitable by attackers able to control tooltips for this component.
>
> Jenkins 2.370 escapes tooltips of the l:helpIcon UI component.


## Affected packages
 - **jenkins**
    - **Fixed:** 2.370
