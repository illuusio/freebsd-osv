# polkit -- Local Privilege Escalation
Qualys reports:

> We discovered a Local Privilege Escalation (from any user to root) in
> polkit\'s pkexec, a SUID-root program that is installed by default on
> every major Linux distribution.


## Affected packages
 - **polkit**
    - **Fixed:** 0.120_1
