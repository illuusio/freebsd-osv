# tailscale -- Security vulnerability in the client
Tailscale team reports:

> A vulnerability identified in the Tailscale client allows a malicious
> website to access the peer API, which can then be used to access
> Tailscale environment variables.


## Affected packages
 - **tailscale**
    - **Fixed:** 1.32.3
