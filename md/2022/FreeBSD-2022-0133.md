# git -- privilege escalation
The git project reports:

> Git is vulnerable to privilege escalation in all platforms. An
> unsuspecting user could still be affected by the issue reported in
> CVE-2022-24765, for example when navigating as root into a shared tmp
> directory that is owned by them, but where an attacker could create a
> git repository.


## Affected packages
 - **git**
    - **Fixed:** 2.37.1
