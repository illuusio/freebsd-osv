# typo3 -- XSS vulnerability in svg-sanitize
The TYPO3 project reports:

> The SVG sanitizer library enshrined/svg-sanitize before version 0.15.0
> did not remove HTML elements wrapped in a CDATA section. As a result,
> SVG content embedded in HTML (fetched as text/html) was susceptible to
> cross-site scripting. Plain SVG files (fetched as image/svg+xml) were
> not affected.
