# FreeBSD -- zlib compression out-of-bounds write
# Problem Description:

Certain inputs can cause zlib\'s compression routine to overwrite an
internal buffer with compressed data. This issue may require the use of
uncommon or non-default compression parameters.

# Impact:

The out-of-bounds write may result in memory corruption and an
application crash or kernel panic.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 13.0
    - **Fixed:** 13.0_11
    - **Introduced:** 12.3
    - **Fixed:** 12.3_5
