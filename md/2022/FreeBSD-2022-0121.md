# OpenSSL -- Command injection vulnerability
The OpenSSL project reports:

> Circumstances where the c_rehash script does not properly sanitise
> shell metacharacters to prevent command injection were found by code
> review.


## Affected packages
 - **openssl**
    - **Fixed:** 1.1.1p,1
 - **openssl-devel**
    - **Fixed:** 3.0.4
 - **openssl-quictls**
    - **Fixed:** 3.0.4
