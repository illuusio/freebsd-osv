# xrdp -- privilege escalation
xrdp project reports:

> An integer underflow leading to a heap overflow in the sesman server
> allows any unauthenticated attacker which is accessible to a sesman
> server (listens by default on localhost when installing xrdp, but can
> be remote if configured otherwise) to execute code as root.


## Affected packages
 - **xrdp**
    - **Introduced:** 0.9.17,1
    - **Fixed:** 0.9.18.1,1
 - **xrdp-devel**
    - **Introduced:** 0.9.17,1
    - **Fixed:** 0.9.18.1,1
