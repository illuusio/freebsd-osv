# mustache - Possible Remote Code Execution
huntr.dev reports:

> In Mustache.php v2.0.0 through v2.14.0, Sections tag can lead to
> arbitrary php code execution even if strict_callables is true when
> section value is controllable.


## Affected packages
 - **phpmustache**
    - **Fixed:** 2.14.1
