# MySQL -- Multiple vulnerabilities
Oracle reports:

> This Critical Patch Update contains 37 new security patches for Oracle
> MySQL. 11 of these vulnerabilities may be remotely exploitable without
> authentication, i.e., may be exploited over a network without
> requiring user credentials


## Affected packages
 - **mysql-connector-c++**
    - **Fixed:** 8.0.31
 - **mysql-connector-odbc**
    - **Fixed:** 8.0.31
 - **mysql-client57**
    - **Fixed:** 5.7.40
 - **mysql-server57**
    - **Fixed:** 5.7.40
 - **mysql-client80**
    - **Fixed:** 8.0.31
 - **mysql-server80**
    - **Fixed:** 8.0.31
