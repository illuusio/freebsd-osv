# FreeBSD -- 802.11 heap buffer overflow
# Problem Description:

The 802.11 beacon handling routine failed to validate the length of an
IEEE 802.11s Mesh ID before copying it to a heap-allocated buffer.

# Impact:

While a FreeBSD Wi-Fi client is in scanning mode (i.e., not associated
with a SSID) a malicious beacon frame may overwrite kernel memory,
leading to remote code execution.


## Affected packages
 - **FreeBSD-kernel**
    - **Introduced:** 13.0
    - **Fixed:** 13.0_11
    - **Introduced:** 12.3
    - **Fixed:** 12.3_5
