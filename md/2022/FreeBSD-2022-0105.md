# go -- syscall.Faccessat checks wrong group on Linux
The Go project reports:

> When called with a non-zero flags parameter, the syscall.Faccessat
> function could incorrectly report that a file is accessible. This bug
> only occurs on Linux systems.


## Affected packages
 - **go**
    - **Fixed:** 1.18.2,1
 - **go117**
    - **Fixed:** 1.17.10
