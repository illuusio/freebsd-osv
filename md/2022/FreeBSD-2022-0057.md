# openvpn -- Potential authentication by-pass with multiple deferred authentication plug-ins
David Sommerseth reports:

> OpenVPN 2.1 until v2.4.12 and v2.5.6 may enable authentication bypass
> in external authentication plug-ins when more than one of them makes
> use of deferred authentication replies, which allows an external user
> to be granted access with only partially correct credentials. This
> issue is resolved in OpenVPN 2.4.12 and v2.5.6.


## Affected packages
 - **openvpn**
    - **Fixed:** 2.5.6
 - **openvpn-mbedtls**
    - **Fixed:** 2.5.6
