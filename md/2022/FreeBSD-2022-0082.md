# Composer -- Command injection vulnerability
Composer developers reports:

> The Composer method VcsDriver::getFileContent() with user-controlled
> \$file or \$identifier arguments is susceptible to an argument
> injection vulnerability. It can be leveraged to gain arbitrary command
> execution if the Mercurial or the Git driver are used.
