# ipython -- Execution with Unnecessary Privileges
IPython project reports:

> IPython 8.0.1, 7.31.1 and 5.11 are security releases that change some
> default values in order to prevent potential Execution with
> Unnecessary Privileges.


## Affected packages
 - **py37-ipython**
    - **Fixed:** 7.31.1
 - **py38-ipython**
    - **Fixed:** 7.31.1
 - **py39-ipython**
    - **Fixed:** 7.31.1
 - **py310-ipython**
    - **Fixed:** 7.31.1
 - **py311-ipython**
    - **Fixed:** 7.31.1
