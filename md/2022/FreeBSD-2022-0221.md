# Grafana -- Data source and plugin proxy endpoints leaking authentication tokens to some destination plugins
Grafana Labs reports:

> On September 7th as a result of an internal security audit we have
> discovered that Grafana could leak the authentication cookie of users
> to plugins. After further analysis the vulnerability impacts data
> source and plugin proxy endpoints under certain conditions.
>
> We believe that this vulnerability is rated at CVSS 6.8
> (CVSS:3.1/AV:N/AC:L/PR:H/UI:R/S:U/C:H/I:H/A:H)


## Affected packages
 - **grafana**
    - **Introduced:** 5.0.0
    - **Fixed:** 8.5.14
    - **Introduced:** 9.0.0
    - **Fixed:** 9.1.8
 - **grafana7**
    - **Introduced:** 7.0.0
 - **grafana8**
    - **Introduced:** 8.0.0
    - **Fixed:** 8.5.14
 - **grafana9**
    - **Introduced:** 9.0.0
    - **Fixed:** 9.1.8
