# darkhttpd -- DOS vulnerability
Mitre reports:

> flaw was found in darkhttpd. Invalid error handling allows remote
> attackers to cause denial-of-service by accessing a file with a large
> modification date. The highest threat from this vulnerability is to
> system availability.
