# mpg123 -- buffer overflow vulnerability
Yuri D\'Elia has found a buffer overflow vulnerability in mpg123\'s
parsing of frame headers in input streams. This vulnerability can
potentially lead to execution of arbitrary code with the permissions of
the user running mpg123, if the user runs mpg123 on a specially crafted
MP2 or MP3 file.


## Affected packages
 - **mpg123**
    - **Fixed:** 0.59r_17
 - **mpg123-nas**
    - **Fixed:** 0.59r_17
 - **mpg123-esound**
    - **Fixed:** 0.59r_17
