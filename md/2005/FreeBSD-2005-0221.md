# drupal -- PHP code execution vulnerabilities
Kuba Zygmunt discovered a flaw in the input validation routines of
Drupal\'s filter mechanism. An attacker could execute arbitrary PHP code
on a target site when public comments or postings are allowed.


## Affected packages
 - **drupal**
    - **Fixed:** 4.6.2
