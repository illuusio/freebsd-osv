# mkbold-mkitalic -- format string vulnerability
The version 0.06_1 and prior have a format string vulnerability which
can be triggered by using a carefully-crafted BDF font file.


## Affected packages
 - **mkbold-mkitalic**
    - **Fixed:** 0.07
