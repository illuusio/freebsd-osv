# xtrlock -- X display locking bypass
The X display locking program `xtrlock` contains an integer overflow
bug. It is possible for an attacker with physical access to the system
to bypass the display lock.


## Affected packages
 - **xtrlock**
    - **Fixed:** 2.0.10
