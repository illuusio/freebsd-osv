# horde -- Cross site scripting vulnerabilities in several of Horde's templates
Announce of Horde H3 3.0.8 (final):

> This \[3.0.8\] is a security release that fixes cross site scripting
> vulnerabilities in several of Horde\'s templates. None of the
> vulnerabilities can be exploited by unauthenticated users; however, we
> strongly recommend that all users of Horde 3.0.7 upgrade to 3.0.8 as
> soon as possible.


## Affected packages
 - **horde**
    - **Fixed:** 3.0.8
 - **horde-php5**
    - **Fixed:** 3.0.8
