# xli -- integer overflows in image size calculations
Tavis Ormandy discovered several integer overflows in xli\'s image size
handling. A maliciously crafted image may be able to cause a heap buffer
overflow and execute arbitrary code.


## Affected packages
 - **xli**
    - **Fixed:** 1.17.0_1
