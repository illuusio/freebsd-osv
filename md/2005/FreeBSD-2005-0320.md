# kronolith -- Cross site scripting vulnerabilities in several of the calendar name and event data fields
Announce of Kronolith H3 (2.0.6) (final):

> This \[2.0.6\] is a security release that fixes cross site scripting
> vulnerabilities in several of the calendar name and event data fields.
> None of the vulnerabilities can be exploited by unauthenticated users;
> however, we strongly recommend that all users of Kronolith 2.0.5
> upgrade to 2.0.6 as soon as possible.


## Affected packages
 - **kronolith**
    - **Fixed:** 2.0.6
