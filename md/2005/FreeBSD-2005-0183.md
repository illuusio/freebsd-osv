# acroread -- XML External Entity vulnerability
Sverre H. Huseby discovered a vulnerability in Adobe Acrobat and Adobe
Reader. Under certain circumstances, using XML scripts it is possible to
discover the existence of local files.


## Affected packages
 - **acroread7**
    - **Introduced:** 7.0.0
    - **Fixed:** 7.0.1
 - **ja-acroread**
    - **Introduced:** 7.0.0
    - **Fixed:** 7.0.1
