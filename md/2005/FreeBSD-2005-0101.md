# libexif -- buffer overflow vulnerability
Sylvain Defresne reports that libexif is vulnerable to a buffer overflow
vulnerability due to insufficient input checking. This could lead crash
of applications using libexif.


## Affected packages
 - **libexif**
    - **Fixed:** 0.6.10_1
