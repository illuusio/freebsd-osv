# gforge -- directory traversal vulnerability
An STG Security Advisory reports:

> GForge CVS module made by Dragos Moinescu and another module made by
> Ronald Petty have a directory traversal vulnerability. \[\...\]
> malicious attackers can read arbitrary directory lists.


## Affected packages
 - **gforge**
    - **Fixed:** 4.0
