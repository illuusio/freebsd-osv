# mailman -- directory traversal vulnerability
A directory traversal vulnerability in mailman allow remote attackers to
read arbitrary files due to inadequate input sanitizing. This could,
among other things, lead remote attackers to gaining access to the
mailman configuration database (which contains subscriber email
addresses and passwords) or to the mail archives for private lists.


## Affected packages
 - **mailman**
    - **Fixed:** 2.1.5_2
 - **ja-mailman**
    - **Fixed:** 2.1.5_2
