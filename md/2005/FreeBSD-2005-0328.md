# nbd-server -- buffer overflow vulnerability
Kurt Fitzner reports a buffer overflow vulnerability within nbd. This
could potentially allow the execution of arbitrary code on the nbd
server.


## Affected packages
 - **nbd-server**
    - **Fixed:** 2.8.2_1
