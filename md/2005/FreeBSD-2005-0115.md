# clamav -- zip handling DoS vulnerability
The clamav daemon is vulnerable to a DoS vulnerability due to
insufficient handling of malformed zip files which can crash the clamav
daemon.


## Affected packages
 - **clamav**
    - **Fixed:** 0.81
 - **clamav-devel**
    - **Fixed:** 20050408
