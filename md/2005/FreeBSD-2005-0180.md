# gaim -- Yahoo! remote crash vulnerability
Jacopo Ottaviani reports that Gaim can be crashed by being offered files
with names containing non-ASCII characters via the Yahoo! protocol.


## Affected packages
 - **gaim**
    - **Fixed:** 1.3.1
 - **ja-gaim**
    - **Fixed:** 1.3.1
 - **ko-gaim**
    - **Fixed:** 1.3.1
 - **ru-gaim**
    - **Fixed:** 1.3.1
