# unrtf -- buffer overflow vulnerability
Yosef Klein and Limin Wang have found a buffer overflow vulnerability in
unrtf that can allow an attacker to execute arbitrary code with the
permissions of the user running unrtf, by running unrtf on a specially
crafted rtf document.


## Affected packages
 - **unrtf**
    - **Fixed:** 0.19.3_1
