# pcal -- buffer overflow vulnerabilities
Danny Lungstrom has found two buffer overflow vulnerabilities in pcal
which can lead to execution of arbitrary code by making a user run pcal
on a specially crafted calendar file.


## Affected packages
 - **pcal**
    - **Fixed:** 4.8.0
