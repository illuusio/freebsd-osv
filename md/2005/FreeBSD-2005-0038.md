# cups-base -- CUPS server remote DoS vulnerability
Kenshi Muto discovered that the CUPS server would enter an infinite loop
when processing a URL containing "`/..`".


## Affected packages
 - **cups-base**
    - **Introduced:** 1.1.21
    - **Fixed:** 1.1.23
