# groff -- pic2graph and eqn2graph are vulnerable to symlink attack through temporary files
The eqn2graph and pic2graph scripts in groff 1.18.1 allow local users to
overwrite arbitrary files via a symlink attack on temporary files.


## Affected packages
 - **ja-groff**
    - **Introduced:** 1.18.1
    - **Fixed:** 1.18.1_8
