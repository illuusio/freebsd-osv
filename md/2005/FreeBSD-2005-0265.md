# urban -- stack overflow vulnerabilities
Several filename-related stack overflow bugs allow a local attacker to
elevate its privileges to the games group, since urban is installed
setgid games.

Issue discovered and fixed by \<shaun@rsc.cx\>.


## Affected packages
 - **urban**
    - **Fixed:** 1.5.3_2
