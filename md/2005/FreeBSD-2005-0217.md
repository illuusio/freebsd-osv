# phpbb -- multiple vulnerabilities
phpBB is vulnerable to remote exploitation of an input validation
vulnerability allows attackers to read the contents of arbitrary system
files under the privileges of the webserver. This also allows remote
attackers to unlink arbitrary system files under the privileges of the
webserver.


## Affected packages
 - **phpbb**
    - **Fixed:** 2.0.12
