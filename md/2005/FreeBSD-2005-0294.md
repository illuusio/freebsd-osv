# fetchmail -- fetchmailconf local password exposure
The fetchmail team reports:

> The fetchmailconf program before and excluding version 1.49 opened the
> run control file, wrote the configuration to it, and only then changed
> the mode to 0600 (rw\-\-\-\-\-\--). Writing the file, which usually
> contains passwords, before making it unreadable to other users, can
> expose sensitive password information.


## Affected packages
 - **fetchmail**
    - **Fixed:** 6.2.5.2_1
