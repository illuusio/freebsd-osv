# nag -- Cross site scripting vulnerabilities in several of the tasklist name and task data fields
Announce of Nag H3 (2.0.4) (final):

> This \[2.0.4\] is a security release that fixes cross site scripting
> vulnerabilities in several of the tasklist name and task data fields.
> None of the vulnerabilities can be exploited by unauthenticated users;
> however, we strongly recommend that all users of Nag 2.0.3 upgrade to
> 2.0.4 as soon as possible.


## Affected packages
 - **nag**
    - **Fixed:** 2.0.4
