# opera -- kfmclient exec command execution vulnerability
Giovanni Delvecchio reports:

> Opera for linux uses \"kfmclient exec\" as \"Default Application\" to
> handle saved files. This could be used by malicious remote users to
> execute arbitrary shell commands on a target system.


## Affected packages
 - **opera**
    - **Fixed:** 7.54.20050131
 - **opera-devel**
    - **Fixed:** 7.54.20050131
 - **linux-opera**
    - **Fixed:** 7.54.20050131
