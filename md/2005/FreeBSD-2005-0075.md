# emacs -- movemail format string vulnerability
Max Vozeler discovered several format string vulnerabilities in the
movemail utility of Emacs. They can be exploited when connecting to a
malicious POP server and can allow an attacker can execute arbitrary
code under the privileges of the user running Emacs.
