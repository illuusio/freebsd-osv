# cups-base -- HPGL buffer overflow vulnerability
Ariel Berkman has discovered a buffer overflow vulnerability in CUPS\'s
HPGL input driver. This vulnerability could be exploited to execute
arbitrary code with the permission of the CUPS server by printing a
specially crated HPGL file.


## Affected packages
 - **cups-base**
    - **Fixed:** 1.1.22.0_2
 - **fr-cups-base**
    - **Fixed:** 1.1.22.0_2
