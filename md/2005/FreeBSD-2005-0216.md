# shtool -- insecure temporary file creation
A Zataz advisory reports that shtool contains a security flaw which
could allow a malicious local user to create or overwrite the contents
of arbitrary files. The attacker could fool a user into executing the
arbitrary file possibly executing arbitrary code.


## Affected packages
 - **shtool**
    - **Fixed:** 2.0.1
