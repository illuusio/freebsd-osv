# imap-uw -- authentication bypass when CRAM-MD5 is enabled
The CRAM-MD5 authentication support of the University of Washington IMAP
and POP3 servers contains a vulnerability that may allow an attacker to
bypass authentication and impersonate arbitrary users. Only
installations with CRAM-MD5 support configured are affected.


## Affected packages
 - **imap-uw**
    - **Fixed:** 2004b,1
