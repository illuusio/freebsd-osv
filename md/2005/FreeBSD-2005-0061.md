# newsfetch -- server response buffer overflow vulnerability
The newsfetch program uses the sscanf function to read information from
server responses into static memory buffers. Unfortunately this is done
without any proper bounds checking. As a result long server responses
may cause an overflow when a newsgroup listing is requested from an NNTP
server.


## Affected packages
 - **newsfetch**
    - **Fixed:** 1.21_1
