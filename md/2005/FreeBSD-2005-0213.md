# ekg -- insecure temporary file creation
Eric Romang reports that ekg creates temporary files in an insecure
manner. This can be exploited by an attacker using a symlink attack to
overwrite arbitrary files and possibly execute arbitrary commands with
the permissions of the user running ekg.


## Affected packages
 - **pl-ekg**
    - **Fixed:** 1.6r2,1
