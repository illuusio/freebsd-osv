# phpmyadmin -- XSS vulnerabilities
A phpMyAdmin security advisory reports:

> It was possible to conduct an XSS attack via the HTTP_HOST variable;
> also, some scripts in the libraries directory that handle header
> generation were vulnerable to XSS.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.7.0
