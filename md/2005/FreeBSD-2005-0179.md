# gaim -- MSN Remote DoS vulnerability
The GAIM team reports:

> Remote attackers can cause a denial of service (crash) via a malformed
> MSN message that leads to a memory allocation of a large size,
> possibly due to an integer signedness error.


## Affected packages
 - **gaim**
    - **Fixed:** 1.3.1
 - **ja-gaim**
    - **Fixed:** 1.3.1
 - **ko-gaim**
    - **Fixed:** 1.3.1
 - **ru-gaim**
    - **Fixed:** 1.3.1
