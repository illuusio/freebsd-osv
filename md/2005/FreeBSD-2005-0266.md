# bind -- buffer overrun vulnerability
An ISC advisory reports a buffer overrun vulnerability within bind. The
vulnerability could result in a Denial of Service. A workaround is
available by disabling recursion and glue fetching.


## Affected packages
 - **bind84**
    - **Introduced:** 8.4.4
    - **Fixed:** 8.4.6
