# grip -- CDDB response multiple matches buffer overflow vulnerability
Joseph VanAndel reports that grip is vulnerability to a buffer overflow
vulnerability when receiving more than 16 CDDB responses. This could
lead to a crash in grip and potentially execution arbitrary code.

A workaround is to disable CDDB lookups.


## Affected packages
 - **grip**
    - **Fixed:** 3.2.0_7
