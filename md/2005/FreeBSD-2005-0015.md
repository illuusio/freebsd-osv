# tiff -- tiffdump integer overflow vulnerability
Dmitry V. Levin found a potential integer overflow in the tiffdump
utility which could lead to execution of arbitrary code. This could be
exploited by tricking an user into executing tiffdump on a specially
crafted tiff image.


## Affected packages
 - **tiff**
    - **Fixed:** 3.7.1_2
 - **linux-tiff**
    - **Fixed:** 3.6.1_1
