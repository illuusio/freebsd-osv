# gaim -- jabber remote crash
The GAIM team reports:

> A remote jabber user can cause Gaim to crash by sending a specific
> file transfer request.


## Affected packages
 - **gaim**
    - **Fixed:** 1.2.1
 - **ja-gaim**
    - **Fixed:** 1.2.1
 - **ko-gaim**
    - **Fixed:** 1.2.1
 - **ru-gaim**
    - **Fixed:** 1.2.1
