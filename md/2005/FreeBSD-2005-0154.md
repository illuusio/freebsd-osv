# oops -- format string vulnerability
A RST/GHC Advisory reports that there is an format string vulnerability
in oops. The vulnerability can be found in the MySQL/PgSQL
authentication module. Succesful exploitation may allow execution of
arbitrary code.


## Affected packages
 - **oops**
    - **Fixed:** 1.5.24
