# phpmyadmin -- HTTP Response Splitting vulnerability
A phpMyAdmin security advisory reports:

> Some scripts in phpMyAdmin are vulnerable to an HTTP Response
> Splitting attack.
>
> Severity:
>
> We consider these vulnerabilities to be serious. However, they can
> only be triggered on systems running with register_globals = on.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.6.4.4
