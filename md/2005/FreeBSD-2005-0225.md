# PowerDNS -- LDAP backend fails to escape all queries
The LDAP backend in PowerDNS has issues with escaping queries which
could cause connection errors. This would make it possible for a
malicious user to temporarily blank domains.

> This is known to affect all releases prior to 2.9.18.


## Affected packages
 - **powerdns**
    - **Fixed:** 2.9.18
