# pear-XML_RPC -- arbitrary remote code execution
GulfTech Security Research Team reports:

> PEAR XML_RPC is vulnerable to a very high risk php code injection
> vulnerability due to unsanatized data being passed into an eval()
> call.


## Affected packages
 - **pear-XML_RPC**
    - **Fixed:** 1.3.1
