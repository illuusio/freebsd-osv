# groff -- groffer uses temporary files unsafely
The groffer script in the groff package 1.18 and later versions allows
local users to overwrite files via a symlink attack on temporary files.


## Affected packages
 - **ja-groff**
    - **Introduced:** 1.18
    - **Fixed:** 1.18.1_8
