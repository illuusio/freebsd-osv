# firefox -- arbitrary code execution from sidebar panel
A Mozilla Foundation Security Advisory states:

> If a user bookmarked a malicious page as a Firefox sidebar panel that
> page could execute arbitrary programs by opening a privileged page and
> injecting javascript into it.


## Affected packages
 - **firefox**
    - **Fixed:** 1.0.2,1
 - **linux-firefox**
    - **Fixed:** 1.0.2
