# razor-agents -- denial of service vulnerability
A Secunia security advisory reports:

> Two vulnerabilities have been reported in Razor-agents, which can be
> exploited by malicious people to cause a DoS (Denial of Service).
>
> 1.  An unspecified error in the preprocessing of certain HTML messages
>     can be exploited to crash the application.
> 2.  A bug in the discovery logic causes Razor-agents to go into an
>     infinite loop and consume a large amount of memory when discovery
>     fails.


## Affected packages
 - **razor-agents**
    - **Fixed:** 2.71
