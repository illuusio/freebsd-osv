# hashcash -- format string vulnerability
A Gentoo Linux Security Advisory reports:

> Tavis Ormandy of the Gentoo Linux Security Audit Team identified a
> flaw in the Hashcash utility that an attacker could expose by
> specifying a malformed reply address.
>
> Successful exploitation would permit an attacker to disrupt Hashcash
> users, and potentially execute arbitrary code.


## Affected packages
 - **hashcash**
    - **Fixed:** 1.17
