# pcre -- regular expression buffer overflow
The pcre library is vulnerable to a buffer overflow vulnerability due to
insufficient validation of quantifier values. This could lead execution
of arbitrary code with the permissions of the program using pcre by way
of a specially crated regular expression.


## Affected packages
 - **pcre**
    - **Fixed:** 6.2
 - **pcre-utf8**
    - **Fixed:** 6.2
