# postgresql -- multiple buffer overflows in PL/PgSQL parser
The PL/PgSQL parser in postgresql is vulnerable to several buffer
overflows. These could be exploited by a remote attacker to execute
arbitrary code with the permissions of the postgresql server by running
a specially crafted query.
