# phpmyadmin -- information disclosure vulnerability
A phpMyAdmin security announcement reports:

> By calling some scripts that are part of phpMyAdmin in an unexpected
> way (especially scripts in the libraries subdirectory), it is possible
> to trigger phpMyAdmin to display a PHP error message which contains
> the full path of the directory where phpMyAdmin is installed.
>
> **Mitigation factor:** This path disclosure is possible on servers
> where the recommended setting of the PHP configuration directive
> `display_errors` is set to on, which is against the recommendations
> given in the PHP manual.


## Affected packages
 - **phpmyadmin**
    - **Fixed:** 2.6.1.2
 - **phpMyAdmin**
    - **Fixed:** 2.6.1.2
