# ImageMagick -- format string vulnerability
Tavis Ormandy reports:

> magemagick-6.2.0-3 fixes an potential issue handling malformed
> filenames, the flaw may affect webapps or scripts that use the
> imagemagick utilities for image processing, or applications linked
> with libMagick.

This vulnerability could crash ImageMagick or potentially lead to the
execution of arbitrary code with the permissions of the user running
ImageMagick.


## Affected packages
 - **ImageMagick**
    - **Fixed:** 6.2.0.3
 - **ImageMagick-nox11**
    - **Fixed:** 6.2.0.3
