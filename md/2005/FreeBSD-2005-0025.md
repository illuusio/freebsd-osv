# squid -- buffer overflow vulnerability in gopherToHTML
The squid patches page notes:

> A malicious gopher server may return a response with very long lines
> that cause a buffer overflow in Squid.
>
> Workaround: Since gopher is very obscure these days, do not allow
> Squid to any gopher servers. Use an ACL rule like:
>
>     acl Gopher proto gopher
>     http_access deny Gopher


## Affected packages
 - **squid**
    - **Fixed:** 2.5.7_6
