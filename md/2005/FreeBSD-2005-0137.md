# gaim -- remote DoS on receiving malformed HTML
The GAIM team reports:

> Receiving malformed HTML can result in an invalid memory access
> causing Gaim to crash.


## Affected packages
 - **gaim**
    - **Fixed:** 1.1.4
 - **ja-gaim**
    - **Fixed:** 1.1.4
 - **ko-gaim**
    - **Fixed:** 1.1.4
 - **ru-gaim**
    - **Fixed:** 1.1.4
