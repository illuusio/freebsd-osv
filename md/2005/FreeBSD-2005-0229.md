# isc-dhcpd -- format string vulnerabilities
The ISC DHCP programs are vulnerable to several format string
vulnerabilities which may allow a remote attacker to execute arbitrary
code with the permissions of the DHCP programs, typically root for the
DHCP server.


## Affected packages
 - **isc-dhcp3-client**
    - **Fixed:** 3.0.1
 - **isc-dhcp3-devel**
    - **Fixed:** 3.0.1
 - **isc-dhcp3-relay**
    - **Fixed:** 3.0.1
 - **isc-dhcp3-server**
    - **Fixed:** 3.0.1
 - **isc-dhcp3**
    - **Fixed:** 3.0.1
 - **isc-dhcp**
    - **Fixed:** 3.0.1
 - **isc-dhcpd**
    - **Fixed:** 3.0.1
