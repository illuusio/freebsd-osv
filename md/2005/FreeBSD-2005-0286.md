# phpmyadmin -- local file inclusion vulnerability
A phpMyAdmin security announcement reports:

> In libraries/grab_globals.lib.php, the \$\_\_redirect parameter was
> not correctly validated, opening the door to a local file inclusion
> attack.
>
> We consider this vulnerability to be serious.


## Affected packages
 - **phpMyAdmin**
    - **Introduced:** 2.6.4.r1
    - **Fixed:** 2.6.4.1
