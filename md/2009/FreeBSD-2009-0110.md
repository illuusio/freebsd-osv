# tor-devel -- DNS resolution vulnerability
The Tor Project reports:

> A malicious exit relay could convince a controller that the client\'s
> DNS question resolves to an internal IP address.


## Affected packages
 - **tor-devel**
    - **Fixed:** 0.2.1.15-rc
