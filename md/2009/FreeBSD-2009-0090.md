# php -- ini database truncation inside dba_replace() function
securityfocus research reports:

> A bug that leads to the emptying of the INI file contents if the
> database key was not found exists in PHP dba extension in versions
> 5.2.6, 4.4.9 and earlier.
>
> Function dba_replace() are not filtering strings key and value. There
> is a possibility for the destruction of the file.


## Affected packages
 - **php4-dba**
    - **Fixed:** 4.4.9_1
 - **php5-dba**
    - **Fixed:** 5.2.7
