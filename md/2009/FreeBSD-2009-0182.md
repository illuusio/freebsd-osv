# monkey -- improper input validation vulnerability
Census Labs reports:

> We have discovered a remotely exploitable \"improper input
> validation\" vulnerability in the Monkey web server that allows an
> attacker to perform denial of service attacks by repeatedly crashing
> worker threads that process HTTP requests.


## Affected packages
 - **monkey**
    - **Fixed:** 0.9.3
