# pngcrush -- libpng Uninitialised Pointer Arrays Vulnerability
Secunia reports:

> A vulnerability has been reported in Pngcrush, which can be exploited
> by malicious people to potentially compromise a user\'s system.
>
> The vulnerability is caused due to the use of vulnerable libpng code.


## Affected packages
 - **pngcrush**
    - **Fixed:** 1.6.14
