# nginx -- remote denial of service vulnerability
nginx development team reports:

> A segmentation fault might occur in worker process while specially
> crafted request handling.


## Affected packages
 - **nginx**
    - **Fixed:** 0.7.62
 - **nginx-devel**
    - **Fixed:** 0.8.15
