# opera -- multiple vulnerabilities
Opera Team reports:

> An unspecified error in the processing of JPEG images can be exploited
> to trigger a memory corruption.
>
> An error can be exploited to execute arbitrary script code in a
> different domain via unspecified plugins.
>
> An unspecified error has a \"moderately severe\" impact. No further
> information is available.


## Affected packages
 - **opera**
    - **Fixed:** 9.64
 - **linux-opera**
    - **Fixed:** 9.64
