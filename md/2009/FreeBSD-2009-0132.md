# apache22 -- several vulnerabilities
Apache ChangeLog reports:

> CVE-2009-1891: Fix a potential Denial-of-Service attack against
> mod_deflate or other modules.
>
> CVE-2009-1195: Prevent the \"Includes\" Option from being enabled in
> an .htaccess file if the AllowOverride restrictions do not permit it.
>
> CVE-2009-1890: Fix a potential Denial-of-Service attack against
> mod_proxy in a reverse proxy configuration.
>
> CVE-2009-1191: mod_proxy_ajp: Avoid delivering content from a previous
> request which failed to send a request body.
>
> CVE-2009-0023, CVE-2009-1955, CVE-2009-1956: The bundled copy of the
> APR-util library has been updated, fixing three different security
> issues which may affect particular configurations and third-party
> modules (was already fixed in 2.2.11_5).


## Affected packages
 - **apache**
    - **Introduced:** 2.2.0,1
    - **Fixed:** 2.2.12
