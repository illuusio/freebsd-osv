# opera -- multiple vulnerabilities
Opera Team Reports:

> - Issue where sites using revoked intermediate certificates might be
>   shown as secure
> - Issue where the collapsed address bar didn\'t show the current
>   domain
> - Issue where pages could trick users into uploading files
> - Some IDNA characters not correctly displaying in the address bar
> - Issue where Opera accepts nulls and invalid wild-cards in
>   certificates


## Affected packages
 - **opera**
    - **Fixed:** 10.00.20090830
 - **opera-devel**
    - **Fixed:** 10.00.b3_1,1
 - **linux-opera**
    - **Fixed:** 10.00
