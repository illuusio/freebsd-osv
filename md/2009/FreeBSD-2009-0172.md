# ruby -- heap overflow vulnerability
The official ruby site reports:

> There is a heap overflow vulnerability in String#ljust, String#center
> and String#rjust. This has allowed an attacker to run arbitrary code
> in some rare cases.


## Affected packages
 - **ruby**
    - **Introduced:** 1.9.1,1
    - **Fixed:** 1.9.1.376,1
