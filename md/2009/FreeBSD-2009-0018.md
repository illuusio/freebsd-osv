# pdfjam -- insecure temporary files
Secunia reports:

> Some security issues have been reported in PDFjam, which can be
> exploited by malicious, local users to perform certain actions with
> escalated privileges.
>
> The security issues are caused due to the \"pdf90\", \"pdfjoin\", and
> \"pdfnup\" scripts using temporary files in an insecure manner. This
> can be exploited to overwrite arbitrary files via symlink attacks.


## Affected packages
 - **pdfjam**
    - **Fixed:** 1.20_4
