# imap-uw -- imap c-client buffer overflow
SANS reports:

> The University of Washington IMAP library is a library implementing
> the IMAP mail protocol. University of Washington IMAP is exposed to a
> buffer overflow issue that occurs due to a boundary error within the
> rfc822_output_char function in the c-client library. The University of
> Washington IMAP library versions prior to 2007e are affected.


## Affected packages
 - **imap-uw**
    - **Fixed:** 2007e
