# opera -- multiple vulnerabilities
Opera Team Reports:

> - Fixed an issue where certain domain names could allow execution of
>   arbitrary code, as reported by Chris Weber of Casaba Security
> - Fixed an issue where scripts can run on the feed subscription page,
>   as reported by Inferno


## Affected packages
 - **opera**
    - **Fixed:** 10.01.20091019
 - **linux-opera**
    - **Fixed:** 10.01
