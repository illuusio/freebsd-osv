# nagios -- web interface privilege escalation vulnerability
securityfocus reports:

> An attacker with low-level privileges may exploit this issue to bypass
> authorization and cause arbitrary commands to run within the context
> of the Nagios server. This may aid in further attacks.


## Affected packages
 - **nagios**
    - **Fixed:** 3.0.5
 - **nagios2**
    - **Fixed:** 2.12_2
