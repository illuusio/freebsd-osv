# Teamspeak Server -- Directory Traversal Vulnerability
SecurityFocus reports:

> TeamSpeak is prone to a directory-traversal vulnerability because it
> fails to sufficiently sanitize user-supplied input data. Exploiting
> the issue may allow an attacker to obtain sensitive information that
> could aid in further attacks.


## Affected packages
 - **teamspeak_server**
    - **Fixed:** 2.0.23.17
