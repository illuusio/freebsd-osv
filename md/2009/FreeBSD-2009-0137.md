# mozilla firefox -- multiple vulnerabilities
Mozilla Foundation reports:

> MFSA 2009-51 Chrome privilege escalation with FeedWriter
>
> MFSA 2009-50 Location bar spoofing via tall line-height Unicode
> characters
>
> MFSA 2009-49 TreeColumns dangling pointer vulnerability
>
> MFSA 2009-48 Insufficient warning for PKCS11 module installation and
> removal
>
> MFSA 2009-47 Crashes with evidence of memory corruption
> (rv:1.9.1.3/1.9.0.14)


## Affected packages
 - **firefox**
    - **Introduced:** 3.5.*,1,1
    - **Fixed:** 3.5.3,1
    - **Introduced:** 3.*,1,1
    - **Fixed:** 3.0.13,1
