# nfsen -- remote command execution
nfsen reports:

> Due to double input checking, a remote command execution security bug
> exists in all NfSen versions 1.3 and 1.3.1. Users are requested to
> update to nfsen-1.3.2.


## Affected packages
 - **nfsen**
    - **Fixed:** 1.3.2
