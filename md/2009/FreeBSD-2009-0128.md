# memcached -- memcached stats maps Information Disclosure Weakness
Secunia reports:

> A weakness has been reported in memcached, which can be exploited by
> malicious people to disclose system information.
>
> The weakness is caused due to the application disclosing the content
> of /proc/self/maps if a stats maps command is received. This can be
> exploited to disclose e.g. the addresses of allocated memory regions.


## Affected packages
 - **memcached**
    - **Fixed:** 1.2.8
