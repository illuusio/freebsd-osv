# tor -- multiple vulnerabilities
Secunia reports:

> Some vulnerabilities have been reported in Tor, where one has an
> unknown impact and others can be exploited by malicious people to
> cause a DoS.
>
> An error when running Tor as a directory authority can be exploited to
> trigger the execution of an infinite loop.
>
> An unspecified error exists when running on Windows systems prior to
> Windows XP. No further information is currently available.


## Affected packages
 - **tor**
    - **Fixed:** 0.2.0.34
 - **tor-devel**
    - **Fixed:** 0.2.12-alpha
