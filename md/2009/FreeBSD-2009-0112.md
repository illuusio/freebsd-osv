# phpmyadmin -- XSS vulnerability
The phpMyAdmin project reports:

> It was possible to conduct an XSS attack via a crafted SQL bookmark.
>
> All 3.x releases on which the \"bookmarks\" feature is active are
> affected, previous versions are not.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 3.2.0.1
