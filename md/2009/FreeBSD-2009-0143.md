# fwbuilder -- security issue in temporary file handling
Firewall Builder release notes reports:

> Vadim Kurland (vadim.kurland@fwbuilder.org) reports:
>
> Fwbuilder and libfwbuilder 3.0.4 through to 3.0.6 generate iptables
> scripts with a security issue when also used to generate static
> routing configurations.


## Affected packages
 - **fwbuilder**
    - **Fixed:** 3.0.7
