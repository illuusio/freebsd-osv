# nsd -- buffer overflow vulnerability
NLnet Labs:

> A one-byte buffer overflow has been reported in NSD. The problem
> affects all versions 2.0.0 to 3.2.1. The bug allows a carefully
> crafted exploit to bring down your DNS server. It is highly unlikely
> that this one byte overflow can lead to other (system) exploits.


## Affected packages
 - **nsd**
    - **Fixed:** 3.2.2
 - **nsd2**
    - **Fixed:** 2.3.7_1
