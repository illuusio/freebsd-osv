# openfire -- multiple vulnerabilities
Core Security Technologies reports:

> Multiple cross-site scripting vulnerabilities have been found which
> may lead to arbitrary remote code execution on the server running the
> application due to unauthorized upload of Java plugin code.


## Affected packages
 - **openfire**
    - **Fixed:** 3.6.3
