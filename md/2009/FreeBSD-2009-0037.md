# phplist -- local file inclusion vulnerability
Secunia reports:

> Input passed to the \"\_SERVER\[ConfigFile\]\" parameter in
> admin/index.php is not properly verified before being used to include
> files. This can be exploited to include arbitrary files from local
> resources.


## Affected packages
 - **phplist**
    - **Fixed:** 2.10.9
