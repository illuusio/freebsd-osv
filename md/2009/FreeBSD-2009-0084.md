# cyrus-sasl -- buffer overflow vulnerability
US-CERT reports:

> The sasl_encode64() function converts a string into base64. The Cyrus
> SASL library contains buffer overflows that occur because of unsafe
> use of the sasl_encode64() function.


## Affected packages
 - **cyrus-sasl**
    - **Fixed:** 2.1.23
