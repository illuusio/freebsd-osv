# isc-dhcp-client -- Stack overflow vulnerability
US-CERT reports:

> The ISC DHCP dhclient application contains a stack buffer overflow,
> which may allow a remote, unauthenticated attacker to execute
> arbitrary code with root privileges.


## Affected packages
 - **isc-dhcp31-client**
    - **Fixed:** 3.1.1
 - **isc-dhcp30-client**
    - **Fixed:** 3.0.7_1
