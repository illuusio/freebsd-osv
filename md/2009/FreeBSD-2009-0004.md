# php5-gd -- uninitialized memory information disclosure vulnerability
According to CVE-2008-5498 entry:

> Array index error in the \"imageRotate\" function in PHP 5.2.8 and
> earlier allows context-dependent attackers to read the contents of
> arbitrary memory locations via a crafted value of the third argument
> (aka the \"bgd_color\" or \"clrBack\" argument) for an indexed image.


## Affected packages
 - **php5-gd**
    - **Fixed:** 5.2.8
