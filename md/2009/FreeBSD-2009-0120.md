# SquirrelMail -- Plug-ins compromise
# Problem Description:

The SquirrelMail Web Server has been compromised, and three plugins are
affected.

The port of squirrelmail-sasql-plugin is safe (right MD5), and
change_pass is not in the FreeBSD ports tree, but multilogin has a wrong
MD5.


## Affected packages
 - **squirrelmail-multilogin-plugin**
    - **Introduced:** 2.3.4
    - **Fixed:** 2.3.4_2
