# vnc -- authentication bypass vulnerability
RealVNC is susceptible to an authentication-bypass vulnerability. A
malicious VNC client can cause a VNC server to allow it to connect
without any authentication regardless of the authentication settings
configured in the server. Exploiting this issue allows attackers to gain
unauthenticated, remote access to the VNC servers.


## Affected packages
 - **vnc**
    - **Introduced:** 4.1.1
    - **Fixed:** 4.1.1
