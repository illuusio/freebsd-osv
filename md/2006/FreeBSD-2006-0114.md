# dokuwiki -- multiple vulnerabilities
Multiple vulnerabilities have been reported within dokuwiki. dokuwiki is
proven vulnerable to:

- arbitrary PHP code insertion via spellcheck module,
- XSS attack via \"Update your account profile,\"
- bypassing of ACL controls when enabled.


## Affected packages
 - **dokuwiki**
    - **Fixed:** 20060309_2
