# plone -- user can masquerade as a group
Plone.org reports:

> PlonePAS-using Plone releases (Plone 2.5 and Plone 2.5.1) has a
> potential vulnerability that allows a user to masquerade as a group.
> Please update your sites.


## Affected packages
 - **plone**
    - **Introduced:** 2.5,1
    - **Fixed:** 2.5.1_1
