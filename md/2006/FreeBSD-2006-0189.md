# ingo -- local arbitrary shell command execution
The Horde team reports a vulnerability within Ingo, the filter
management suite. The vulnerability is caused due to inadequete
escaping, possibly allowing a local user to execute arbitrary shell
commands via procmail.


## Affected packages
 - **ingo**
    - **Fixed:** 1.1.2
