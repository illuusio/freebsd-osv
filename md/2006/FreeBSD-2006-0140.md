# mysql -- format string vulnerability
Jean-David Maillefer reports a Denial of Service vulnerability within
MySQL. The vulnerability is caused by improper checking of the
data_format routine, which cause the MySQL server to crash. The crash is
triggered by the following code:\
`"SELECT date_format('%d%s', 1);`


## Affected packages
 - **mysql-server**
    - **Introduced:** 5.1
    - **Fixed:** 5.1.6
    - **Introduced:** 5.0
    - **Fixed:** 5.0.19
    - **Introduced:** 4.1
    - **Fixed:** 4.1.18
