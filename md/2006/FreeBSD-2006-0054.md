# linux-realplayer -- buffer overrun
Secunia Advisories Reports:

> A boundary error when processing SWF files can be exploited to cause a
> buffer overflow. This may allow execution of arbitrary code on the
> user\'s system.


## Affected packages
 - **linux-realplayer**
    - **Introduced:** 10.0.1
    - **Fixed:** 10.0.7.785.20060201
