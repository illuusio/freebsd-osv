# mambo -- SQL injection vulnerabilities
The Team Mambo reports that two SQL injection vulnerabilities have been
found in Mambo. The vulnerabilities exists due to missing sanitation of
the `title` and `catid` parameters in the `weblinks.php` page and can
lead to execution of arbitrary SQL code.


## Affected packages
 - **mambo**
    - **Fixed:** 4.5.4
