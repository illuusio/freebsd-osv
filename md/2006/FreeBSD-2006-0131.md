# zope -- information disclosure vulnerability
Zope team reports:

> Unspecified vulnerability in (Zope2) allows local users to obtain
> sensitive information via unknown attack vectors related to the
> docutils module and \"restructured text\".


## Affected packages
 - **zope**
    - **Introduced:** 2.7.0
    - **Fixed:** 2.7.9
    - **Introduced:** 2.8.0
    - **Fixed:** 2.8.7
    - **Introduced:** 2.9.0
    - **Fixed:** 2.9.3
