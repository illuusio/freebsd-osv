# eyeOS -- multiple XSS security bugs
eyeOS team reports:

> \[EyeOS 0.9.1\] release fixes two XSS security bugs, so we recommend
> all users to upgrade to this new version in order to have the best
> security. These two bugs were discovered by Jose Carlos Norte, who is
> a new eyeOS developer.


## Affected packages
 - **eyeOS**
    - **Fixed:** 0.9.1
