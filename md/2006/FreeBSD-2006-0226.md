# proftpd -- remote code execution vulnerabilities
The proftpd development team reports that several remote buffer
overflows had been found in the proftpd server.


## Affected packages
 - **proftpd**
    - **Fixed:** 1.3.0_5
 - **proftpd-mysql**
    - **Fixed:** 1.3.0_5
