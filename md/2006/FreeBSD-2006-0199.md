# mysql -- database suid privilege escalation
Dmitri Lenev reports a privilege escalation in MySQL. MySQL evaluates
arguments of suid routines in the security context of the routine\'s
definer instead of the routine\'s caller, which allows remote and local
authenticated users to gain privileges through a routine that has been
made available using GRANT EXECUTE.


## Affected packages
 - **mysql-server**
    - **Introduced:** 5.1
    - **Fixed:** 5.1.12
    - **Introduced:** 5.0
    - **Fixed:** 5.0.25
