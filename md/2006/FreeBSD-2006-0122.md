# webmin, usermin -- arbitrary file disclosure vulnerability
The webmin development team reports:

> An attacker without a login to Webmin can read the contents of any
> file on the server using a specially crafted URL. All users should
> upgrade to version 1.290 as soon as possible, or setup IP access
> control in Webmin.


## Affected packages
 - **webmin**
    - **Fixed:** 1.290
 - **usermin**
    - **Fixed:** 1.220
