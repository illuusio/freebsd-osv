# mozilla -- multiple vulnerabilities
The Mozilla Foundation reports of multiple security issues in Firefox,
Seamonkey, and Thunderbird. Several of these issues can probably be used
to run arbitrary code with the privilege of the user running the
program.

> - MFSA 2006-64 Crashes with evidence of memory corruption (rv:1.8.0.7)
> - MFSA 2006-63 JavaScript execution in mail via XBL
> - MFSA 2006-62 Popup-blocker cross-site scripting (XSS)
> - MFSA 2006-61 Frame spoofing using document.open()
> - MFSA 2006-60 RSA Signature Forgery
> - MFSA 2006-59 Concurrency-related vulnerability
> - MFSA 2006-58 Auto-Update compromise through DNS and SSL spoofing
> - MFSA 2006-57 JavaScript Regular Expression Heap Corruption
