# jabberd -- SASL Negotiation Denial of Service Vulnerability
Secunia reports:

> A vulnerability has been reported in jabberd, which can be exploited
> by malicious people to cause a DoS (Denial of Service).
>
> The vulnerability is caused due to an error within the handling of
> SASL negotiation. This can be exploited to cause a crash by sending a
> \"response\" stanza before an \"auth\" stanza.


## Affected packages
 - **jabberd**
    - **Fixed:** 2.0.11
