# p5-DBI -- insecure temporary file creation vulnerability
Javier Fernández-Sanguino Peña reports:

> The DBI library, the Perl5 database interface, creates a temporary PID
> file in an insecure manner. This can be exploited by a malicious user
> to overwrite arbitrary files owned by the person executing the parts
> of the library.


## Affected packages
 - **p5-DBI-137**
 - **p5-DBI**
    - **Fixed:** 1.37_1
    - **Introduced:** 1.38
    - **Fixed:** 1.48
