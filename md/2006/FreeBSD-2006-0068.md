# phpmyadmin -- XSS vulnerabilities
phpMyAdmin security announcement:

> It was possible to conduct an XSS attack with a direct call to some
> scripts under the themes directory.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.8.0.3
