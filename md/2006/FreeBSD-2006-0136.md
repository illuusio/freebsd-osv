# drupal -- XSS vulnerability
The Drupal project reports:

> A malicious user can execute a cross site scripting attack by enticing
> someone to visit a Drupal site via a specially crafted link.


## Affected packages
 - **drupal**
    - **Fixed:** 4.6.9
