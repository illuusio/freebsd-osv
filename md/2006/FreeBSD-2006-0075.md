# plone -- "member_id" Parameter Portrait Manipulation Vulnerability
Secunia reports:

> The vulnerability is caused due to missing security declarations in
> \"changeMemberPortrait\" and \"deletePersonalPortrait\". This can be
> exploited to manipulate or delete another user\'s portrait via the
> \"member_id\" parameter.


## Affected packages
 - **plone**
    - **Fixed:** 2.1.2_1
