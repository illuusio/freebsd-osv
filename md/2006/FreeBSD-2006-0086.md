# amaya -- Attribute Value Buffer Overflow Vulnerabilities
Secunia reports:

> Amaya have two vulnerabilities, which can be exploited by malicious
> people to compromise a user\'s system.
>
> The vulnerabilities are caused due to boundary errors within the
> parsing of various attribute values. This can be exploited to cause
> stack-based buffer overflows when a user opens a specially crafted
> HTML document containing certain tags with overly long attribute
> values.
>
> Successful exploitation allows execution of arbitrary code.


## Affected packages
 - **amaya**
    - **Fixed:** 9.5
