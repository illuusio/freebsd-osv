# MT -- Search Unspecified XSS
Secunia reports:

> Arai has reported a vulnerability in Movable Type and Movable Type
> Enterprise, which can be exploited by malicious people to conduct
> cross-site scripting attacks.
>
> Some unspecified input passed via the search functionality isn\'t
> properly sanitised before being returned to the user. This can be
> exploited to execute arbitrary HTML and script code in a user\'s
> browser session in context of an affected site.


## Affected packages
 - **MT**
    - **Introduced:** 3.3
    - **Fixed:** 3.33
