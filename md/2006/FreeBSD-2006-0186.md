# tkdiff -- temporary file symlink privilege escalation
Javier Fernández-Sanguino Peña reports a vulnerability in tkdiff which
allows local users to gain priveleges of the user running tkdiff due to
insecure temporary file creation.


## Affected packages
 - **tkdiff**
    - **Fixed:** 4.1.1
