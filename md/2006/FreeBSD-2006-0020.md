# phpicalendar -- cross site scripting vulnerability
Francesco Ongaro reports that phpicalendar is vulnerable for a cross
site scripting attack. The vulnerability is caused by improper
validation of the index.php file allowing attackers to include an
arbitrary file with the .php extension


## Affected packages
 - **phpicalendar**
    - **Fixed:** 2.1
