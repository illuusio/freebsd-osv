# squirrelmail -- random variable overwrite vulnerability
The SquirrelMail developers report:

> A logged in user could overwrite random variables in compose.php,
> which might make it possible to read/write other users\' preferences
> or attachments.


## Affected packages
 - **ja-squirrelmail**
    - **Introduced:** 1.4.0
    - **Fixed:** 1.4.8,2
 - **squirrelmail**
    - **Introduced:** 1.4.0
    - **Fixed:** 1.4.8
