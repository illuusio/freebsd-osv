# phpicalendar -- file disclosure vulnerability
The phpicalendar team reports that there is an unspecified vulnerability
within phpicalendar. This seems to be a file disclosure vulnerability
caused by improper checking of the template parsing function. This would
allow an attacker to disclose any file readable by the user under which
the webserver runs.


## Affected packages
 - **phpicalendar**
    - **Fixed:** 2.21
