# joomla -- multiple vulnerabilities
The Joomla development team reports multiple vulnerabilities within the
joomla application. Joomla is vulnerable to the following
vulnerabilities:

- Improper validation of the mosMail function
- Improper validation of the JosIsValidEmail function.
- Remote code execution in PEAR.php
- Zend Hash del key or index vulnerability


## Affected packages
 - **joomla**
    - **Fixed:** 1.0.11
