# wordpress -- full path disclosure
Dedi Dwianto reports:

> A remote user can access the file directly to cause the system to
> display an error message that indicates the installation path. The
> resulting error message will disclose potentially sensitive
> installation path information to the remote attacker.


## Affected packages
 - **wordpress**
    - **Fixed:** 1.5.2
