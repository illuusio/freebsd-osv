# screen -- combined UTF-8 characters vulnerability
A vulnerability in the handling of combined UTF-8 characters in screen
may allow an user-assisted attacker to crash screen or potentially allow
code execution as the user running screen. To exploit this issue the
user running scren must in some way interact with the attacker.


## Affected packages
 - **screen**
    - **Fixed:** 4.0.3
