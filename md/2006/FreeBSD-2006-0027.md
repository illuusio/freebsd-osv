# libtomcrypt -- weak signature scheme with ECC keys
The Secure Science Corporation reports that libtomcrypt is vulnerable to
a weak signature scheme. This allows an attacker to create a valid
random signature and use that to sign arbitrary messages without
requiring the private key.


## Affected packages
 - **libtomcrypt**
    - **Fixed:** 1.02
