# hlstats -- multiple cross site scripting vulnerabilities
Kefka reports multiple cross site scripting vulnerabilities within
hlstats. The vulnerabilities are caused due to improper checking of
variables, allowing an attacker to perform cross site scripting.


## Affected packages
 - **hlstats**
    - **Fixed:** 1.35
