# squirrelmail -- plugin.php local file inclusion vulnerability
The SquirrelMail Project Team reports:

> A security issue has been uncovered in functions/plugin.php that could
> allow a remote user to access local files on the server without
> requiring login. This issue manifests itself if register_globals is
> enabled, and magic_quotes_gpc is disabled.


## Affected packages
 - **squirrelmail**
    - **Fixed:** 1.4.6_1
