# x11vnc -- authentication bypass vulnerability
Ludwig Nussel reports that x11vnc is vulnerable to an authentication
bypass vulnerability. The vulnerability is caused by an error in auth.c.
This could allow a remote attacker to gain unauthorized and
unauthenticated access to the system.


## Affected packages
 - **x11vnc**
    - **Fixed:** 0.8.2
