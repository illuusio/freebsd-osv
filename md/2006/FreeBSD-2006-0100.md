# coppermine -- "file" Local File Inclusion Vulnerability
Secunia reports:

> Coppermine Photo Gallery have a vulnerability, which can be exploited
> by malicious people to disclose sensitive information.
>
> Input passed to the \"file\" parameter in \"index.php\" isn\'t
> properly verified, before it is used to include files. This can be
> exploited to include arbitrary files from local resources.
>
> Example:
> http://\[host\]/index.php?file=.//././/././/././/./\[file\]%00
>
> Successful exploitation requires that \"magic_quotes_gpc\" is
> disabled.


## Affected packages
 - **coppermine**
    - **Fixed:** 1.4.5
