# gedit -- format string vulnerability
Yan Feng reports a format string vulnerability in gedit. This
vulnerability could cause a denial of service with a binary file that
contains format string characters within the filename. It had been
reported that web browsers and email clients can be configured to
provide a filename as an argument to gedit.:


## Affected packages
 - **gedit**
    - **Fixed:** 2.10.3
