# py-matrix-synapse -- federation denial of service via malformed events
element-hq/synapse developers report:

> A malicious server can craft events which, when received, prevent
> Synapse version up to 1.127.0 from federating with other servers. The
> vulnerability has been exploited in the wild.


## Affected packages
 - **py38-matrix-synapse**
    - **Fixed:** 1.127.1
 - **py39-matrix-synapse**
    - **Fixed:** 1.127.1
 - **py310-matrix-synapse**
    - **Fixed:** 1.127.1
 - **py311-matrix-synapse**
    - **Fixed:** 1.127.1
