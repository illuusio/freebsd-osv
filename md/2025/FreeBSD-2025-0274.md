# Mozilla -- memory safety bugs
security@mozilla.org reports:

> Memory safety bugs present in Firefox ESR 115.26, Firefox ESR 128.13,
> Thunderbird ESR 128.13, Firefox ESR 140.1, Thunderbird ESR 140.1,
> Firefox 141 and Thunderbird 141. Some of these bugs showed evidence of
> memory corruption and we presume that with enough effort some of these
> could have been exploited to run arbitrary code.


## Affected packages
 - **firefox**
    - **Fixed:** 142,2
 - **firefox-esr**
    - **Fixed:** 128.14
 - **thunderbird**
    - **Fixed:** 140.2
