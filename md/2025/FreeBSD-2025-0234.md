# 7-Zip -- Multi-byte write heap buffer overflow in NCompress::NRar5::CDecoder
security-advisories@github.com reports:

> 7-Zip is a file archiver with a high compression ratio. Zeroes written
> outside heap buffer in RAR5 handler may lead to memory corruption and
> denial of service in versions of 7-Zip prior to 25.0.0. Version 25.0.0
> contains a fix for the issue.


## Affected packages
 - **7-zip**
    - **Fixed:** 25.00
