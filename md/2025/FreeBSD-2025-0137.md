# Mozilla -- insufficient character escaping
security@mozilla.org reports:

> Due to insufficient escaping of special characters in the \"copy as
> cURL\" feature, an attacker could trick a user into using this
> command, potentially leading to local code execution on the user\'s
> system.


## Affected packages
 - **firefox**
    - **Fixed:** 138.0,2
 - **thunderbird**
    - **Fixed:** 138.0
