# Gitlab -- vulnerabilities
Gitlab reports:

> Partial Bypass for Device OAuth flow using Cross Window Forgery
>
> Denial of service by abusing Github import API
>
> Group IP restriction bypass allows disclosing issue title of
> restricted project
