# Mozilla -- local code execution
security@mozilla.org reports:

> Due to insufficient escaping of the newline character in the Copy as
> cURL feature, an attacker could trick a user into using this command,
> potentially leading to local code execution on the user\'s system.


## Affected packages
 - **firefox-esr**
    - **Fixed:** 115.24.0
 - **firefox**
    - **Fixed:** 139.0,2
