# Mozilla -- memory corruption
security@mozilla.org reports:

> Memory safety bugs present in Firefox 136 and Thunderbird 136. Some of
> these bugs showed evidence of memory corruption and we presume that
> with enough effort some of these could have been exploited to run
> arbitrary code.


## Affected packages
 - **firefox**
    - **Fixed:** 137.0,2
 - **librewolf**
    - **Fixed:** 137.0
 - **thunderbird**
    - **Fixed:** 137.0
