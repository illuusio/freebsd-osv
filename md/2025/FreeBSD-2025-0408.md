# python -- several vulnerabilities
Hugo van Kemenade reports:

> Python 3.14.2 and 3.13.11 are now available \[\... and\] come with
> some bonus security fixes.
>
> - gh-142145: Remove quadratic behavior in node ID cache clearing
>   (CVE-2025-12084)
> - gh-119451: Fix a potential denial of service in http.client \[only
>   in 3.13; CVE-2025-13836\]
> - gh-119452: Fix a potential virtual memory allocation denial of
>   service in http.server \[affects platforms without fork()\]


## Affected packages
 - **python39**
 - **python310**
 - **python311**
 - **python312**
 - **python313**
    - **Introduced:** 3.13.0
    - **Fixed:** 3.13.11
 - **python314**
    - **Introduced:** 3.14.0
    - **Fixed:** 3.14.2
