# age -- age vulnerable to malicious plugin names, recipients, or identities causing arbitrary binary execution
Filippo Valsorda reports:

> A plugin name containing a path separator may allow an attacker to
> execute an arbitrary binary.
>
> Such a plugin name can be provided to the age CLI through an
> attacker-controlled recipient or identity string, or to the
> plugin.NewIdentity, plugin.NewIdentityWithoutData, or
> plugin.NewRecipient APIs.


## Affected packages
 - **age**
    - **Fixed:** 1.2.1
