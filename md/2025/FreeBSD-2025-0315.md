# mongodb -- MongoDB Server access to non-initialized memory
cna@mongodb.com reports:

> MongoDB Server may access non-initialized region of memory leading to
> unexpected behaviour when zero arguments are called in internal
> aggregation stage.


## Affected packages
 - **mongodb6**
    - **Fixed:** 6.0.3
