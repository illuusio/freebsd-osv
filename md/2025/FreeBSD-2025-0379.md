# privatebin XSS
privatebin reports:

> Dragging a file whose filename contains HTML is reflected verbatim
> into the page via the drag-and-drop helper, so any user who drops a
> crafted file on PrivateBin will execute arbitrary JavaScript within
> their own session (self-XSS). This allows an attacker who can entice a
> victim to drag or otherwise attach such a file to exfiltrate
> plaintext, encryption keys, or stored pastes before they are encrypted
> or sent.


## Affected packages
 - **privatebin**
    - **Fixed:** 2.0.3
