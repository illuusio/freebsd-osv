# spotipy -- CWE-79: Improper Neutralization of Input During Web Page Generation ('Cross-site Scripting')
https://github.com/spotipy-dev/spotipy/security/advisories/GHSA-r77h-rpp9-w2xm
reports:

> Spotipy is a Python library for the Spotify Web API. Prior to version
> 2.25.2, there is a cross-site scripting (XSS) vulnerability in the
> OAuth callback server that allows for JavaScript injection through the
> unsanitized error parameter. Attackers can execute arbitrary
> JavaScript in the user\'s browser during OAuth authentication. This
> issue has been patched in version 2.25.2.
