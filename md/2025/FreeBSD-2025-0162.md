# Mozilla -- memory corruption
security@mozilla.org reports:

> Memory safety bug present in Firefox ESR 128.10, and Thunderbird
> 128.10. This bug showed evidence of memory corruption and we presume
> that with enough effort this could have been exploited to run
> arbitrary code.


## Affected packages
 - **firefox-esr**
    - **Fixed:** 128.11.0
 - **thunderbird**
    - **Fixed:** 128.11.0
