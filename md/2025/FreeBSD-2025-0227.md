# mod_http2 -- Multiple vulnerabilities
The mod_http2 project reports:

> a client can increase memory consumption for a HTTP/2 connection via
> repeated request header names,leading to denial of service
>
> certain proxy configurations whith mod_proxy_http2 as the backend, an
> assertion can be triggered by certain requests, leading to denial of
> service


## Affected packages
 - **mod_http2**
    - **Fixed:** 2.0.33
