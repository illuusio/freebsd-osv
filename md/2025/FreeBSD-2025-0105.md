# Mozilla -- Memory corruption
security@mozilla.org reports:

> Memory safety bugs present in Firefox 136, Thunderbird 136, Firefox
> ESR 128.8, and Thunderbird 128.8. Some of these bugs showed evidence
> of memory corruption and we presume that with enough effort some of
> these could have been exploited to run arbitrary code.


## Affected packages
 - **firefox**

 - **firefox-esr**
    - **Fixed:** 128.9.0
