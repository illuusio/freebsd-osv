# Vaultwarden -- Multiple vulnerabilities
The Vaultwarden project reports:

> RCE in the admin panel.
>
> Getting access to the Admin Panel via CSRF.
>
> Escalation of privilege via variable confusion in OrgHeaders trait.


## Affected packages
 - **vaultwarden**
    - **Fixed:** 1.33.0
