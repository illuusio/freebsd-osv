# mozilla -- 64 bit JIT WASM read on left over memory
security@mozilla.org reports:

> On 64-bit CPUs, when the JIT compiles WASM i32 return values they can
> pick up bits from left over memory. This can potentially cause them to
> be treated as a different type.
