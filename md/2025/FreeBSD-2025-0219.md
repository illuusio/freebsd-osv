# MongoDB -- Incomplete Redaction of Sensitive Information in MongoDB Server Logs
cna@mongodb.com reports:

> An issue has been identified in MongoDB Server where unredacted
> queries may inadvertently appear in server logs when certain error
> conditions are encountered.


## Affected packages
 - **mongodb60**
    - **Fixed:** 6.0.21
 - **mongodb70**
    - **Fixed:** 7.0.18
 - **mongodb80**
    - **Fixed:** 8.0.5
