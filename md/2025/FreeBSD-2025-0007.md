# redis,valkey -- Denial-of-service valnerability due to malformed ACL selectors
Redis core team reports:

> An authenticated with sufficient privileges may create a malformed ACL
> selector which, when accessed, triggers a server panic and subsequent
> denial of service.The problem exists in Redis 7.0.0 or newer.


## Affected packages
 - **redis**
    - **Introduced:** 7.0.0
    - **Fixed:** 7.4.2
 - **redis72**
    - **Fixed:** 7.2.7
 - **redis-devel**
    - **Fixed:** 7.4.2.20250201
 - **valkey**
    - **Fixed:** 8.0.2
