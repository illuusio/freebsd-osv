# Vieb -- Remote Code Execution via Visiting Untrusted URLs
Zhengyu Liu, Jianjia Yu, Jelmer van Arnhem report:

> We discovered a remote code execution (RCE) vulnerability in the
> latest release of the Vieb browser (v12.3.0). By luring a user to
> visit a malicious website, an attacker can achieve arbitrary code
> execution on the victim's machine.


## Affected packages
 - **linux-vieb**
    - **Fixed:** 12.4.0
