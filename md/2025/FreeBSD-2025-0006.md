# redis,valkey -- Remote code execution valnerability
Redis core team reports:

> An authenticated user may use a specially crafted Lua script to
> manipulate the garbage collector and potentially lead to remote code
> execution. The problem exists in all versions of Redis with Lua
> scripting.


## Affected packages
 - **redis**
    - **Fixed:** 7.4.2
 - **redis72**
    - **Fixed:** 7.2.7
 - **redis62**
    - **Fixed:** 6.2.17
 - **redis-devel**
    - **Fixed:** 7.4.2.20250201
 - **valkey**
    - **Fixed:** 8.0.2
