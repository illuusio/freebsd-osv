# MongoDB -- Improper Handling of Length Parameter Inconsistency
https://jira.mongodb.org/browse/SERVER-115508 reports:

> Mismatched length fields in Zlib compressed protocol headers may allow
> a read of uninitialized heap memory by an unauthenticated client.


## Affected packages
 - **mongodb80**
    - **Fixed:** 8.2.3
 - **mongodb70**
    - **Fixed:** 7.0.28
