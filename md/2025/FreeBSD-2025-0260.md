# FreeBSD -- Integer overflow in libarchive leading to double free
# Problem Description:

An integer overflow in the archive_read_format_rar_seek_data() function
may lead to a double free problem.

# Impact:

Exploiting a double free vulnerability can cause memory corruption. This
in turn could enable a threat actor to execute arbitrary code. It might
also result in denial of service.
