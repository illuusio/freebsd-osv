# Grafana -- DOM XSS vulnerability
Grafana Labs reports:

> An external security researcher responsibly reported a security
> vulnerability in Grafana's built-in [XY chart
> plugin](https://grafana.com/docs/grafana/latest/panels-visualizations/visualizations/xy-chart/)
> that is vulnerable to a [DOM XSS
> vulnerability](https://grafana.com/blog/2023/07/11/trusted-types-how-we-mitigate-xss-threats-in-grafana-10/#what-is-dom-xss).
>
> The CVSS score for this vulnerability is [6.8
> MEDIUM](https://nvd.nist.gov/vuln-metrics/cvss/v3-calculator?vector=AV:N/AC:L/PR:L/UI:R/S:U/C:H/I:L/A:L&version=3.1).
