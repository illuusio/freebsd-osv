# viewvc -- Arbitrary server filesystem content
cmpilato reports:

> The ViewVC standalone web server (standalone.py) is a script provided
> in the ViewVC distribution for the purposes of quickly testing a
> ViewVC configuration. This script can in particular configurations
> expose the contents of the host server\'s filesystem though a
> directory traversal-style attack.
