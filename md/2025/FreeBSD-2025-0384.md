# lightdm-kde-greeter -- Privilege Escalation from lightdm Service User to root
SUSE Security Team reports:

> A Execution with Unnecessary Privileges vulnerability in
> lightdm-kde-greeter allows escalation from the service user to root.
> This issue affects lightdm-kde-greeter before 6.0.4.


## Affected packages
 - **lightdm-kde-greeter**
    - **Fixed:** 6.0.4
