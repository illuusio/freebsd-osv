# Yelp -- arbitrary file read
secalert@redhat.com reports:

> A flaw was found in Yelp. The Gnome user help application allows the
> help document to execute arbitrary scripts. This vulnerability allows
> malicious users to input help documents, which may exfiltrate user
> files to an external environment.


## Affected packages
 - **yelp**
    - **Fixed:** 42.3
