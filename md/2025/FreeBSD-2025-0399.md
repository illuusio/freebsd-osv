# MongoDB -- Improper Validation of Specified Quantity in Input
https://jira.mongodb.org/browse/SERVER-108565 reports:

> Inconsistent object size validation in time series processing logic
> may result in later processing of oversized BSON documents leading to
> an assert failing and process termination.


## Affected packages
 - **mongodb80**
    - **Fixed:** 8.0.16
 - **mongodb70**
    - **Fixed:** 7.0.26
