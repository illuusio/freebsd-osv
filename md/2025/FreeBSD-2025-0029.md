# dendrite -- Server-side request forgery vulnerability
Dendrite team reports:

> This is a security release, gomatrixserverlib was vulnerable to
> server-side request forgery, serving content from a private network it
> can access, under certain conditions.


## Affected packages
 - **dendrite**
    - **Fixed:** 0.14.1
