# mozilla -- double free error
security@mozilla.org reports:

> A double-free issue could have occurred in
> \`sec_pkcs7_decoder_start_decrypt()\` when handling an error path.
> Under specific conditions, the same symmetric key could have been
> freed twice, potentially leading to memory corruption.


## Affected packages
 - **firefox**
    - **Fixed:** 133.0,2
 - **firefox-esr**
    - **Fixed:** 128.7.0
 - **thunderbird**
    - **Fixed:** 133.0
