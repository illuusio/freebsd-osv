# MongoDB -- Reachable Assertion
https://jira.mongodb.org/browse/SERVER-101180 reports:

> MongoDB Server may experience an invariant failure during batched
> delete operations when handling documents. The issue arises when the
> server mistakenly assumes the presence of multiple documents in a
> batch based solely on document size exceeding BSONObjMaxSize.


## Affected packages
 - **mongodb80**
    - **Fixed:** 8.0.13
 - **mongodb70**
    - **Fixed:** 7.0.26
