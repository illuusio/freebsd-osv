# Mozilla -- Memory corruption bug
security@mozilla.org reports:

> Memory safety bugs present in Firefox 133, Thunderbird 133, Firefox
> ESR 128.5, and Thunderbird 128.5. Some of these bugs showed evidence
> of memory corruption and we presume that with enough effort some of
> these could have been exploited to run arbitrary code.


## Affected packages
 - **firefox**
    - **Fixed:** 134.0,2
 - **librewolf**
    - **Fixed:** 134.0
 - **firefox-esr**
    - **Fixed:** 128.6.0
 - **thunderbird**
    - **Fixed:** 134.0
