# sudo-rs -- Partial password reveal when password timeout occurs
Trifecta Tech Foundation reports:

> When typing partial passwords but not pressing return for a long time,
> a password timeout can occur. When this happens, the keys pressed are
> replayed onto the console.


## Affected packages
 - **sudo-rs**
    - **Introduced:** 0.2.7
    - **Fixed:** 0.2.10
 - **sudo-rs-coexist**
    - **Introduced:** 0.2.7
    - **Fixed:** 0.2.10
