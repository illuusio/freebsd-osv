# SQLite -- Integer Overflow vulnerability
http://sqlite3.com reports:

> Integer Overflow vulnerability in SQLite SQLite3 v.3.50.0 allows a
> remote attacker to cause a denial of service via the setupLookaside
> function


## Affected packages
 - **sqlite3**
    - **Fixed:** 3.50.1,1
 - **linux_base-rl9-9.6**
    - **Fixed:** 9.6_1
 - **linux-c7-sqlite**
    - **Fixed:** 3.50.1
