# Mailpit -- Performance information disclosure
Ralph Slooten (Mailpit developer) reports:

> An HTTP endpoint was found which exposed expvar runtime information
> (memory usage, goroutine counts, GC behavior, uptime and potential
> runtime flags) due to the Prometheus client library dependency.


## Affected packages
 - **mailpit**
    - **Fixed:** 1.27.10
