# PostgreSQL -- vulnerabilities
PostgreSQL project reports:

> Tighten security checks in planner estimation functions.
>
> Prevent pg_dump scripts from being used to attack the user running the
> restore.
>
> Convert newlines to spaces in names included in comments in pg_dump
> output.


## Affected packages
 - **postgresql17-server**
    - **Fixed:** 17.6
 - **postgresql16-server**
    - **Fixed:** 16.10
 - **postgresql15-server**
    - **Fixed:** 14.14
 - **postgresql14-server**
    - **Fixed:** 14.19
 - **postgresql13-server**
    - **Fixed:** 13.22
