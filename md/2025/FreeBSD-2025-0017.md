# clamav -- Possbile denial-of-service vulnerability
The ClamAV project reports:

> A possible buffer overflow read bug is found in the OLE2 file parser
> that could cause a denial-of-service (DoS) condition.


## Affected packages
 - **clamav**
    - **Introduced:** 1.0.0,1
    - **Fixed:** 1.4.2,1
 - **clamav-lts**
    - **Introduced:** 1.0.0,1
    - **Fixed:** 1.0.8,1
