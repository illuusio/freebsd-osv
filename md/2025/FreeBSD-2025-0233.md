# libwasmtime -- host panic with fd_renumber WASIp1 function
WasmTime development team reports:

> A bug in Wasmtime\'s implementation of the WASIp1 set of import
> functions can lead to a WebAssembly guest inducing a panic in the host
> (embedder).


## Affected packages
 - **libwasmtime**
    - **Introduced:** 24.0.0
    - **Fixed:** 24.0.4
    - **Introduced:** 33.0.0
    - **Fixed:** 33.0.2
    - **Introduced:** 34.0.0
    - **Fixed:** 34.0.2
