# mozilla -- multiple vulnerabilities
security@mozilla.org reports:

> Memory safety bugs present in Firefox 134 and Thunderbird 134. Some of
> these bugs showed evidence of memory corruption and we presume that
> with enough effort some of these could have been exploited to run
> arbitrary code.
>
> The fullscreen notification is prematurely hidden when fullscreen is
> re-requested quickly by the user. This could have been leveraged to
> perform a potential spoofing attack.


## Affected packages
 - **mozilla**
    - **Fixed:** 135.0.0,2
