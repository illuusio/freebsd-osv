# Gitlab -- vulnerabilities
Gitlab reports:

> Race condition issue in CI/CD cache impacts GitLab CE/EE
>
> Denial of Service issue in JSON input validation middleware impacts
> GitLab CE/EE
>
> Authentication bypass issue in account registration impacts GitLab
> CE/EE
>
> Denial of Service issue in HTTP response processing impacts GitLab
> CE/EE
>
> Improper authorization issue in markdown rendering impacts GitLab EE
>
> Information disclosure issue in terraform registry impacts GitLab
> CE/EE
