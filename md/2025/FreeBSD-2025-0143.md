# Mozilla -- memory safety bugs
security@mozilla.org reports:

> Memory safety bugs present in Firefox 137, Thunderbird 137, Firefox
> ESR 128.9, and Thunderbird 128.9. Some of these bugs showed evidence
> of memory corruption and we presume that with enough effort some of
> these could have been exploited to run arbitrary code.


## Affected packages
 - **firefox**
    - **Fixed:** 138.0,2
 - **firefox-esr**
    - **Fixed:** 128.10
 - **thunderbird**
    - **Fixed:** 138.0
