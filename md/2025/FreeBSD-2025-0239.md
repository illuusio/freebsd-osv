# Mozilla -- IonMonkey-JIT bad stack write
security@mozilla.org reports:

> On 64-bit platforms IonMonkey-JIT only wrote 32 bits of the 64-bit
> return value space on the stack. Baseline-JIT, however, read the
> entire 64 bits.
