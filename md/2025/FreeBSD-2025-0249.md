# Mozilla -- cookie shadowing
security@mozilla.org reports:

> Setting a nameless cookie with an equals sign in the value shadowed
> other cookies. Even if the nameless cookie was set over HTTP and the
> shadowed cookie included the \`Secure\` attribute.


## Affected packages
 - **firefox**
    - **Fixed:** 141.0,2
 - **firefox-esr**
    - **Fixed:** 140.1
 - **thunderbird**
    - **Fixed:** 141.0
 - **thunderbird-esr**
    - **Fixed:** 140.1
