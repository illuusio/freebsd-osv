# mozilla -- Memory safety bugs
security@mozilla.org reports:

> Memory safety bugs present in Firefox 135, Thunderbird 135, Firefox
> ESR 115.20, Firefox ESR 128.7, and Thunderbird 128.7. Some of these
> bugs showed evidence of memory corruption and we presume that with
> enough effort some of these could have been exploited to run arbitrary
> code.

 - **librewolf**

    - **Fixed:** 115.21,1
