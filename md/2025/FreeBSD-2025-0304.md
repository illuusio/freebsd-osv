# LibreSSL -- overwrite and -read vulnerability
The LibreSSL project reports:

> An incorrect length check can result in a 4-byte overwrite and an
> 8-byte overread.


## Affected packages
 - **libressl**
    - **Fixed:** 4.1.1
 - **libressl-devel**
    - **Fixed:** 4.1.1
