# Mozilla -- Memory safety bugs
security@mozilla.org reports:

> Memory safety bugs present in Firefox 138 and Thunderbird 138. Some of
> these bugs showed evidence of memory corruption and we presume that
> with enough effort some of these could have been exploited to run
> arbitrary code.


## Affected packages
 - **firefox**
    - **Fixed:** 139.0,2
 - **thunderbird**
    - **Fixed:** 129.0
