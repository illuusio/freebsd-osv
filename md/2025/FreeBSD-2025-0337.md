# Mozilla -- Memory disclosure
security@mozilla.org reports:

> A compromised web process using malicious IPC messages could have
> caused the privileged browser process to reveal blocks of its memory
> to the compromised process.


## Affected packages
 - **firefox**
    - **Fixed:** 144.0.0,2
 - **firefox-esr**
    - **Fixed:** 140.4.0
 - **thunderbird**
    - **Fixed:** 144.0.0
