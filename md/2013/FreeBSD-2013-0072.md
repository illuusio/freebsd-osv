# ModSecurity -- XML External Entity Processing Vulnerability
Positive Technologies has reported a vulnerability in ModSecurity, which
can be exploited by malicious people to disclose potentially sensitive
information or cause a DoS (Denial Of Serice).

The vulnerability is caused due to an error when parsing external XML
entities and can be exploited to e.g. disclose local files or cause
excessive memory and CPU consumption.

> .


## Affected packages
 - **mod_security**
    - **Introduced:** 2.*,1
    - **Fixed:** 2.7.3
