# asterisk -- multiple vulnerabilities
The Asterisk project reports:

> Remote Crash From Late Arriving SIP ACK With SDP
>
> Remote Crash when Invalid SDP is sent in SIP Request


## Affected packages
 - **asterisk11**
    - **Introduced:** 11.*,1
    - **Fixed:** 11.5.1
 - **asterisk10**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.12.3
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.21.1
