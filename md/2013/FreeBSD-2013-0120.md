# otrs -- Sql Injection + Xss Issue
The OTRS Project reports:

> An attacker with a valid agent login could manipulate URLs leading to
> SQL injection. An attacker with a valid agent login could manipulate
> URLs in the ITSM ConfigItem search, leading to a JavaScript code
> injection (XSS) problem.


## Affected packages
 - **otrs**
    - **Fixed:** 3.2.9
