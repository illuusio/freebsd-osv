# nagios -- buffer overflow in history.cgi
full disclosure reports:

> history.cgi is vulnerable to a buffer overflow due to the use of
> sprintf with user supplied data that has not been restricted in size.


## Affected packages
 - **nagios**
    - **Fixed:** 3.4.3_1
