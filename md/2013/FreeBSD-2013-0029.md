# nss-pam-ldapd -- file descriptor buffer overflow
Garth Mollett reports:

> A file descriptor overflow issue in the use of FD_SET() in
> nss-pam-ldapd can lead to a stack-based buffer overflow. An attacker
> could, under some circumstances, use this flaw to cause a process that
> has the NSS or PAM module loaded to crash or potentially execute
> arbitrary code.


## Affected packages
 - **nss-pam-ldapd**
    - **Fixed:** 0.8.12
