# django-cms -- XSS Vulnerability
Cross-site scripting (XSS) vulnerability

> Jonas Obrist reports: The security issue allows users with limited
> admin access to elevate their privileges through XSS injection using
> the page_attribute template tag. Only users with admin access and the
> permission to edit at least one django CMS page object could exploit
> this vulnerability. Websites that do not use the page_attribute
> template tag are not affected.


## Affected packages
 - **py-django-cms**
    - **Fixed:** 2.3.5
