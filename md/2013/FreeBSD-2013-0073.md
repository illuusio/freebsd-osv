# jasper -- buffer overflow
Fedora reports:

> JasPer fails to properly decode marker segments and other sections in
> malformed JPEG2000 files. Malformed inputs can cause heap buffer
> overflows which in turn may result in execution of attacker-controlled
> code.


## Affected packages
 - **jasper**
    - **Fixed:** 1.900.1_12
