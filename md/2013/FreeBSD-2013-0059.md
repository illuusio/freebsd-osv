# asterisk -- multiple vulnerabilities
Asterisk project reports:

> Buffer Overflow Exploit Through SIP SDP Header
>
> Username disclosure in SIP channel driver
>
> Denial of Service in HTTP server


## Affected packages
 - **asterisk11**
    - **Introduced:** 11.*,1
    - **Fixed:** 11.2.2
 - **asterisk10**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.12.2
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.20.2
