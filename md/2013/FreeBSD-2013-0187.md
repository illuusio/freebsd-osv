# zabbix -- shell command injection vulnerability
Recurity Labs Team project reports:

> Zabbix agent is vulnerable to remote command execution from the Zabbix
> server in some cases.


## Affected packages
 - **zabbix2-agent**
    - **Fixed:** 2.0.10
