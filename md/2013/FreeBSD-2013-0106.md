# php5 -- Heap based buffer overflow in quoted_printable_encode
The PHP development team reports:

> A Heap-based buffer overflow flaw was found in the php
> quoted_printable_encode() function. A remote attacker could use this
> flaw to cause php to crash or execute arbirary code with the
> permission of the user running php


## Affected packages
 - **php5**
    - **Fixed:** 5.4.16
 - **php53**
    - **Fixed:** 5.3.26
