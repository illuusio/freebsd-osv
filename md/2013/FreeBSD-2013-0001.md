# ircd-ratbox and charybdis -- remote DoS vulnerability
atheme.org reports:

> All versions of Charybdis are vulnerable to a remotely-triggered crash
> bug caused by code originating from ircd-ratbox 2.0. (Incidentally,
> this means all versions since ircd-ratbox 2.0 are also vulnerable.)


## Affected packages
 - **ircd-ratbox**
    - **Introduced:** 2.*,1
    - **Fixed:** 3.0.8
 - **charybdis**
    - **Fixed:** 3.4.2
