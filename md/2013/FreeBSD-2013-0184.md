# samba -- multiple vulnerabilities
The Samba project reports:

> These are security releases in order to address CVE-2013-4408 (DCE-RPC
> fragment length field is incorrectly checked) and CVE-2012-6150
> (pam_winbind login without require_membership_of restrictions).


## Affected packages
 - **samba34**
    - **Introduced:** 0,1
 - **samba35**
    - **Introduced:** 0,1
 - **samba36**
    - **Introduced:** 3.6.*,1
    - **Fixed:** 3.6.22
 - **samba4**
    - **Introduced:** 4.0.*,1
    - **Fixed:** 4.0.13
 - **samba41**
    - **Introduced:** 4.1.*,1
    - **Fixed:** 4.1.3
