# py-graphite-web -- Multiple vulnerabilities
Graphite developers report:

> This release contains several security fixes for cross-site scripting
> (XSS) as well as a fix for a remote-execution exploit in graphite-web
> (CVE-2013-5903).


## Affected packages
 - **py26-graphite-web**
    - **Introduced:** 0.9.5
    - **Fixed:** 0.9.11
 - **py27-graphite-web**
    - **Introduced:** 0.9.5
    - **Fixed:** 0.9.11
 - **py31-graphite-web**
    - **Introduced:** 0.9.5
    - **Fixed:** 0.9.11
 - **py32-graphite-web**
    - **Introduced:** 0.9.5
    - **Fixed:** 0.9.11
 - **py33-graphite-web**
    - **Introduced:** 0.9.5
    - **Fixed:** 0.9.11
