# xinetd -- ignores user and group directives for TCPMUX services
xinetd would execute configured TCPMUX services without dropping
privilege to match the service configuration allowing the service to run
with same privilege as the xinetd process (root).


## Affected packages
 - **xinetd**
    - **Fixed:** 2.3.15_1
