# gnutls -- denial of service
Salvatore Bonaccorso reports:

> This vulnerability affects the DANE library of gnutls 3.1.x and gnutls
> 3.2.x. A server that returns more 4 DANE entries could corrupt the
> memory of a requesting client.


## Affected packages
 - **gnutls3**
    - **Fixed:** 3.1.16
