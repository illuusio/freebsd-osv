# asterisk -- multiple vulnerabilities
Asterisk project reports:

> Crashes due to large stack allocations when using TCP
>
> Denial of Service Through Exploitation of Device State Caching


## Affected packages
 - **asterisk11**
    - **Introduced:** 11.*,1
    - **Fixed:** 11.1.2
 - **asterisk10**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.11.1
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.19.1
