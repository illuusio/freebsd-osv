# opera -- execution of arbitrary code
Opera reports:

> Particular DOM event manipulations can cause Opera to crash. In some
> cases, this crash might occur in a way that allows execution of
> arbitrary code. To inject code, additional techniques would have to be
> employed.

 - **opera**
    - **Fixed:** 12.13
 - **opera-devel**
    - **Fixed:** 12.13
 - **linux-opera**
    - **Fixed:** 12.13
 - **linux-opera-devel**
    - **Fixed:** 12.13
