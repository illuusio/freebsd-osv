# libexif -- multiple remote vulnerabilities
libexif project security advisory:

> A number of remotely exploitable issues were discovered in libexif and
> exif, with effects ranging from information leakage to potential
> remote code execution.


## Affected packages
 - **libexif**
    - **Fixed:** 0.6.21
