# FreeBSD -- NFS remote denial of service
> Insufficient input validation in the NFS server allows an attacker to
> cause the underlying file system to treat a regular file as a
> directory.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 8.3
    - **Fixed:** 8.3_8
    - **Introduced:** 9.1
    - **Fixed:** 9.1_3
