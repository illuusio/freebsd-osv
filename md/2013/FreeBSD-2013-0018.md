# mysql/mariadb/percona server -- multiple vulnerabilities
ORACLE reports:

> Multiple SQL injection vulnerabilities in the replication code
>
> Stack-based buffer overflow
>
> Heap-based buffer overflow


## Affected packages
 - **mysql-server**
    - **Introduced:** 5.1
    - **Fixed:** 5.1.67
    - **Introduced:** 5.5
    - **Fixed:** 5.5.29
 - **mariadb-server**
    - **Introduced:** 5.3
    - **Fixed:** 5.3.12
    - **Introduced:** 5.5
    - **Fixed:** 5.5.29
 - **percona-server**
    - **Introduced:** 5.5
    - **Fixed:** 5.5.29.29.4
