# squid -- denial of service
Squid project reports:

> Due to incorrect data validation Squid is vulnerable to a denial of
> service attack when processing specially crafted HTTP requests
>
> This problem allows any client who can generate HTTP requests to
> perform a denial of service attack on the Squid service.


## Affected packages
 - **squid**
    - **Introduced:** 3.2
    - **Fixed:** 3.2.12
    - **Introduced:** 3.3
    - **Fixed:** 3.3.8
