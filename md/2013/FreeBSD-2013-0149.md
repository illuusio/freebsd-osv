# django -- multiple vulnerabilities
The Django project reports:

> These releases address a directory-traversal vulnerability in one of
> Django\'s built-in template tags. While this issue requires some
> fairly specific factors to be exploitable, we encourage all users of
> Django to upgrade promptly.
