# FreeBSD -- BIND remote DoS with deliberately crafted DNS64 query
Problem description:

> Due to a software defect a crafted query can cause named(8) to crash
> with an assertion failure.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 9.0
    - **Fixed:** 9.0_6
    - **Introduced:** 9.1
    - **Fixed:** 9.1_1
