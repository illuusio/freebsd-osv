# ruby -- Hostname check bypassing vulnerability in SSL client
Ruby Developers report:

> Ruby\'s SSL client implements hostname identity check but it does not
> properly handle hostnames in the certificate that contain null bytes.


## Affected packages
 - **ruby19**
    - **Fixed:** 1.9.3.448,1
 - **ruby18**
    - **Fixed:** 1.8.7.374,1
