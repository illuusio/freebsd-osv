# OpenSSL -- DTLS Denial of Service
The OpenSSL Team reports:

> A flaw in the fix to CVE-2011-4108 can be exploited in a denial of
> service attack. Only DTLS applications using OpenSSL 1.0.0f and 0.9.8s
> are affected.


## Affected packages
 - **openssl**
    - **Fixed:** 1.0.0_9
