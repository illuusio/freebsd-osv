# asterisk -- multiple vulnerabilities
Asterisk project reports:

> Remote crash vulnerability in IAX2 channel driver.
>
> Skinny Channel Driver Remote Crash Vulnerability

 - **asterisk16**
    - **Introduced:** 1.6.*,1
    - **Fixed:** 1.6.2.24
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.12.1
 - **asterisk10**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.4.1
