# freetype -- multiple vulnerabilities
The Freetype project reports:

> Multiple vulnerabilities exist in freetype that can result in
> application crashes and remote code execution. Please review the
> details in each of the CVEs for additional information.


## Affected packages
 - **freetype2**
    - **Fixed:** 2.4.9
