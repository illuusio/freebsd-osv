# png -- memory corruption/possible remote code execution
The PNG project reports:

> libpng fails to correctly handle malloc() failures for text chunks (in
> png_set_text_2()), which can lead to memory corruption and the
> possibility of remote code execution.


## Affected packages
 - **png**
    - **Fixed:** 1.4.11
 - **linux-f10-png**
    - **Fixed:** 1.4.11
