# phpmyfaq -- Remote PHP Code Execution Vulnerability
The phpMyFAQ project reports:

> The bundled ImageManager library allows injection of arbitrary PHP
> code to execute arbitrary PHP code and upload malware and trojan
> horses.


## Affected packages
 - **phpmyfaq**
    - **Fixed:** 2.7.5
