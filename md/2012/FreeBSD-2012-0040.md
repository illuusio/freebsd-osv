# phpMyAdmin -- XSS in replication setup
The phpMyAdmin development team reports:

> It was possible to conduct XSS using a crafted database name.


## Affected packages
 - **phpMyAdmin**
    - **Introduced:** 3.4,1
    - **Fixed:** 3.4.10.1
