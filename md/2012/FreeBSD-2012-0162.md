# asterisk -- multiple vulnerabilities
Asterisk project reports:

> Asterisk Manager User Unauthorized Shell Access
>
> ACL rules ignored when placing outbound calls by certain IAX2 users


## Affected packages
 - **asterisk**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.7.1
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.15.1
