# asterisk -- multiple vulnerabilities
Asterisk project reports:

> Possible resource leak on uncompleted re-invite transactions.
>
> Remote crash vulnerability in voice mail application.


## Affected packages
 - **asterisk**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.5.2
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.13.1
