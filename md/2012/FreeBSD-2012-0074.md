# asterisk -- multiple vulnerabilities
Asterisk project reports:

> Remote Crash Vulnerability in SIP Channel Driver
>
> Heap Buffer Overflow in Skinny Channel Driver
>
> Asterisk Manager User Unauthorized Shell Access


## Affected packages
 - **asterisk16**
    - **Introduced:** 1.6.*,1
    - **Fixed:** 1.6.2.24
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.11.1
 - **asterisk10**
    - **Introduced:** 10.*,1
    - **Fixed:** 10.3.1
