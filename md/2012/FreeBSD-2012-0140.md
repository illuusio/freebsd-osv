# phpMyAdmin -- Path disclosure due to missing library
The phpMyAdmin development team reports:

> The show_config_errors.php script does not include a library, so an
> error message shows the full path of this file, leading to possible
> further attacks.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 3.5.2.1
