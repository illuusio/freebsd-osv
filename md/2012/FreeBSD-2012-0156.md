# Calligra, KOffice -- input validation failure
KDE Security Advisory reports:

> A flaw has been found which can allow malicious code to take advantage
> of an input validation failure in the Microsoft import filter in
> Calligra and KOffice. Exploitation can allow the attacker to gain
> control of the running process and execute code on its behalf.


## Affected packages
 - **koffice**
    - **Fixed:** 1.6.3_18,2
 - **koffice-kde4**
    - **Fixed:** 2.3.3_7
 - **calligra**
    - **Fixed:** 2.5.0
