# php5-sqlite -- open_basedir bypass
MITRE CVE team reports:

> The SQLite functionality in PHP before 5.3.15 allows remote attackers
> to bypass the open_basedir protection mechanism via unspecified
> vectors.


## Affected packages
 - **php5-sqlite**
    - **Introduced:** 5.2
    - **Fixed:** 5.2.17_11
    - **Introduced:** 5.3
    - **Fixed:** 5.3.15
 - **php52-sqlite**
    - **Fixed:** 5.2.17_11
 - **php53-sqlite**
    - **Fixed:** 5.3.15
