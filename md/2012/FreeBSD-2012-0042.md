# libxml2 -- heap buffer overflow
Google chrome team reports:

> Heap-based buffer overflow in libxml2, allows remote attackers to
> cause a denial of service or possibly have unspecified other impact
> via unknown vectors.


## Affected packages
 - **libxml2**
    - **Fixed:** 2.7.8_2
 - **linux-f10-libxml2**
    - **Fixed:** 2.7.8_2
