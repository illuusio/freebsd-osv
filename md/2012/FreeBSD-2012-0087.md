# PivotX -- 'ajaxhelper.php' Cross Site Scripting Vulnerability
High-Tech Bridge reports:

> Input passed via the \"file\" GET parameter to /pivotx/ajaxhelper.php
> is not properly sanitised before being returned to the user. This can
> be exploited to execute arbitrary HTML and script code in
> administrator\'s browser session in context of the affected website.


## Affected packages
 - **pivotx**
    - **Fixed:** 2.3.2
