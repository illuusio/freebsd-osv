# FreeBSD -- Insufficient message length validation for EAP-TLS messages
Problem description:

> The internal authentication server of hostapd does not sufficiently
> validate the message length field of EAP-TLS messages.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 8.3
    - **Fixed:** 8.3_5
    - **Introduced:** 9.0
    - **Fixed:** 9.0_5
