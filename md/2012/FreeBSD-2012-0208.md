# DomainKeys Identified Mail (DKIM) Verifiers may inappropriately convey message trust
US-CERT reports:

> DomainKeys Identified Mail (DKIM) Verifiers may inappropriately convey
> message trust when messages are signed using test or small bit signing
> keys.


## Affected packages
 - **opendkim**
    - **Fixed:** 2.7.1
