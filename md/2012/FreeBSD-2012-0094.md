# inspircd -- buffer overflow
InspIRCd reports:

> InspIRCd contains a heap corruption vulnerability that exists in the
> dns.cpp code. The res\[\] buffer is allocated on the heap and can be
> overflowed. The res\[\] buffer can be exploited during its
> deallocation. The number of overflowed bytes can be controlled with
> DNS compression features.


## Affected packages
 - **inspircd**
    - **Introduced:** 1.2
    - **Fixed:** 1.2.9
    - **Introduced:** 2.0
    - **Fixed:** 2.0.5_2
