# mozilla -- multiple vulnerabilities
The Mozilla Project reports:

> MFSA 2012-34 Miscellaneous memory safety hazards (rv:13.0/ rv:10.0.5)
>
> MFSA 2012-36 Content Security Policy inline-script bypass
>
> MFSA 2012-37 Information disclosure though Windows file shares and
> shortcut files
>
> MFSA 2012-38 Use-after-free while replacing/inserting a node in a
> document
>
> MFSA 2012-39 NSS parsing errors with zero length items
>
> MFSA 2012-40 Buffer overflow and use-after-free issues found using
> Address Sanitizer


## Affected packages
 - **firefox**
    - **Introduced:** 11.0,1,1
    - **Fixed:** 13.0,1
    - **Fixed:** 10.0.5,1
 - **linux-firefox**
    - **Fixed:** 10.0.5,1
 - **linux-seamonkey**
    - **Fixed:** 2.10
 - **linux-thunderbird**
    - **Fixed:** 10.0.5
 - **seamonkey**
    - **Fixed:** 2.10
 - **thunderbird**
    - **Introduced:** 11.0,1
    - **Fixed:** 13.0
    - **Fixed:** 10.0.5
 - **libxul**
    - **Introduced:** 1.9.2.*,1
    - **Fixed:** 10.0.5
