# py-Scrapy -- cookie injection vulnerability
> Responses from domain names whose public domain name suffix contains 1
> or more periods (e.g. responses from \`example.co.uk\`, given its
> public domain name suffix is \`co.uk\`) are able to set cookies that
> are included in requests to any other domain sharing the same domain
> name suffix.
