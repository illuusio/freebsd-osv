# redis -- Heap overflow in the cjson and cmsgpack libraries
Redis core team reports:

> A specially crafted Lua script executing in Redis can trigger a heap
> overflow in the cjson and cmsgpack libraries, and result in heap
> corruption and potentially remote code execution.


## Affected packages
 - **redis**
    - **Fixed:** 7.0.12
 - **redis-devel**
    - **Fixed:** 7.0.12.20230710
 - **redis62**
    - **Fixed:** 6.2.13
 - **redis60**
    - **Fixed:** 6.0.20
