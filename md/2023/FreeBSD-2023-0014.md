# redis -- multiple vulnerabilities
The Redis core team reports:

> 
>
> CVE-2022-35977
> :   Integer overflow in the Redis SETRANGE and SORT/SORT_RO commands
>     can drive Redis to OOM panic.
>
> CVE-2023-22458
> :   Integer overflow in the Redis HRANDFIELD and ZRANDMEMBER commands
>     can lead to denial-of-service.


## Affected packages
 - **redis**
    - **Fixed:** 7.0.8
 - **redis-devel**
    - **Fixed:** 7.0.8.20230116
 - **redis62**
    - **Fixed:** 6.2.9
 - **redis6**
    - **Fixed:** 6.0.17
