# Grafana -- Stored XSS in Graphite FunctionDescription tooltip
Grafana Labs reports:

> When a user adds a Graphite data source, they can then use the data
> source in a dashboard. This capability contains a feature to use
> Functions. Once a function is selected, a small tooltip appears when
> hovering over the name of the function. This tooltip allows you to
> delete the selected Function from your query or show the Function
> Description. However, no sanitization is done when adding this
> description to the DOM.
>
> Since it is not uncommon to connect to public data sources, an
> attacker could host a Graphite instance with modified Function
> Descriptions containing XSS payloads. When the victim uses it in a
> query and accidentally hovers over the Function Description, an
> attacker-controlled XSS payload will be executed.
>
> The severity of this vulnerability is of CVSSv3.1 5.7 Medium (CVSS:
> AV:N/AC:H/PR:H/UI:R/S:U/C:H/I:H/A:N (5.7)).
