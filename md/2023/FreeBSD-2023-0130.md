# py-tensorflow -- unchecked argument causing crash
Jingyi Shi reports:

> The \'AvgPoolOp\' function takes an argument \`ksize\` that must be
> positive but is not checked.
>
> A negative \`ksize\` can trigger a \`CHECK\` failure and crash the
> program.
