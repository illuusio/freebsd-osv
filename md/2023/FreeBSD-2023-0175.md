# qpress -- directory traversal
cve@mitre.org reports:

> qpress before PierreLvx/qpress 20220819 and before version 11.3, as
> used in Percona XtraBackup and other products, allows directory
> traversal via ../ in a .qp file.


## Affected packages
 - **qpress**
    - **Fixed:** 11.3
 - **xtrabackup8**
    - **Fixed:** 8.0.32
