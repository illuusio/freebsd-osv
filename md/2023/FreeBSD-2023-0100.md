# py39-rencode -- infinite loop that could lead to Denial of Service
NIST reports:

> The rencode package through 1.0.6 for Python allows an infinite loop
> in typecode decoding (such as via ;\\x2f\\x7f), enabling a remote
> attack that consumes CPU and memory.


## Affected packages
 - **py39-rencode**
    - **Fixed:** 1.0.6_1
