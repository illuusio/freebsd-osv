# py-httpx -- input validation vulnerability
lebr0nli reports:

> Encode OSS httpx \<=1.0.0.beta0 is affected by improper input
> validation in \`httpx.URL\`, \`httpx.Client\` and some functions using
> \`httpx.URL.copy_with\`.
