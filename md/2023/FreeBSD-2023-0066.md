# Grafana -- Stored XSS in text panel plugin
Grafana Labs reports:

> During an internal audit of Grafana on January 1, a member of the
> security team found a stored XSS vulnerability affecting the core text
> plugin.
>
> The stored XSS vulnerability requires several user interactions in
> order to be fully exploited. The vulnerability was possible due to
> React's render cycle that will pass through the unsanitized HTML code,
> but in the next cycle, the HTML is cleaned up and saved in Grafana's
> database.
>
> The CVSS score for this vulnerability is 6.4 Medium
> (CVSS:6.4/AV:N/AC:H/PR:L/UI:R/S:U/C:H/I:H/A:N).
