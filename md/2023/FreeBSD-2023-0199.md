# phpldapadmin -- XSS vulnerability
cve@mitre.org reports:

> An XSS issue has been discovered in phpLDAPadmin before 1.2.6.2 that
> allows users to store malicious values that may be executed by other
> users at a later time via get_request in lib/function.php.


## Affected packages
 - **phpldapadmin-php80**
    - **Fixed:** 1.2.6.2
 - **phpldapadmin-php81**
    - **Fixed:** 1.2.6.2
