# virtualbox-ose -- multiple vulnerabilities
secalert_us@oracle.com reports:

> Vulnerability in the Oracle VM VirtualBox product of Oracle
> Virtualization (component: Core). Supported versions that are affected
> are Prior to 6.1.46 and Prior to 7.0.10. Difficult to exploit
> vulnerability allows unauthenticated attacker with network access via
> RDP to compromise Oracle VM VirtualBox. Successful attacks of this
> vulnerability can result in takeover of Oracle VM VirtualBox. CVSS 3.1
> Base Score 8.1 (Confidentiality, Integrity and Availability impacts).
> CVSS Vector: (CVSS:3.1/AV:N/AC:H/PR:N/UI:N/S:U/C:H/I:H/A:H).


## Affected packages
 - **virtualbox-ose**
    - **Fixed:** 6.1.46
