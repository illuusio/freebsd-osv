# redis -- specially crafted MSETNX command can lead to denial-of-service
Yupeng Yang reports:

> Authenticated users can use the MSETNX command to trigger a runtime
> assertion and termination of the Redis server process.


## Affected packages
 - **redis**
    - **Fixed:** 7.0.10
 - **redis-devel**
    - **Fixed:** 7.0.10.20230320
