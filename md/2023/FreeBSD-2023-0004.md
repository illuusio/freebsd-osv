# net-mgmt/cacti is vulnerable to remote command injection
cacti team reports:

> A command injection vulnerability allows an unauthenticated user to
> execute arbitrary code on a server running Cacti, if a specific data
> source was selected for any monitored device.


## Affected packages
 - **cacti**
    - **Fixed:** 1.2.23
