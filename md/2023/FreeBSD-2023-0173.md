# OpenSSL -- Possible DoS translating ASN.1 identifiers
The OpenSSL project reports:

> Severity: Moderate. Processing some specially crafted ASN.1 object
> identifiers or data containing them may be very slow.


## Affected packages
 - **openssl**
    - **Fixed:** 1.1.1u,1
 - **openssl30**
    - **Fixed:** 3.0.9
 - **openssl31**
    - **Fixed:** 3.1.1
 - **openssl-quictls**
    - **Fixed:** 3.0.9
