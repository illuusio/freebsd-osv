# postgresql-server -- CREATE SCHEMA ... schema elements defeats protective search_path changes
PostgreSQL Project reports

> This enabled an attacker having database-level CREATE privilege to
> execute arbitrary code as the bootstrap superuser. Database owners
> have that right by default, and explicit grants may extend it to other
> users.
