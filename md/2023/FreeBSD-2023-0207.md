# OpenSSL -- AES-SIV implementation ignores empty associated data entries
The OpenSSL project reports:

> The AES-SIV cipher implementation contains a bug that causes it to
> ignore empty associated data entries which are unauthenticated as a
> consequence.


## Affected packages
 - **openssl30**
    - **Fixed:** 3.0.9_1
 - **openssl31**
    - **Fixed:** 3.1.1_1
