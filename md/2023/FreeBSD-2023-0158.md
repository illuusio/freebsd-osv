# redis -- HINCRBYFLOAT can be used to crash a redis-server process
Redis core team reports:

> Authenticated users can use the HINCRBYFLOAT command to create an
> invalid hash field that may later crash Redis on access.


## Affected packages
 - **redis**
    - **Fixed:** 7.0.11
 - **redis62**
    - **Fixed:** 6.2.12
 - **redis6**
    - **Fixed:** 6.0.19
