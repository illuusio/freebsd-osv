# LibreSSL -- Arbitrary memory read
The OpenBSD project reports:

> A malicious certificate revocation list or timestamp response token
> would allow an attacker to read arbitrary memory.


## Affected packages
 - **libressl**
    - **Fixed:** 3.5.4
 - **libressl-devel**
    - **Fixed:** 3.6.2
