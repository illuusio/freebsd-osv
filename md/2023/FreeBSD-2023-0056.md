# git -- "git apply" overwriting paths outside the working tree
git team reports:

> By feeding a crafted input to \"git apply\", a path outside the
> working tree can be overwritten as the user who is running \"git
> apply\".


## Affected packages
 - **git**
    - **Fixed:** 2.39.2
