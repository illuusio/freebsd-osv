# go -- crypto/elliptic: incorrect P-256 ScalarMult and ScalarBaseMult results
The Go project reports:

> crypto/elliptic: incorrect P-256 ScalarMult and ScalarBaseMult results
>
> The ScalarMult and ScalarBaseMult methods of the P256 Curve may return
> an incorrect result if called with some specific unreduced scalars (a
> scalar larger than the order of the curve).


## Affected packages
 - **go119**
    - **Fixed:** 1.19.7
 - **go120**
    - **Fixed:** 1.20.2
