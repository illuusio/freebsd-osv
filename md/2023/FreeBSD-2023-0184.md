# vscode -- VS Code Information Disclosure Vulnerability
VSCode developers reports:

> VS Code Information Disclosure Vulnerability
>
> A information disclosure vulnerability exists in VS Code 1.79.0 and
> earlier versions on Windows when file system operations are performed
> on malicious UNC paths. Examples include reading or resolving metadata
> of such paths. An authorised attacker must send the user a malicious
> file and convince the user to open it for the vulnerability to occur.
> Exploiting this vulnerability could allow the disclosure of NTLM
> hashes.


## Affected packages
 - **vscode**
    - **Fixed:** 1.79.1
