# py-markdown2 -- regular expression denial of service vulnerability
Ben Caller reports:

> markdown2 \>=1.0.1.18, fixed in 2.4.0, is affected by a regular
> expression denial of service vulnerability.
>
> If an attacker provides a malicious string, it can make markdown2
> processing difficult or delayed for an extended period of time.


## Affected packages
 - **py37-markdown2**
    - **Fixed:** 2.4.0
 - **py38-markdown2**
    - **Fixed:** 2.4.0

    - **Fixed:** 2.4.0
 - **py310-markdown2**
    - **Fixed:** 2.4.0
 - **py311-markdown2**
    - **Fixed:** 2.4.0
