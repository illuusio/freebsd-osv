# Grafana -- Broken access control: viewer can send test alerts
Grafana Labs reports:

> Grafana can allow an attacker in the **Viewer** role to send alerts by
> **API Alert - Test**. This option, however, is not available in the
> user panel UI for the Viewer role.
>
> The CVSS score for this vulnerability is 4.1 Medium
> (CVSS:3.1/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:N/A:N).
