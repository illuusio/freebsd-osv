# Grafana -- Grafana DS proxy race condition
Grafana Labs reports:

> We have discovered a vulnerability with Grafana's data source query
> endpoints that could end up crashing a Grafana instance.
>
> If you have public dashboards (PD) enabled, we are scoring this as a
> CVSS 7.5 High.
>
> If you have disabled PD, this vulnerability is still a risk, but
> triggering the issue requires data source read privileges and access
> to the Grafana API through a developer script.


## Affected packages
 - **grafana**
    - **Introduced:** 9.4.0
    - **Fixed:** 9.4.12
    - **Introduced:** 9.5.0
    - **Fixed:** 9.5.3
 - **grafana9**
    - **Introduced:** 9.4.0
    - **Fixed:** 9.4.12
    - **Introduced:** 9.5.0
    - **Fixed:** 9.5.3
