# py39-OWSLib -- arbitrary file read vulnerability
Jorge Rosillo reports:

> OWSLib\'s XML parser (which supports both \`lxml\` and \`xml.etree\`)
> does not disable entity resolution for \`lxml\`, and could lead to
> arbitrary file reads from an attacker-controlled XML payload.
>
> This affects all XML parsing in the codebase.


## Affected packages
 - **py39-OWSLib**
    - **Fixed:** 0.28.1
