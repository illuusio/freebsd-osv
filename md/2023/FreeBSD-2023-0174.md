# Kanboard -- Multiple vulnerabilities
Kanboard is project management software that focuses on the Kanban
methodology. The last update includes 4 vulnerabilities:

security-advisories@github.com reports:

> - Missing access control in internal task links feature
> - Stored Cross site scripting in the Task External Link Functionality
>   in Kanboard
> - Missing Access Control allows User to move and duplicate tasks in
>   Kanboard
> - Parameter based Indirect Object Referencing leading to private file
>   exposure in Kanboard


## Affected packages
 - **php80-kanboard**
    - **Fixed:** 1.2.30
