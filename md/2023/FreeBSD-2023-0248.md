# py-dparse -- REDoS vulnerability
yeisonvargasf reports:

> dparse is a parser for Python dependency files.
>
> dparse in versions before 0.5.2 contain a regular expression that is
> vulnerable to a Regular Expression Denial of Service.
>
> All the users parsing index server URLs with dparse are impacted by
> this vulnerability.
>
> Users unable to upgrade should avoid passing index server URLs in the
> source file to be parsed.


## Affected packages
 - **py37-dparse**
    - **Fixed:** 0.5.2
 - **py38-dparse**
    - **Fixed:** 0.5.2
 - **py39-dparse**
    - **Fixed:** 0.5.2
 - **py310-dparse**
    - **Fixed:** 0.5.2
 - **py311-dparse**
    - **Fixed:** 0.5.2
