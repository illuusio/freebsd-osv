# MariaDB -- Nullpointer dereference
The MariaDB project reports:

> MariaDB Server is vulnerable to Denial of Service. It is possible for
> function spider_db_mbase::print_warnings to dereference a null
> pointer.


## Affected packages
 - **mariadb1011-server**
    - **Fixed:** 10.11.3
 - **mariadb106-server**
    - **Fixed:** 10.6.13
 - **mariadb105-server**
    - **Fixed:** 10.5.20
 - **mariadb104-server**
    - **Fixed:** 10.4.29
 - **mariadb103-server**
    - **Fixed:** 10.3.39
