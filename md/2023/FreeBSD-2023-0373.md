# GLPI vulnerable to unauthenticated access to Dashboard data
security-advisories@github.com reports:

> GLPI is a free asset and IT management software package. Starting in
> version 9.5.0 and prior to version 10.0.8, an incorrect rights check
> on a file allows an unauthenticated user to be able to access
> dashboards data. Version 10.0.8 contains a patch for this issue.


## Affected packages
 - **glpi**
    - **Introduced:** 9.5.0,1
    - **Fixed:** 10.0.8,1
