# py39-sqlalchemy11 -- multiple SQL Injection vulnerabilities
21k reports:

> SQLAlchemy through 1.2.17 and 1.3.x through 1.3.0b2 allows SQL
> Injection via the order_by parameter.

nosecurity reports:

> SQLAlchemy 1.2.17 has SQL Injection when the group_by parameter can be
> controlled.


## Affected packages
 - **py39-sqlalchemy11**
    - **Fixed:** 1.3.0
