# FreeBSD -- arm64 boot CPUs may lack speculative execution protections
# Problem Description:

On CPU 0 the check for the SMCCC workaround is called before SMCCC
support has been initialized.

# Impact:

No speculative execution workarounds are installed on CPU 0.


## Affected packages
 - **FreeBSD-kernel**
    - **Introduced:** 13.2
    - **Fixed:** 13.2_4
