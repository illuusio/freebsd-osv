# py-nicotine-plus -- Denial of service vulnerability
ztauras reports:

> Denial of service (DoS) vulnerability in Nicotine+ starting with
> version 3.0.3 and prior to version 3.2.1 allows a user with a modified
> Soulseek client to crash Nicotine+ by sending a file download request
> with a file path containing a null character.


## Affected packages
 - **py37-nicotine-plus**
    - **Fixed:** 3.2.1
 - **py38-nicotine-plus**
    - **Fixed:** 3.2.1
 - **py39-nicotine-plus**
    - **Fixed:** 3.2.1
 - **py310-nicotine-plus**
    - **Fixed:** 3.2.1
 - **py311-nicotine-plus**
    - **Fixed:** 3.2.1
