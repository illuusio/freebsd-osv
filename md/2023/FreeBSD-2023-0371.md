# GLPI vulnerable to SQL injection through Computer Virtual Machine information
security-advisories@github.com reports:

> GLPI is a free asset and IT management software package. Starting in
> version 0.80 and prior to version 10.0.8, Computer Virtual Machine
> form and GLPI inventory request can be used to perform a SQL injection
> attack. Version 10.0.8 has a patch for this issue. As a workaround,
> one may disable native inventory.


## Affected packages
 - **glpi**
    - **Fixed:** 10.0.8,1
