# py39-Elixir -- weak use of cryptography
Red Hat Security Response Team reports:

> Elixir 0.8.0 uses Blowfish in CFB mode without constructing a unique
> initialization vector (IV), which makes it easier for
> context-dependent users to obtain sensitive information and decrypt
> the database.


## Affected packages
 - **py39-Elixir**
    - **Fixed:** 0.8.0
