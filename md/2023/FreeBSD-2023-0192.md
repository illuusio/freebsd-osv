# Grafana -- Account takeover / authentication bypass
Grafana Labs reports:

> Grafana validates Azure Active Directory accounts based on the email
> claim. On Azure AD, the profile email field is not unique across Azure
> AD tenants. This can enable a Grafana account takeover and
> authentication bypass when Azure AD OAuth is configured with a
> multi-tenant Azure AD OAuth application.
>
> The CVSS score for this vulnerability is 9.4 Critical.
