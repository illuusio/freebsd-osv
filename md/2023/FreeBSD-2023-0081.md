# tailscale -- security vulnerability in Tailscale SSH
Tailscale team reports:

> A vulnerability identified in the implementation of Tailscale SSH in
> FreeBSD allowed commands to be run with a higher privilege group ID
> than that specified by Tailscale SSH access rules.


## Affected packages
 - **tailscale**
    - **Fixed:** 1.38.2
