# MinIO -- unprivileged users can create service accounts for admin users
MinIO reports:

> A security issue was found where an unprivileged user is able to
> create service accounts for root or other admin users and then is able
> to assume their access policies via the generated credentials.


## Affected packages
 - **minio**
    - **Fixed:** 2022.04.12.06.55.35
