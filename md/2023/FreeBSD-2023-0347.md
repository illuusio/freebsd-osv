# strongSwan -- vulnerability in charon-tkm
strongSwan reports:

> A vulnerability in charon-tkm related to processing DH public values
> was discovered in strongSwan that can result in a buffer overflow and
> potentially remote code execution. All versions since 5.3.0 are
> affected.


## Affected packages
 - **strongswan**
    - **Introduced:** 5.3.0
    - **Fixed:** 5.9.11_3
