# rubygem-uri -- ReDoS vulnerability
Dominic Couture reports:

> A ReDoS issue was discovered in the URI component. The URI parser
> mishandles invalid URLs that have specific characters. It causes an
> increase in execution time for parsing strings to URI objects.
