# MySQL -- Multiple vulnerabilities
Oracle reports:

> This Critical Patch Update contains 37 new security patches for Oracle
> MySQL. 8 of these vulnerabilities may be remotely exploitable without
> authentication, i.e., may be exploited over a network withouti
> requiring user credentials.


## Affected packages
 - **mysql-connector-c++**
    - **Fixed:** 8.0.33
 - **mysql-connector-odbc**
    - **Fixed:** 8.0.33
 - **mysql-client57**
    - **Fixed:** 5.7.42
 - **mysql-server57**
    - **Fixed:** 5.7.42
 - **mysql-client80**
    - **Fixed:** 8.0.33
 - **mysql-server80**
    - **Fixed:** 8.0.33
