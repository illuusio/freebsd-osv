# cups -- potential buffer overflow in PNG reading code
CUPS reports:

> The PNG image reading code did not validate the image size properly,
> leading to a potential buffer overflow (STR #2974)


## Affected packages
 - **cups-base**
    - **Fixed:** 1.3.9_2
