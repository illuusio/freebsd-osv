# pcre -- buffer overflow vulnerability
PCRE developers report:

> A character class containing a very large number of characters with
> codepoints greater than 255 (in UTF-8 mode, of course) caused a buffer
> overflow.


## Affected packages
 - **pcre**
    - **Fixed:** 7.6
