# mysql -- MyISAM table privileges security bypass vulnerability
SecurityFocus reports:

> MySQL is prone to a security-bypass vulnerability. An attacker can
> exploit this issue to overwrite existing table files in the MySQL data
> directory, bypassing certain security restrictions.


## Affected packages
 - **mysql-server**
    - **Introduced:** 6.0
    - **Fixed:** 6.0.5
    - **Introduced:** 5.1
    - **Fixed:** 5.1.24
    - **Introduced:** 5.0
    - **Fixed:** 5.0.67
    - **Introduced:** 4.1
    - **Fixed:** 4.1.22_1
