# mozilla -- multiple vulnerabilities
The Mozilla Foundation reports of multiple security issues in Firefox,
Seamonkey, and Thunderbird. Several of these issues can probably be used
to run arbitrary code with the privilege of the user running the
program.

> - [MFSA 2008-19](/security/announce/2008/mfsa2008-19.html) XUL popup
>   spoofing variant (cross-tab popups)
> - [MFSA 2008-18](/security/announce/2008/mfsa2008-18.html) Java socket
>   connection to any local port via LiveConnect
> - [MFSA 2008-17](/security/announce/2008/mfsa2008-17.html) Privacy
>   issue with SSL Client Authentication
> - [MFSA 2008-16](/security/announce/2008/mfsa2008-16.html) HTTP
>   Referrer spoofing with malformed URLs
> - [MFSA 2008-15](/security/announce/2008/mfsa2008-15.html) Crashes
>   with evidence of memory corruption (rv:1.8.1.13)
> - [MFSA 2008-14](/security/announce/2008/mfsa2008-14.html) JavaScript
>   privilege escalation and arbitrary code execution
