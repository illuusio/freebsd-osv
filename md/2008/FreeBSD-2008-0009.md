# IRC Services-- Denial of Service Vulnerability
Secunia reports:

> A vulnerability has been reported in IRC Services, which can be
> exploited by malicious people to cause a Denial of Service. The
> vulnerability is caused due to the improper handling of overly long
> passwords within the \"default_encrypt()\" function in encrypt.c and
> can be exploited to crash an affected server.


## Affected packages
 - **ircservices**
    - **Fixed:** 5.0.63
