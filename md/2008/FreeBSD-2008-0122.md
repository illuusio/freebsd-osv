# wordpress -- snoopy "_httpsrequest()" shell command execution vulnerability
The Wordpress development team reports:

> A [vulnerability](http://secunia.com/Advisories/32361/) in the
> [Snoopy](http://sourceforge.net/projects/snoopy/) library was
> announced today. WordPress uses Snoopy to fetch the feeds shown in the
> Dashboard. Although this seems to be a low risk vulnerability for
> WordPress users, we wanted to get an update out immediately.


## Affected packages
 - **wordpress**
    - **Fixed:** 2.6.3
 - **de-wordpress**
    - **Fixed:** 2.6.3
 - **wordpress-mu**
    - **Fixed:** 2.6.3
