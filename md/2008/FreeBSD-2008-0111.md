# bitlbee -- account recreation security issues
Secunia reports:

> Some security issues have been reported in BitlBee, which can be
> exploited by malicious people to bypass certain security restrictions
> and hijack accounts.
>
> The security issues are caused due to unspecified errors, which can be
> exploited to overwrite existing accounts.


## Affected packages
 - **bitlbee**
    - **Fixed:** 1.2.3
