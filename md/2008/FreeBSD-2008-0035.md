# suphp -- multiple local privilege escalation vulnerabilities
Multiple local privilege escalation are found in the symlink
verification code. An attacker may use it to run a PHP script with the
victim\'s privilege. This attack is a little harder when suphp operates
in paranoid mode. For suphp that runs in owner mode which is the default
in ports, immediate upgrade to latest version is advised.


## Affected packages
 - **suphp**
    - **Fixed:** 0.6.3
