# hplip -- hpssd Denial of Service
Secunia reports:

> A security issue has been reported in hplip, which can be exploited by
> malicious, local users to cause a DoS.
>
> The security issue is caused due to an error within hpssd.py when
> parsing certain requests. This can be exploited to crash the service
> by sending specially crafted requests to the default port 2207/TCP.


## Affected packages
 - **hplip**
    - **Fixed:** 2.8.2_3
