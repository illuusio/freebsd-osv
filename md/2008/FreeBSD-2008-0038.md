# ikiwiki -- cross site request forging
The ikiwiki development team reports:

> Cross Site Request Forging could be used to construct a link that
> would change a logged-in user\'s password or other preferences if they
> clicked on the link. It could also be used to construct a link that
> would cause a wiki page to be modified by a logged-in user.


## Affected packages
 - **ikiwiki**
    - **Fixed:** 2.42
