# dovecot -- ACL plugin bypass vulnerabilities
Timo Sirainen reports in dovecot 1.1.4 release notes:

> ACL plugin fixes: Negative rights were actually treated as positive
> rights. \'k\' right didn\'t prevent creating parent/child/child
> mailbox. ACL groups weren\'t working.


## Affected packages
 - **dovecot**
    - **Fixed:** 1.1.4
