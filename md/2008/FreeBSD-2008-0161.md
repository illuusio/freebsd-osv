# phpmyadmin -- cross-site request forgery vulnerability
The phpMyAdmin Team reports:

> A logged-in user can be subject of SQL injection through cross site
> request forgery. Several scripts in phpMyAdmin are vulnerable and the
> attack can be made through table parameter.


## Affected packages
 - **phpMyAdmin211**
    - **Fixed:** 2.11.9.4
 - **phpMyAdmin**
    - **Fixed:** 3.1.1
