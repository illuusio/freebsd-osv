# ruby -- multiple integer and buffer overflow vulnerabilities
The official ruby site reports:

> Multiple vulnerabilities in Ruby may lead to a denial of service (DoS)
> condition or allow execution of arbitrary code.


## Affected packages
 - **ruby**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.6.111_3,1
 - **ruby+pthreads**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.6.111_3,1
 - **ruby+pthreads+oniguruma**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.6.111_3,1
 - **ruby+oniguruma**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.6.111_3,1
 - **ruby_static**
    - **Introduced:** 1.8.*,1
