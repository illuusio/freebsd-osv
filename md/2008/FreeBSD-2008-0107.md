# phpmyadmin -- Cross-Site Scripting Vulnerability
Secunia reports:

> An error exists in the \"PMA_escapeJsString()\" function in
> libraries/js_escape.lib.php, which can be exploited to bypass certain
> filters and execute arbitrary HTML and script code in a user\'s
> browser session in context of an affected site when e.g. Microsoft
> Internet Explorer is used.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.11.9.2
