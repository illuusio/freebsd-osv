# Nagios -- Cross Site Scripting Vulnerability
Secunia reports:

> A vulnerability has been reported in Nagios, which can be exploited by
> malicious people to conduct cross-site scripting attacks.


## Affected packages
 - **nagios**
    - **Fixed:** 2.12
    - **Introduced:** 3.*,1
    - **Fixed:** 3.0.2
 - **nagios-devel**
    - **Fixed:** 3.0.2
