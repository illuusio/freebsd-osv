# gallery -- multiple vulnerabilities
Secunia reports:

> An error in the handing of ZIP archives with symbolic links can be
> exploited to disclose the contents of arbitrary files.

> Input from uploaded Flash animations is not properly sanitised before
> being used. This can be exploited to insert arbitrary HTML and script
> code, which is executed in a user\'s browser session in context of an
> affected site when the malicious data is viewed.


## Affected packages
 - **gallery**
    - **Fixed:** 1.5.9
 - **gallery2**
    - **Fixed:** 2.2.6
