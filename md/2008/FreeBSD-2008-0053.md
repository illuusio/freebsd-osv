# extman -- password bypass vulnerability
Extmail team reports:

> Emergency update #4 fixes a serious security vulnerability.

Successful exploit of this vulnerability would allow attacker to change
user\'s password without knowing it by using specifically crafted HTTP
request.


## Affected packages
 - **extman**
    - **Introduced:** 0.2.4
    - **Fixed:** 0.2.4_1
