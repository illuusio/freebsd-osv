# xfce -- multiple vulnerabilities
Gentoo reports:

> A remote attacker could entice a user to install a specially crafted
> \"rc\" file to execute arbitrary code via long strings in the \"Name\"
> and \"Comment\" fields or via unspecified vectors involving the second
> vulnerability.


## Affected packages
 - **xfce4-panel**
    - **Fixed:** 4.4.2
 - **libxfce4gui**
    - **Fixed:** 4.4.2
