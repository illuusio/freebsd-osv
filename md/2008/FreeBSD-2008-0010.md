# claws-mail -- insecure temporary file creation
Nico Golde reports:

> A local attacker could exploit this vulnerability to conduct symlink
> attacks to overwrite files with the privileges of the user running
> Claws Mail.


## Affected packages
 - **claws-mail**
    - **Fixed:** 3.1.0
