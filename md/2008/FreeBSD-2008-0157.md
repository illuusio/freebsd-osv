# mgetty+sendfax -- symlink attack via insecure temporary files
Debian reports:

> Faxspool in mgetty 1.1.36 allows local users to overwrite arbitrary
> files via a symlink attack on a /tmp/faxsp.#### temporary file.


## Affected packages
 - **mgetty+sendfax**
    - **Fixed:** 1.1.35_2
