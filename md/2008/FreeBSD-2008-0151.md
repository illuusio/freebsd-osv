# squirrelmail -- Cross site scripting vulnerability
Squirrelmail team reports:

> An issue was fixed that allowed an attacker to send specially- crafted
> hyperlinks in a message that could execute cross-site scripting (XSS)
> when the user viewed the message in SquirrelMail.


## Affected packages
 - **squirrelmail**
    - **Fixed:** 1.4.17
