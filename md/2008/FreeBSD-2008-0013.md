# libxine -- buffer overflow vulnerability
xine project reports:

> A new xine-lib version is now available. This release contains a
> security fix (remotely-expoitable buffer overflow, CVE-2006-1664).
> (This is not the first time that that bug has been fixed\...) It also
> fixes a few more recent bugs, such as the audio output problems in
> 1.1.9.


## Affected packages
 - **libxine**
    - **Fixed:** 1.1.10
