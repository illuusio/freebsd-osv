# phpmyadmin -- Code execution vulnerability
A phpMyAdmin security announcement:

> The server_databases.php script was vulnerable to an attack coming
> from a user who is already logged-on to phpMyAdmin, where he can
> execute shell code (if the PHP configuration permits commands like
> exec).


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.11.9.1
