# libxine -- buffer overflow vulnerability
xine project reports:

> A new xine-lib version is now available. This release contains a
> security fix (remotely-expoitable buffer overflow, CVE-2008-0225). It
> also contains a read-past-end fix for an internal library function
> which is only used if the OS does not supply it and a rendering fix
> for Darwin/PPC.


## Affected packages
 - **libxine**
    - **Fixed:** 1.1.9.1
