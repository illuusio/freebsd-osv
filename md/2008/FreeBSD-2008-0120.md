# libxine -- denial of service vulnerability
xine team reports:

> A new xine-lib version is now available. This release contains some
> security fixes, notably a DoS via corrupted Ogg files (CVE-2008-3231),
> some related fixes, and fixes for a few possible buffer overflows.


## Affected packages
 - **libxine**
    - **Fixed:** 1.1.15
