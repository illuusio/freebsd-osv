# clamav -- CHM Processing Denial of Service
Hanno Boeck reports:

> A fuzzing test showed weakness in the chm parser of clamav, which can
> possibly be exploited. The clamav team has disabled the chm module in
> older versions though freshclam updates and has released 0.94 with a
> fixed parser.


## Affected packages
 - **clamav**
    - **Fixed:** 0.94
 - **clamav-devel**
    - **Fixed:** 20080902
