# openx -- sql injection vulnerability
Secunia reports:

> OpenX can be exploited by malicious people to conduct SQL injection
> attacks.
>
> Input passed to the \"bannerid\" parameter in www/delivery/ac.php is
> not properly sanitised before being used in SQL queries. This can be
> exploited to manipulate SQL queries by injecting arbitrary SQL code.


## Affected packages
 - **openx**
    - **Fixed:** 2.6.2
