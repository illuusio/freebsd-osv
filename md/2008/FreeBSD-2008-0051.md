# mksh -- TTY attachment privilege escalation
Secunia reports:

> The vulnerability is caused due to an error when attaching to a TTY
> via the -T command line switch. This can be exploited to execute
> arbitrary commands with the privileges of the user running mksh via
> characters previously written to the attached virtual console.


## Affected packages
 - **mksh**
    - **Fixed:** R33d
