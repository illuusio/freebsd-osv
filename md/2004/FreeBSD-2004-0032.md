# mod_python denial-of-service vulnerability in parse_qs
An attacker may cause Apache with mod_python to crash by using a
specially constructed query string.


## Affected packages
 - **mod_python**
    - **Introduced:** 2.7
    - **Fixed:** 2.7.10
    - **Introduced:** 3.0
    - **Fixed:** 3.0.4
