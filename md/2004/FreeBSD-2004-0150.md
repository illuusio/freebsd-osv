# samba3 DoS attack
Code found in nmbd and smbd may allow a remote attacker to effectively
crash the nmbd server or use the smbd server to exhaust the system
memory.


## Affected packages
 - **samba3**
    - **Fixed:** 3.0.7
    - **Introduced:** *,1,1
    - **Fixed:** 3.0.7,1
