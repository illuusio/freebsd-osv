# mysql -- heap buffer overflow with prepared statements
There is a buffer overflow in the prepared statements API
(libmysqlclient) when a statement containing thousands of placeholders
is executed.


## Affected packages
 - **mysql-server**
    - **Introduced:** 4.1.0
    - **Fixed:** 4.1.4
 - **mysql-client**
    - **Introduced:** 4.1.0
    - **Fixed:** 4.1.4
