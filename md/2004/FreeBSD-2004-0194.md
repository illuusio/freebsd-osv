# tiff -- RLE decoder heap overflows
Chris Evans discovered several heap buffer overflows in libtiff\'s RLE
decoder. These overflows could be triggered by a specially-crafted TIFF
image file, resulting in an application crash and possibly arbitrary
code execution.
