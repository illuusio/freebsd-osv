# apache -- heap overflow in mod_proxy
A buffer overflow exists in mod_proxy which may allow an attacker to
launch local DoS attacks and possibly execute arbitrary code.


## Affected packages
 - **apache**
    - **Fixed:** 1.3.31_1
 - **apache13-ssl**
    - **Fixed:** 1.3.29.1.53_2
 - **apache13-modssl**
    - **Fixed:** 1.3.31+2.8.18_4
 - **apache13+ipv6**
    - **Fixed:** 1.3.29_2
 - **apache13-modperl**
    - **Fixed:** 1.3.31
