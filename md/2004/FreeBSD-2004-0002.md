# ProFTPD ASCII translation bug resulting in remote root compromise
A buffer overflow exists in the ProFTPD code that handles translation of
newline characters during ASCII-mode file uploads. An attacker may
exploit this buffer overflow by uploading a specially crafted file,
resulting in code execution and ultimately a remote root compromise.


## Affected packages
 - **proftpd**
    - **Fixed:** 1.2.8_1
