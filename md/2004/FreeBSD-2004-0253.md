# unarj -- directory traversal vulnerability
unarj has insufficient checks for filenames that contain "..". This can
allow an attacker to overwrite arbitrary files with the permissions of
the user running unarj.


## Affected packages
 - **unarj**
    - **Fixed:** 2.43_2
