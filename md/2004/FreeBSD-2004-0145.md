# ImageMagick -- BMP decoder buffer overflow
Marcus Meissner discovered that ImageMagick\'s BMP decoder would crash
when loading the test BMP file created by Chris Evans for testing the
previous Qt vulnerability.


## Affected packages
 - **ImageMagick**
    - **Fixed:** 6.0.6.2
 - **ImageMagick-nox11**
    - **Fixed:** 6.0.6.2
