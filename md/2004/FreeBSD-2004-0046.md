# squid ACL bypass due to URL decoding bug
From the Squid advisory:

> Squid versions 2.5.STABLE4 and earlier contain a bug in the \"%xx\"
> URL decoding function. It may insert a NUL character into decoded
> URLs, which may allow users to bypass url_regex ACLs.


## Affected packages
 - **squid**
    - **Fixed:** 2.5.5
