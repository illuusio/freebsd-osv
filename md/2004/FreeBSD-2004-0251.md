# ProZilla -- server response buffer overflow vulnerabilities
Buffer overflow vulnerabilities have been reported to exist in this
software package. The vulnerabilities can be triggered by a remote
server and can be used to inject malicious code in the ProZilla process.


## Affected packages
 - **prozilla**
    - **Fixed:** 1.3.6_3
