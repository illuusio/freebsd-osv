# gnats -- format string vulnerability
Gnats suffers from a format string bug, which may enable an attacker to
execute arbitary code.

 - **gnats**
    - **Introduced:** 4.*
    - **Fixed:** 4.0_2
    - **Fixed:** 3.113.1_9
