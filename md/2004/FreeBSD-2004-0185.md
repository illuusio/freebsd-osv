# php -- vulnerability in RFC 1867 file upload processing
Stefano Di Paola discovered an issue with PHP that could allow someone
to upload a file to any directory writeable by the httpd process. Any
sanitizing performed on the prepended directory path is ignored. This
bug can only be triggered if the \$\_FILES element name contains an
underscore.


## Affected packages
 - **php4**
    - **Fixed:** 4.3.8_2
 - **php4-cgi**
    - **Fixed:** 4.3.8_2
 - **mod_php4**
    - **Fixed:** 4.3.8_2,1
 - **php5**
    - **Fixed:** 5.0.1
 - **php5-cgi**
    - **Fixed:** 5.0.1
 - **mod_php5**
    - **Fixed:** 5.0.1,1
