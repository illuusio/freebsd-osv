# libxine -- buffer-overflow vulnerability in aiff support
Due to a buffer overflow in the open_aiff_file function in demux_aiff.c,
a remote attacker is able to execute arbitrary code via a modified AIFF
file.


## Affected packages
 - **libxine**
    - **Fixed:** 1.0.r5_3
