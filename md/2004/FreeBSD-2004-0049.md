# racoon fails to verify signature during Phase 1
Ralf Spenneberg discovered a serious flaw in racoon. When using Phase 1
main or aggressive mode, racoon does not verify the client\'s RSA
signature. Any installations using *X.509 authentication* are **strongly
urged** to upgrade.

Installations using *pre-shared keys* are believed to be unaffected.


## Affected packages
 - **racoon**
    - **Fixed:** 20040407b
