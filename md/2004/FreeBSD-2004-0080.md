# MySQL insecure temporary file creation (mysqlbug)
Shaun Colley reports that the script \`mysqlbug\' included with MySQL
sometimes creates temporary files in an unsafe manner. As a result, an
attacker may create a symlink in /tmp so that if another user invokes
\`mysqlbug\' and *quits without making **any** changes*, an arbitrary
file may be overwritten with the bug report template.


## Affected packages
 - **mysql-client**
    - **Introduced:** 4.0
    - **Fixed:** 4.0.20
    - **Introduced:** 4.1
    - **Fixed:** 4.1.1_2
    - **Introduced:** 5.0
    - **Fixed:** 5.0.0_2
