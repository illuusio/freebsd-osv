# insecure temporary file creation in xine-check, xine-bugreport
Some scripts installed with xine create temporary files insecurely. It
is recommended that these scripts (xine-check, xine-bugreport) not be
used. They are not needed for normal operation.


## Affected packages
 - **xine**
    - **Fixed:** 0.9.23_3
