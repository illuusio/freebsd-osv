# oftpd denial-of-service vulnerability (PORT command)
Philippe Oechslin reported a denial-of-service vulnerability in oftpd.
The oftpd server can be crashed by sending a PORT command containing an
integer over 8 bits long (over 255).


## Affected packages
 - **oftpd**
    - **Fixed:** 0.3.7
