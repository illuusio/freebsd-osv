# Buffer overflow in Mutt 1.4
Mutt 1.4 contains a buffer overflow that could be exploited with a
specially formed message, causing Mutt to crash or possibly execute
arbitrary code.


## Affected packages
 - **mutt**
    - **Introduced:** 1.4
    - **Fixed:** 1.4.2
 - **ja-mutt**
    - **Introduced:** 1.4
    - **Fixed:** 1.4.2
