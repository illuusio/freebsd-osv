# postgresql-contrib -- insecure temporary file creation
The make_oidjoins_check script in the PostgreSQL RDBMS has insecure
handling of temporary files, which could lead to an attacker overwriting
arbitrary files with the credentials of the user running the
make_oidjoins_check script.


## Affected packages
 - **postgresql-contrib**
    - **Fixed:** 7.2.6
    - **Introduced:** 7.3.*,1
    - **Fixed:** 7.3.8
    - **Introduced:** 7.4.*,1
    - **Fixed:** 7.4.6
