# icecast -- Cross-Site Scripting Vulnerability
Caused by improper filtering of HTML code in the status display, it is
possible for a remote user to execute scripting code in the target
user\'s browser.


## Affected packages
 - **icecast**
    - **Fixed:** 1.3.12_2
