# mysql -- FTS request denial of service vulnerability
A special crafted MySQL FTS request can cause the server to crash.
Malicious MySQL users can abuse this bug in a denial of service attack
against systems running an affected MySQL daemon.

Note that because this bug is related to the parsing of requests, it may
happen that this bug is triggered accidently by a user when he or she
makes a typo.


## Affected packages
 - **mysql-server**
    - **Introduced:** 4.*
    - **Fixed:** 4.0.21
