# bmon -- unsafe set-user-ID application
Jon Nistor reported that the FreeBSD port of bmon was installed
set-user-ID root, and executes commands using relative paths. This could
allow local user to easily obtain root privileges.


## Affected packages
 - **bmon**
    - **Fixed:** 1.2.1_2
