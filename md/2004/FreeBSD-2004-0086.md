# Vulnerabilities in H.323 implementations
The [NISCC](http://www.niscc.gov.uk/) and the
[OUSPG](http://www.ee.oulu.fi/research/ouspg/) developed a test suite
for the H.323 protocol. This test suite has uncovered vulnerabilities in
several H.323 implementations with impacts ranging from
denial-of-service to arbitrary code execution.

In the FreeBSD Ports Collection, \`pwlib\' is directly affected. Other
applications such as \`asterisk\' and \`openh323\' incorporate \`pwlib\'
statically and so are also independently affected.


## Affected packages
 - **pwlib**
    - **Fixed:** 1.5.0_5
 - **asterisk**
    - **Fixed:** 0.7.2
 - **openh323**
    - **Fixed:** 1.12.0_4
