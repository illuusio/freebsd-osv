# imlib2 -- BMP decoder buffer overflow
Marcus Meissner discovered that imlib2\'s BMP decoder would crash when
loading the test BMP file created by Chris Evans for testing the
previous Qt vulnerability. There appears to be both a stack-based and a
heap-based buffer overflow that are believed to be exploitable for
arbitrary code execution.


## Affected packages
 - **imlib2**
    - **Fixed:** 1.1.1
