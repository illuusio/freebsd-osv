# rssh -- format string vulnerability
There is a format string bug in rssh that enables an attacker to execute
arbitrary code from an account configured to use rssh. On FreeBSD it is
only possible to compromise the rssh running account, not root.


## Affected packages
 - **rssh**
    - **Fixed:** 2.2.1
