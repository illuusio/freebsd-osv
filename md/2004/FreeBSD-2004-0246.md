# phpMyAdmin -- cross-site scripting vulnerabilities
Multiple cross-site scripting vulnerabilities, caused by improper input
parameter sanitizing, were detected in phpMyAdmin, which may enable an
attacker to do cross-site scripting attacks.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.6.0.2
