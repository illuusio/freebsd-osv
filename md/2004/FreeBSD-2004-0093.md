# ModSecurity for Apache 2.x remote off-by-one overflow
When the directive \"SecFilterScanPost\" is enabled, the Apache 2.x
version of ModSecurity is vulnerable to an off-by-one overflow


## Affected packages
 - **mod_security**
    - **Fixed:** 1.7.5
