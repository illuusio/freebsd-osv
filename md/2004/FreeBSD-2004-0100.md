# Remote Denial of Service of HTTP server and client
giFT-FastTrack is susceptible to a remote Denial of Service attack which
could allow a remote attacker to render HTTP services unusable.
According to the developers, no code execution is possible; however,
they recommend an immediate upgrade.


## Affected packages
 - **giFT-FastTrack**
    - **Fixed:** 0.8.7
