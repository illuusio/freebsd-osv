# ImageMagick -- EXIF parser buffer overflow
There exists a buffer overflow vulnerability in ImageMagick\'s EXIF
parsing code which may lead to execution of arbitrary code.


## Affected packages
 - **ImageMagick**
    - **Fixed:** 6.1.3
 - **ImageMagick-nox11**
    - **Fixed:** 6.1.3
