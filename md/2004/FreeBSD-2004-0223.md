# horde -- cross-site scripting vulnerability in help window
A Horde Team announcement states that a potential cross-site scripting
vulnerability in the help window has been corrected. The vulnerability
appears to involve the handling of the `topic` and `module` parameters
of the help window template.


## Affected packages
 - **horde**
    - **Fixed:** 2.2.7
 - **horde-devel**
    - **Fixed:** 2.2.7
