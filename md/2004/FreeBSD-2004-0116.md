# popfile file disclosure
John Graham-Cumming reports that certain configurations of POPFile may
allow the retrieval of any files with the extensions .gif, .png, .ico,
.css, as well as some files with the extension .html.


## Affected packages
 - **popfile**
    - **Fixed:** 0.21.1_2
