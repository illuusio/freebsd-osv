# tor -- remote DoS and loss of anonymity
Tor has various remote crashes which could lead to a remote
denial-of-service and be used to defeat clients anonymity. It is not
expected that these vulnerabilities are exploitable for arbitrary code
execution.


## Affected packages
 - **tor**
    - **Fixed:** 0.0.8.1
