# xerces-c2 -- Attribute blowup denial-of-service
Amit Klein reports about Xerces-C++:

> An attacker can craft a malicious XML document, which uses XML
> attributes in a way that inflicts a denial of service condition on the
> target machine (XML parser). The result of this attack is that the XML
> parser consumes all the CPU.


## Affected packages
 - **xerces-c2**
    - **Fixed:** 2.6.0
