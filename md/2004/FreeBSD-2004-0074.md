# proftpd IP address access control list breakage
Jindrich Makovicka reports a regression in proftpd\'s handling of IP
address access control lists (IP ACLs). Due to this regression, some IP
ACLs are treated as \`\`allow all\'\'.


## Affected packages
 - **proftpd**
    - **Introduced:** 1.2.9
    - **Fixed:** 1.2.10.r1
