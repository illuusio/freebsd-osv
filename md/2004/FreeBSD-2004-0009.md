# pine remotely exploitable vulnerabilities
Pine versions prior to 4.58 are affected by two vulnerabilities
discovered by iDEFENSE, a buffer overflow in mailview.c and an integer
overflow in strings.c. Both vulnerabilities can result in arbitrary code
execution when processing a malicious message.


## Affected packages
 - **pine**
    - **Fixed:** 4.58
 - **zh-pine**
    - **Fixed:** 4.58
 - **iw-pine**
    - **Fixed:** 4.58
