# rsync buffer overflow in server mode
When rsync is run in server mode, a buffer overflow could allow a remote
attacker to execute arbitrary code with the privileges of the rsync
server. Anonymous rsync servers are at the highest risk.


## Affected packages
 - **rsync**
    - **Fixed:** 2.5.7
