# Buffer overflow in INN control message handling
A small, fixed-size stack buffer is used to construct a filename based
on a received control message. This could result in a stack buffer
overflow.


## Affected packages
 - **inn**
    - **Introduced:** 2.4.*
    - **Fixed:** 2.4.1
 - **inn-stable**
    - **Fixed:** 20031022_1
