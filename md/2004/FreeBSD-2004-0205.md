# cacti -- SQL injection
Fernando Quintero reports that Cacti 0.8.5a suffers from a SQL injection
attack where an attacker can change the password for any Cacti user.
This attack is not possible if the PHP option magic_quotes_gpc is set to
On, which is the default for PHP in FreeBSD.


## Affected packages
 - **cacti**
    - **Fixed:** 0.8.6
