# ifmail -- unsafe set-user-ID application
Niels Heinen reports that ifmail allows one to specify a configuration
file. Since ifmail runs set-user-ID \`news\', this may allow a local
attacker to write to arbitrary files or execute arbitrary commands as
the \`news\' user.


## Affected packages
 - **ifmail**
    - **Fixed:** 2.15_4
