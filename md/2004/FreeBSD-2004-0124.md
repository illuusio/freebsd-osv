# courier-imap -- format string vulnerability in debug mode
An iDEFENSE security advisory describes a format string vulnerability
that could be exploited when Courier-IMAP is run in debug mode
(DEBUG_LOGIN set).


## Affected packages
 - **courier-imap**
    - **Fixed:** 3.0.7,1
