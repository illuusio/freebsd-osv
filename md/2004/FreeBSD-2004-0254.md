# unarj -- long filename buffer overflow
Ludwig Nussel has discovered a buffer overflow vulnerability in unarj\'s
handling of long filenames which could potentially lead to execution of
arbitrary code with the permissions of the user running unarj.


## Affected packages
 - **unarj**
    - **Fixed:** 2.43_2
