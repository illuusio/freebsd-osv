# clamav remote denial-of-service
clamav will exit when a programming assertion is not met. A malformed
uuencoded message can trigger this assertion, allowing an attacker to
trivially crash clamd or other components of clamav.


## Affected packages
 - **clamav**
    - **Fixed:** 0.65_7
