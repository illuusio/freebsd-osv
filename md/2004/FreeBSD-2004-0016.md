# icecast 1.x multiple vulnerabilities
icecast 1.3.11 and earlier contained numerous security vulnerabilities,
the most severe allowing a remote attacker to execute arbitrary code as
root.


## Affected packages
 - **icecast**
    - **Fixed:** 1.3.12
