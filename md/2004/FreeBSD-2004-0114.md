# libpng denial-of-service
Steve Grubb reports a buffer read overrun in libpng\'s png_format_buffer
function. A specially constructed PNG image processed by an application
using libpng may trigger the buffer read overrun and possibly result in
an application crash.


## Affected packages
 - **linux-png**
    - **Fixed:** 1.0.14_3
    - **Introduced:** 1.2
    - **Fixed:** 1.2.2
 - **png**
    - **Fixed:** 1.2.5_4
