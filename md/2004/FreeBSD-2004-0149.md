# mozilla -- POP client heap overflow
zen-parse discovered a heap buffer overflow in Mozilla\'s POP client
implementation. A malicious POP server could exploit this vulnerability
to cause Mozilla to execute arbitrary code.


## Affected packages
 - **mozilla**
    - **Fixed:** 1.7,2
 - **linux-mozilla**
    - **Fixed:** 1.7
 - **netscape7**
    - **Fixed:** 7.2
 - **thunderbird**
    - **Fixed:** 0.7
