# qt -- image loader vulnerabilities
Qt contains several vulnerabilities related to image loading, including
possible crashes when loading corrupt GIF, BMP, or JPEG images. Most
seriously, Chris Evans reports that the BMP crash is actually due to a
heap buffer overflow. It is believed that an attacker may be able to
construct a BMP image that could cause a Qt-using application to execute
arbitrary code when it is loaded.


## Affected packages
 - **qt**
    - **Fixed:** 3.3.3
