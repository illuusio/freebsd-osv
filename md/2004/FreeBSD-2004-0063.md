# OpenSSL ChangeCipherSpec denial-of-service vulnerability
A remote attacker could cause an application using OpenSSL to crash by
performing a specially crafted SSL/TLS handshake.


## Affected packages
 - **openssl**
    - **Fixed:** 0.9.7d
 - **openssl-beta**
    - **Fixed:** 0.9.7d
 - **FreeBSD**
    - **Introduced:** 4.0

    - **Introduced:** 5.2
