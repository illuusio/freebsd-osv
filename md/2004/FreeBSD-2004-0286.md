# a2ps -- insecure temporary file creation
A Secunia Security Advisory reports that Javier Fernández-Sanguino Peña
has found temporary file creation vulnerabilities in the fixps and
psmandup scripts which are part of a2ps. These vulnerabilities could
lead to an attacker overwriting arbitrary files with the credentials of
the user running the vulnerable scripts.


## Affected packages
 - **a2ps-a4**
    - **Fixed:** 4.13b_3
 - **a2ps-letter**
    - **Fixed:** 4.13b_3
 - **a2ps-letterdj**
    - **Fixed:** 4.13b_3
