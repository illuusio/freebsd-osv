# ezbounce remote format string vulnerability
A security hole exists that can be used to crash the proxy and execute
arbitrary code. An exploit is circulating that takes advantage of this,
and in some cases succeeds in obtaining a login shell on the machine.


## Affected packages
 - **ezbounce**
    - **Fixed:** 1.04.a_1
