# kdepim exploitable buffer overflow in VCF reader
A buffer overflow is present in some versions of the KDE personal
information manager (kdepim) which may be triggered when processing a
specially crafted VCF file.


## Affected packages
 - **kdepim**
    - **Introduced:** 3.1.0
    - **Fixed:** 3.1.4_1
