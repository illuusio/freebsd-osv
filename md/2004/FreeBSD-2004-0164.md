# mozilla -- automated file upload
A malicious web page can cause an automated file upload from the
victim\'s machine when viewed with Mozilla with Javascript enabled. This
is due to a bug permitting default values for type=\"file\" \<input\>
elements in certain situations.


## Affected packages
 - **mozilla**
    - **Introduced:** 1.7.a,2
    - **Fixed:** 1.7,2
    - **Introduced:** 1.8.a,2
    - **Fixed:** 1.8.a2,2
