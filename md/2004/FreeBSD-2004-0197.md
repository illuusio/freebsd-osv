# tiff -- multiple integer overflows
Dmitry V. Levin discovered numerous integer overflow bugs in libtiff.
Most of these bugs are related to memory management, and are believed to
be exploitable for arbitrary code execution.


## Affected packages
 - **tiff**
    - **Fixed:** 3.6.1_2
 - **linux-tiff**
    - **Fixed:** 3.6.1
 - **pdflib**
    - **Fixed:** 6.0.1

    - **Fixed:** 6.0.1
