# file disclosure in phpMyAdmin
Lack of proper input validation in phpMyAdmin may allow an attacker to
obtain the contents of any file on the target system that is readable by
the web server.


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.5.4
