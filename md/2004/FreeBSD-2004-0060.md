# xchat remotely exploitable buffer overflow (Socks5)
A straightforward stack buffer overflow exists in XChat\'s Socks5 proxy
support.

The XChat developers report that \`tsifra\' discovered this issue.

NOTE: XChat Socks5 support is disabled by support in the FreeBSD Ports
Collection.


## Affected packages
 - **xchat2**
    - **Introduced:** 1.8
    - **Fixed:** 2.0.8_2
