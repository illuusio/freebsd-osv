# freeradius -- denial-of-service vulnerability
A remote attacker may be able to crash the freeRADIUS Server due to
three independant bugs in the function which does improper checking
values while processing RADIUS attributes.


## Affected packages
 - **freeradius**
    - **Introduced:** 0.8.0
    - **Fixed:** 1.0.1
