# phpmyadmin -- command execution vulnerability
A phpMyAdmin security announcement reports:

> Command execution: since phpMyAdmin 2.6.0-pl2, on a system where
> external MIME-based transformations are activated, an attacker can put
> into MySQL data an offensive value that starts a shell command when
> browsed.

Enabling "PHP safe mode" on the server can be used as a workaround for
this vulnerability.


## Affected packages
 - **phpMyAdmin**
    - **Introduced:** 2.6.0.2
    - **Fixed:** 2.6.1.r1
