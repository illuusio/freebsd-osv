# cyrus-sasl -- potential buffer overflow in DIGEST-MD5 plugin
The Cyrus SASL DIGEST-MD5 plugin contains a potential buffer overflow
when quoting is required in the output.


## Affected packages
 - **cyrus-sasl**
    - **Introduced:** 2.*
    - **Fixed:** 2.1.19
