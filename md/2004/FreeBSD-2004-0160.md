# sudo -- sudoedit information disclosure
A new feature of sudo 1.6.8 called \"sudoedit\" (a safe editing
facility) may allow users to read files to which they normally have no
access.


## Affected packages
 - **sudo**
    - **Introduced:** 1.6.8
    - **Fixed:** 1.6.8
