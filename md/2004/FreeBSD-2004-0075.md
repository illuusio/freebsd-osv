# fsp buffer overflow and directory traversal vulnerabilities
The [Debian security team](http://www.debian.org/security) reported a
pair of vulnerabilities in fsp:

> A vulnerability was discovered in fsp, client utilities for File
> Service Protocol (FSP), whereby a remote user could both escape from
> the FSP root directory (CAN-2003-1022), and also overflow a
> fixed-length buffer to execute arbitrary code (CAN-2004-0011).


## Affected packages
 - **fspd**
    - **Fixed:** 2.8.1.19
