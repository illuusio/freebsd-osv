# viewcvs -- information leakage
The `hide_cvsroot` and `forbidden` configuration options are not
properly honored by viewcvs when exporting to a tar file which can lead
to information leakage.


## Affected packages
 - **viewcvs**
    - **Fixed:** 0.9.2_2
