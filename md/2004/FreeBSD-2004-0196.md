# CUPS -- local information disclosure
Certain methods of authenticated remote printing in CUPS can disclose
user names and passwords in the log files.

A workaround for this problem is to set more strict access permissions
on the CUPS logfiles.


## Affected packages
 - **cups-base**
    - **Fixed:** 1.1.22
