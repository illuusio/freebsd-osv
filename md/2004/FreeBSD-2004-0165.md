# mozilla -- NULL bytes in FTP URLs
When handling FTP URLs containing NULL bytes, Mozilla will interpret the
file content as HTML. This may allow unexpected execution of Javascript
when viewing plain text or other file types via FTP.


## Affected packages
 - **firefox**
    - **Fixed:** 0.9.3
 - **linux-mozilla**
    - **Fixed:** 1.7.2
 - **linux-mozilla-devel**
    - **Fixed:** 1.7.2
 - **mozilla**
