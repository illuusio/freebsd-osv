# xpdf -- integer overflow vulnerabilities
Chris Evans discovered several integer arithmetic overflows in the xpdf
2 and xpdf 3 code bases. The flaws have impacts ranging from
denial-of-service to arbitrary code execution.


## Affected packages
 - **gpdf**
    - **Fixed:** 1.1.22.0
 - **cups-base**
    - **Fixed:** 1.1.22.0
 - **xpdf**
    - **Fixed:** 3.00_4
 - **kdegraphics**
    - **Fixed:** 3.3.0_1
 - **koffice**
    - **Fixed:** 1.3.2_1,1
 - **teTeX-base**
    - **Fixed:** 2.0.2_4
