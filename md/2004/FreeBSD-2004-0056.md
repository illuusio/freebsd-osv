# phpBB IP address spoofing
The common.php script always trusts the \`X-Forwarded-For\' header in
the client\'s HTTP request. A remote user could forge this header in
order to bypass any IP address access control lists (ACLs).


## Affected packages
 - **phpbb**
    - **Fixed:** 2.0.8_2
