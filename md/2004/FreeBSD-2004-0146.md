# mpg123 buffer overflow
The mpg123 software version 0.59r contains a buffer overflow
vulnerability which may permit the execution of arbitrary code as the
owner of the mpg123 process.


## Affected packages
 - **mpg123**
    - **Fixed:** 0.59r
 - **mpg123-nas**
    - **Fixed:** 0.59r
 - **mpg123-esound**
    - **Fixed:** 0.59r
