# gaim -- heap overflow exploitable by malicious GroupWise server
Sean "infamous42md" reports that a malicious GroupWise messaging server
may be able to exploit a heap buffer overflow in gaim, leading to
arbitrary code execution.

 - **gaim**

 - **ko-gaim**
