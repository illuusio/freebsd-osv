# Arbitrary code execution via a format string vulnerability in jftpgw
The log functions in jftpgw may allow remotely authenticated user to
execute arbitrary code via the format string specifiers in certain
syslog messages.


## Affected packages
 - **jftpgw**
    - **Fixed:** 0.13.5
