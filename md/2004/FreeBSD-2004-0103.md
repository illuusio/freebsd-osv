# Remote code injection in phpMyAdmin
This vulnerability would allow remote user to inject PHP code to be
executed by eval() function. This vulnerability is only exploitable if
variable \$cfg\[\'LeftFrameLight\'\] is set to FALSE (in file
config.inc.php).


## Affected packages
 - **phpMyAdmin**
    - **Fixed:** 2.5.7.1
