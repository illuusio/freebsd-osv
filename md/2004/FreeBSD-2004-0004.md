# L2TP, ISAKMP, and RADIUS parsing vulnerabilities in tcpdump
Jonathan Heusser discovered vulnerabilities in tcpdump\'s L2TP, ISAKMP,
and RADIUS protocol handlers. These vulnerabilities may be used by an
attacker to crash a running \`tcpdump\' process.


## Affected packages
 - **tcpdump**
    - **Fixed:** 3.8.1_351
 - **FreeBSD**
    - **Fixed:** 5.2.1
