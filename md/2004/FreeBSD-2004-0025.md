# SQL injection vulnerability in phpnuke
Multiple researchers have discovered multiple SQL injection
vulnerabilities in some versions of Php-Nuke. These vulnerabilities may
lead to information disclosure, compromise of the Php-Nuke site, or
compromise of the back-end database.


## Affected packages
 - **phpnuke**
    - **Fixed:** 6.9
