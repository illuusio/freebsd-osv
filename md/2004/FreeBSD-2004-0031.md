# mpg123 vulnerabilities
In 2003, two vulnerabilities were discovered in mpg123 that could result
in remote code execution when using untrusted input or streaming from an
untrusted server.


## Affected packages
 - **mpg123**
    - **Fixed:** 0.59r_12
 - **mpg123-nas**
    - **Fixed:** 0.59r_12
 - **mpg123-esound**
    - **Fixed:** 0.59r_12
