# ChiTeX/ChiLaTeX unsafe set-user-id root
Niels Heinen reports that ChiTeX installs set-user-id root executables
that invoked system(3) without setting up the environment, trivially
allowing local root compromise.


## Affected packages
 - **zh-chitex**
    - **Introduced:** 0,1
