# rockdodger -- buffer overflows
The environment variable HOME is copied without regard to buffer size,
which can be used to gain elevated privilege if the binary is installed
setgid games, and a string is read from the high score file without
bounds check.

The port installs the binary without setgid, but with a world-writable
high score file.


## Affected packages
 - **rockdodger**
    - **Fixed:** 0.6_3
