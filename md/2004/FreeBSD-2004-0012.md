# pine remotely exploitable buffer overflow in newmail.c
Kris Kennaway reports a remotely exploitable buffer overflow in
newmail.c. Mike Silbersack submitted the fix.


## Affected packages
 - **zh-pine**
    - **Fixed:** 4.21
 - **iw-pine**
    - **Fixed:** 4.21
 - **pine**
    - **Fixed:** 4.21
 - **pine4-ssl**
    - **Fixed:** 4.21
