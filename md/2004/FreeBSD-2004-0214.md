# mod_ssl -- SSLCipherSuite bypass
It is possible for clients to use any cipher suite configured by the
virtual host, whether or not a certain cipher suite is selected for a
specific directory. This might result in clients using a weaker
encryption than originally configured.


## Affected packages
 - **ru-apache+mod_ssl**
    - **Fixed:** 1.3.31+30.20+2.8.18
 - **apache+mod_ssl**
    - **Fixed:** 1.3.31+2.8.20
 - **apache+mod_ssl+ipv6**
    - **Fixed:** 1.3.31+2.8.18_4
 - **apache2**
    - **Fixed:** 2.0.52_1
