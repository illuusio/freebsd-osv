# mail-notification -- denial-of-service vulnerability
Caused by an untested return value, and a resulting null pointer
dereference, it is possible for an attacker to crash the application.
However, the attacker must first hijack the connection between Mail
Notification and the Gmail or IMAP server.


## Affected packages
 - **mail-notification**
    - **Fixed:** 0.7.0
