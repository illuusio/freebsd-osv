# Overflow error in fetch
An integer overflow condition in
[fetch(1)](http://www.freebsd.org/cgi/man.cgi?query=fetch) in the
processing of HTTP headers can result in a buffer overflow.

A malicious server or CGI script can respond to an HTTP or HTTPS request
in such a manner as to cause arbitrary portions of the client\'s memory
to be overwritten, allowing for arbitrary code execution.
