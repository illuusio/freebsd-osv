# mod_dav -- lock related denial-of-service
A malicious user with DAV write privileges can trigger a null pointer
dereference in the Apache mod_dav module. This could cause the server to
become unavailable.


## Affected packages
 - **apache**
    - **Introduced:** 2.0
    - **Fixed:** 2.0.50_3
 - **mod_dav**
    - **Fixed:** 1.0.3_1
