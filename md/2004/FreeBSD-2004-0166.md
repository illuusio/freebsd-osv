# mozilla -- security icon spoofing
Under certain situations it is possible for the security icon which
Mozilla displays when connected to a site using SSL to be spoofed. This
could be used to make so-called \"phishing attacks\" more difficult to
detect.


## Affected packages
 - **firefox**
    - **Fixed:** 0.9
 - **linux-mozilla**
    - **Fixed:** 1.7
 - **linux-mozilla-devel**
    - **Fixed:** 1.7
 - **mozilla**
    - **Fixed:** 1.7,2
 - **mozilla-gtk1**
    - **Fixed:** 1.7
