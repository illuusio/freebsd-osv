# libxml -- remote buffer overflows
"infamous41md" reports that libxml contains multiple buffer overflows in
the URL parsing and DNS name resolving functions. These vulnerabilities
could lead to execution of arbitrary code.


## Affected packages
 - **libxml**
    - **Fixed:** 1.8.17_3
 - **libxml2**
    - **Fixed:** 2.6.15
