# seti@home remotely exploitable buffer overflow
The seti@home client contains a buffer overflow in the HTTP response
handler. A malicious, spoofed seti@home server can exploit this buffer
overflow to cause remote code execution on the client. Exploit programs
are widely available.


## Affected packages
 - **setiathome**
    - **Fixed:** 3.0.8
