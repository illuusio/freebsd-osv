# Cyrus IMSPd multiple vulnerabilities
The Cyrus team reported multiple vulnerabilities in older versions of
Cyrus IMSPd:

> These releases correct a recently discovered buffer overflow
> vulnerability, as well as clean up a significant amount of buffer
> handling throughout the code.


## Affected packages
 - **cyrus-imspd**
    - **Fixed:** 1.6a5
