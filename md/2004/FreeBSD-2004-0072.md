# exim buffer overflow when verify = header_syntax is used
A remote exploitable buffer overflow has been discovered in exim when
verify = header_syntax is used in the configuration file. This does not
affect the default configuration.


## Affected packages
 - **exim**
    - **Fixed:** 4.33+20_1
 - **exim-ldap2**
    - **Fixed:** 4.33+20_1
 - **exim-mysql**
    - **Fixed:** 4.33+20_1
 - **exim-postgresql**
    - **Fixed:** 4.33+20_1
