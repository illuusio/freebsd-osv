# samba -- Unauthenticated domain takeover via netlogon
The Samba Team reports:

> An unauthenticated attacker on the network can gain administrator
> access by exploiting a netlogon protocol flaw.


## Affected packages
 - **samba410**
    - **Fixed:** 4.10.18
 - **samba411**
    - **Fixed:** 4.11.13
 - **samba412**
    - **Fixed:** 4.12.7
