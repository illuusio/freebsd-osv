# xorg-server -- Multiple input validation failures in X server XKB extension
The X.org project reports:

> These issues can lead to privileges elevations for authorized clients
> on systems where the X server is running privileged.
>
> Insufficient checks on the lengths of the XkbSetMap request can lead
> to out of bounds memory accesses in the X server.
>
> Insufficient checks on input of the XkbSetDeviceInfo request can lead
> to a buffer overflow on the head in the X server.
