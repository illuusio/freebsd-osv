# LibreOffice Security Advisory
LibreOffice reports:

> Two flaws were found in LibreOffice:
>
> - CVE-2020-12802: remote graphics contained in docx format retrieved
>   in \'stealth mode\'
> - CVE-2020-12803: XForms submissions could overwrite local files


## Affected packages
 - **libreoffice**
    - **Fixed:** 6.4.4
