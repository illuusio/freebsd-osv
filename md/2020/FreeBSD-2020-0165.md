# drupal -- Multiple Vulnerabilities
Drupal Security Team reports:

> The Drupal core Form API does not properly handle certain form input
> from cross-site requests, which can lead to other vulnerabilities.


## Affected packages
 - **drupal7**
    - **Fixed:** 7.72
