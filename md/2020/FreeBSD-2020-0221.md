# Icinga Web 2 -- directory traversal vulnerability
Icinga development team reports:

> CVE-2020-24368
>
> Icinga Icinga Web2 2.0.0 through 2.6.4, 2.7.4 and 2.8.2 has a
> Directory Traversal vulnerability which allows an attacker to access
> arbitrary files that are readable by the process running Icinga Web 2.
> This issue is fixed in Icinga Web 2 in v2.6.4, v2.7.4 and v2.8.2.


## Affected packages
 - **icingaweb2**
    - **Fixed:** 2.8.1
