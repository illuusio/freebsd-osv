# glpi -- bypass of the open redirect protection
MITRE Corporation reports:

> In GLPI before version 9.4.6, there is a vulnerability that allows
> bypassing the open redirect protection based which is based on a
> regexp. This is fixed in version 9.4.6.


## Affected packages
 - **glpi**
    - **Fixed:** 9.4.6,1
