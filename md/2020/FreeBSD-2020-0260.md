# tt-rss -- multiple vulnerabilities
tt-rss project reports:

> The cached_url feature mishandles JavaScript inside an SVG document.
>
> imgproxy in plugins/af_proxy_http/init.php mishandles
> \$\_REQUEST\[\"url\"\] in an error message.
>
> It does not validate all URLs before requesting them.

> Allows remote attackers to execute arbitrary PHP code via a crafted
> plural forms header.


## Affected packages
 - **tt-rss**
    - **Fixed:** g20200919
