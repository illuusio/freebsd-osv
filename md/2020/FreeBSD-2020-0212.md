# snmptt -- malicious shell code
Snmptt reports:

> Fixed a security issue with EXEC / PREXEC / unknown_trap_exec that
> could allow malicious shell code to be executed.
>
> Fixed a bug with EXEC / PREXEC / unknown_trap_exec that caused
> commands to be run as root instead of the user defined in daemon_uid.


## Affected packages
 - **snmptt**
    - **Fixed:** 1.4.2
