# freetype2 -- heap buffer overlfow
The freetype project reports:

> A heap buffer overflow has been found in the handling of embedded PNG
> bitmaps, introduced in FreeType version 2.6.


## Affected packages
 - **freetype2**
    - **Fixed:** 2.10.4
