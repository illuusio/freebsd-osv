# MySQL -- Multiple vulerabilities
Oracle reports:

> This Critical Patch Update contains 17 new security fixes for Oracle
> MySQL. 5 of these vulnerabilities may be remotely exploitable without
> authentication, i.e., may be exploited over a network without
> requiring user credentials.


## Affected packages
 - **mysql56-server**
    - **Fixed:** 5.6.47
 - **mysql57-server**
    - **Fixed:** 5.7.29
 - **mysql80-server**
    - **Fixed:** 8.0.19
 - **percona55-server**
    - **Fixed:** 5.5.67
 - **percona56-server**
    - **Fixed:** 5.6.47
 - **percona57-server**
    - **Fixed:** 5.7.29
