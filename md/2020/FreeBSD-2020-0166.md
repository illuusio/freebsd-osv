# powerdns-recursor -- access restriction bypass
PowerDNS Team reports:

> CVE-2020-14196: An issue has been found in PowerDNS Recursor where the
> ACL applied to the internal web server via webserver-allow-from is not
> properly enforced, allowing a remote attacker to send HTTP queries to
> the internal web server, bypassing the restriction. In the default
> configuration the API webserver is not enabled. Only installations
> using a non-default value for webserver and webserver-address are
> affected.


## Affected packages
 - **powerdns-recursor**
    - **Introduced:** 4.3.0
