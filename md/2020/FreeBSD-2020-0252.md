# Nextcloud -- Password share by mail not hashed
The Nextcloud project reports:

> NC-SA-2020-026 (low): Password of share by mail is not hashed when
> given on the create share call\
> A logic error in Nextcloud Server 19.0.0 caused a plaintext storage of
> the share password when it was given on the initial create API call.


## Affected packages
 - **nextcloud**
    - **Fixed:** 19.0.1
