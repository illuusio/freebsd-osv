# gitea -- multiple vulnerabilities
The Gitea Team reports:

> - Hide credentials when submitting migration
> - Never allow an empty password to validate
> - Prevent redirect to Host
> - Hide public repos owned by private orgs


## Affected packages
 - **gitea**
    - **Fixed:** 1.10.3
