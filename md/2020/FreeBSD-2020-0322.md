# glpi -- Any CalDAV calendars is read-only for every authenticated user
MITRE Corporation reports:

> In GLPI before version 9.5.3, any authenticated user has read-only
> permissions to the planning of every other user, even admin ones. This
> issue is fixed in version 9.5.3. As a workaround, one can remove the
> caldav.php file to block access to CalDAV server.


## Affected packages
 - **glpi**
    - **Introduced:** 9.5.0,1
    - **Fixed:** 9.5.3,1
