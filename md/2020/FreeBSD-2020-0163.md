# MongoDB -- Ensure RoleGraph can serialize authentication restrictions to BSON
reports:

> Improper serialization of MongoDB Server\'s internal authorization
> state permits a user with valid credentials to bypass IP source
> address protection mechanisms following administrative action.
>
> Credit\
> Discovered by Tony Yesudas.
