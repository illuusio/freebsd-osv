# LibreSSL -- NULL pointer dereference
The LibreSSL project reports:

> Malformed ASN.1 in a certificate revocation list or a timestamp
> response token can lead to a NULL pointer dereference.


## Affected packages
 - **libressl**
    - **Introduced:** 3.2.0,1
    - **Fixed:** 3.2.3
    - **Fixed:** 3.1.5
 - **libressl-devel**
    - **Fixed:** 3.3.1
