# glpi -- Reflexive XSS in Dropdown menus
MITRE Corporation reports:

> In GLPI after 0.68.1 and before 9.4.6, multiple reflexive XSS occur in
> Dropdown endpoints due to an invalid Content-Type. This has been fixed
> in version 9.4.6.


## Affected packages
 - **glpi**
    - **Fixed:** 9.4.6,1
