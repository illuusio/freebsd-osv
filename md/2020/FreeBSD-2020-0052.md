# FreeBSD -- TCP IPv6 SYN cache kernel information disclosure
# Problem Description:

When a TCP server transmits or retransmits a TCP SYN-ACK segment over
IPv6, the Traffic Class field is not initialized. This also applies to
challenge ACK segments, which are sent in response to received RST
segments during the TCP connection setup phase.

# Impact:

For each TCP SYN-ACK (or challenge TCP-ACK) segment sent over IPv6, one
byte of kernel memory is transmitted over the network.


## Affected packages
 - **FreeBSD-kernel**
    - **Introduced:** 12.1
    - **Fixed:** 12.1_3
    - **Introduced:** 11.3
    - **Fixed:** 11.3_7
