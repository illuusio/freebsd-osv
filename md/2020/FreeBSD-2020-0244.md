# zeek -- Various vulnerabilities

> This release fixes the following security issue:
>
> - The AYIYA and GTPv1 parsing/decapsulation logic may leak memory \--
>   These leaks have potential for remote exploitation to cause Denial
>   of Service via resource exhaustion.
