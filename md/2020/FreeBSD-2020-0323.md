# glpi -- Insecure Direct Object Reference on ajax/comments.ph
MITRE Corporation reports:

> In GLPI before 9.5.3, ajax/comments.php has an Insecure Direct Object
> Reference (IDOR) vulnerability that allows an attacker to read data
> from any database table (e.g., glpi_tickets, glpi_users, etc.).


## Affected packages
 - **glpi**
    - **Fixed:** 9.5.3,1
