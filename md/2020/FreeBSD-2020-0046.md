# py-matrix-synapse -- users of single-sign-on are vulnerable to phishing
Matrix developers report:

> \[The 1.11.1\] release includes a security fix impacting installations
> using Single Sign-On (i.e. SAML2 or CAS) for authentication.
> Administrators of such installations are encouraged to upgrade as soon
> as possible.


## Affected packages
 - **py35-matrix-synapse**
    - **Fixed:** 1.11.1
 - **py36-matrix-synapse**
    - **Fixed:** 1.11.1
 - **py37-matrix-synapse**
    - **Fixed:** 1.11.1
