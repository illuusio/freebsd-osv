# consul -- Fix Consul Connect CA private key configuration
Hashicorp reports:

> Increase the permissions to read from the /connect/ca/configuration
> endpoint to operator:write. Previously Connect CA configuration,
> including the private key, set via this endpoint could be read back by
> an operator with operator:read privileges.


## Affected packages
 - **consul**
    - **Fixed:** 1.9.0
