# libX11 -- Doublefree in locale handlng code
The X.org project reports:

> There is an integer overflow and a double free vulnerability in the
> way LibX11 handles locales. The integer overflow is a necessary
> precursor to the double free.


## Affected packages
 - **libX11**
    - **Fixed:** 1.6.12,1
