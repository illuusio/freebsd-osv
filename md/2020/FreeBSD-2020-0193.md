# libsndfile -- out-of-bounds read memory access
RedHat reports:

> It was discovered the fix for CVE-2018-19758 was not complete and
> still allows a read beyond the limits of a buffer in
> wav_write_header() function in wav.c. A local attacker may use this
> flaw to make the application crash.


## Affected packages
 - **libsndfile**
    - **Fixed:** 1.0.29.p.20200620
