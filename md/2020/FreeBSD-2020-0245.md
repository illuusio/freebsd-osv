# Rails -- Potential XSS vulnerability
Ruby on Rails blog:

> Rails 5.2.4.4 and 6.0.3.3 have been released! These releases contain
> an important security fix, so please upgrade when you can.
>
> Both releases contain the following fix: \[CVE-2020-15169\] Potential
> XSS vulnerability in Action View


## Affected packages
 - **rubygem-actionview52**
    - **Fixed:** 5.2.4.4
 - **rubygem-actionview60**
    - **Fixed:** 6.0.3.3
