# glpi -- SQL injection for all helpdesk instances
MITRE Corporation reports:

> In GLPI before version 9.4.6, there is a SQL injection vulnerability
> for all helpdesk instances. Exploiting this vulnerability requires a
> technician account. This is fixed in version 9.4.6.


## Affected packages
 - **glpi**
    - **Fixed:** 9.4.6,1
