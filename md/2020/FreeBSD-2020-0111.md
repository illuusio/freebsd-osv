# FreeBSD -- Insufficient cryptodev MAC key length check
# Problem Description:

Requests to create cryptography sessions using a MAC did not validate
the user-supplied MAC key length. The cryptodev module allocates a
buffer whose size is this user-suppled length.

# Impact:

An unprivileged process can trigger a kernel panic.


## Affected packages
 - **FreeBSD-kernel**
    - **Introduced:** 12.1
    - **Fixed:** 12.1_5
