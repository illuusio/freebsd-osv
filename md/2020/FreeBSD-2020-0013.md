# pkg -- vulnerability in libfetch
A programming error allows an attacker who can specify a URL with a
username and/or password components to overflow libfetch(3) buffers.


## Affected packages
 - **pkg**
    - **Fixed:** 1.12.0_1
 - **pkg-devel**
    - **Fixed:** 1.12.99_1
