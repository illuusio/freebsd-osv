# glpi -- SQL injection for all usages of "Clone" feature
MITRE Corporation reports:

> In glpi before 9.5.1, there is a SQL injection for all usages of
> \"Clone\" feature. This has been fixed in 9.5.1.


## Affected packages
 - **glpi**
    - **Introduced:** 9.5.0,1
    - **Fixed:** 9.5.1,1
