# asterisk -- Remote crash in res_pjsip_diversion
The Asterisk project reports:

> AST-2020-003: A crash can occur in Asterisk when a SIP message is
> received that has a History-Info header, which contains a tel-uri.
>
> AST-2020-004: A crash can occur in Asterisk when a SIP 181 response is
> received that has a Diversion header, which contains a tel-uri.


## Affected packages
 - **asterisk13**
    - **Fixed:** 13.38.1
 - **asterisk16**
    - **Fixed:** 16.15.1
 - **asterisk18**
    - **Fixed:** 18.1.1
