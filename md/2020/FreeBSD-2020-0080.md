# webkit2-gtk3 -- Denial of service
The WebKitGTK project reports the following vulnerability.

> Processing maliciously crafted web content may lead to arbitrary code
> execution or application crash (denial of service). Description: A
> memory corruption issue (use-after-free) was addressed with improved
> memory handling.


## Affected packages
 - **webkit2-gtk3**
    - **Fixed:** 2.28.1
