# cacti -- multiple vulnerabilities
The Cacti developers reports:

> When guest users have access to realtime graphs, remote code could be
> executed (CVE-2020-8813).
>
> Lack of escaping on some pages can lead to XSS exposure
> (CVE-2020-7106).
>
> Remote Code Execution due to input validation failure in Performance
> Boost Debug Log (CVE-2020-7237).


## Affected packages
 - **cacti**
    - **Fixed:** 1.2.10
