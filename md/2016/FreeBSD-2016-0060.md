# curl -- Credentials not checked
The cURL project reports:

> libcurl will reuse NTLM-authenticated proxy connections without
> properly making sure that the connection was authenticated with the
> same credentials as set for this transfer.


## Affected packages
 - **curl**
    - **Introduced:** 7.10.0
    - **Fixed:** 7.47.0
