# wolfssl -- DDoS amplification in DTLS
Sebastian Ramacher identified an error in wolfSSL\'s implementation of
the server side of the DTLS handshake, which could be abused for DDoS
amplification or a DoS on the DTLS server itself.


## Affected packages
 - **wolfssl**
    - **Fixed:** 3.6.8
