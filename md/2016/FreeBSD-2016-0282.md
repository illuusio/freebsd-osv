# MySQL -- Multiple vulnerabilities
Oracle reports:

> The quarterly Critical Patch Update contains 22 new security fixes for
> Oracle MySQL 5.5.49, 5.6.30, 5.7.13 and earlier


## Affected packages
 - **mariadb55-server**
    - **Fixed:** 5.5.49
 - **mariadb100-server**
    - **Fixed:** 10.0.25
 - **mariadb101-server**
    - **Fixed:** 10.1.14
 - **mysql55-server**
    - **Fixed:** 5.5.49
 - **mysql56-server**
    - **Fixed:** 5.6.30
 - **mysql57-server**
    - **Fixed:** 5.7.12_1
 - **percona55-server**
    - **Fixed:** 5.5.49
 - **percona56-server**
    - **Fixed:** 5.6.30
