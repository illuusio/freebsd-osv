# libotr -- integer overflow
X41 D-Sec reports:

> A remote attacker may crash or execute arbitrary code in libotr by
> sending large OTR messages.


## Affected packages
 - **libotr**
    - **Fixed:** 4.1.1
 - **libotr3**
