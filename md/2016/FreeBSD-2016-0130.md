# exim -- local privillege escalation
The Exim development team reports:

> All installations having Exim set-uid root and using \'perl_startup\'
> are vulnerable to a local privilege escalation. Any user who can start
> an instance of Exim (and this is normally **any** user) can gain root
> privileges. If you do not use \'perl_startup\' you **should** be safe.


## Affected packages
 - **exim**
    - **Fixed:** 4.86.2
    - **Fixed:** 4.85.2
    - **Fixed:** 4.84.2
