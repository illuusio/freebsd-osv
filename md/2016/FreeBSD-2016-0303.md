# FreeBSD -- integer overflow in IP_MSFILTER
Problem Description:

An integer overflow in computing the size of a temporary buffer can
result in a buffer which is too small for the requested operation.

Impact:

An unprivileged process can read or write pages of memory which belong
to the kernel. These may lead to exposure of sensitive information or
allow privilege escalation.
