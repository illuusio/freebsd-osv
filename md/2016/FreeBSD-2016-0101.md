# xdelta3 -- buffer overflow vulnerability
Stepan Golosunov reports:

> Buffer overflow was found and fixed in xdelta3 binary diff tool that
> allows arbitrary code execution from input files at least on some
> systems.


## Affected packages
 - **xdelta3**
    - **Fixed:** 3.0.9,1
