# Rails 4 -- Possible XSS Vulnerability in Action View
Ruby Security team reports:

> There is a possible XSS vulnerability in Action View. Text declared as
> \"HTML safe\" will not have quotes escaped when used as attribute
> values in tag helpers. This vulnerability has been assigned the CVE
> identifier CVE-2016-6316.


## Affected packages
 - **rubygem-actionview**
    - **Introduced:** 3.0.0,1
    - **Fixed:** 4.2.7.1
