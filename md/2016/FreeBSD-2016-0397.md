# ImageMagick -- multiple vulnerabilities
Debian reports:

> Various memory handling problems and cases of missing or incomplete
> input sanitizing may result in denial of service or the execution of
> arbitrary code if malformed SIXEL, PDB, MAP, SGI, TIFF and CALS files
> are processed.


## Affected packages
 - **ImageMagick**
    - **Fixed:** 6.9.5.10,1
 - **ImageMagick-nox11**
    - **Fixed:** 6.9.5.10,1
