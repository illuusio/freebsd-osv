> - CVE-2015-5346 Apache Tomcat Session fixation
>
> - CVE-2015-5351 Apache Tomcat CSRF token leak
>
> - CVE-2016-0763 Apache Tomcat Security Manager Bypass


## Affected packages
 - **tomcat7**
    - **Fixed:** 7.0.68
 - **tomcat8**
    - **Fixed:** 8.0.30
