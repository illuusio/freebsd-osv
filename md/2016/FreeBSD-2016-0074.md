# nghttp2 -- use after free
nghttp2 reports:

> This release fixes heap-use-after-free bug in idle stream handling
> code. We strongly recommend to upgrade the older installation to this
> latest version as soon as possible.


## Affected packages
 - **nghttp2**
    - **Fixed:** 1.6.0
