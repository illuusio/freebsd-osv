# libgd -- integer overflow which could lead to heap buffer overflow
LibGD reports:

> An integer overflow issue was found in function gdImageWebpCtx of file
> gd_webp.c which could lead to heap buffer overflow.


## Affected packages
 - **gd**
    - **Fixed:** 2.2.3
 - **php70-gd**
    - **Fixed:** 7.0.11
 - **php56-gd**
    - **Fixed:** 5.6.26
