# webkit -- UI spoof
webkit reports:

> The ScrollView::paint function in platform/scroll/ScrollView.cpp in
> Blink, as used in Google Chrome before 35.0.1916.114, allows remote
> attackers to spoof the UI by extending scrollbar painting into the
> parent frame.


## Affected packages
 - **webkit-gtk2**
    - **Fixed:** 2.4.9_1
 - **webkit-gtk3**
    - **Fixed:** 2.4.9_1
