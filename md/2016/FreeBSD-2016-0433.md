# jenkins -- Remote code execution vulnerability in remoting module
Jenkins Security Advisory:

> An unauthenticated remote code execution vulnerability allowed
> attackers to transfer a serialized Java object to the Jenkins CLI,
> making Jenkins connect to an attacker-controlled LDAP server, which in
> turn can send a serialized payload leading to code execution,
> bypassing existing protection mechanisms.


## Affected packages
 - **jenkins**
    - **Fixed:** 2.31
 - **jenkins-lts**
    - **Fixed:** 2.19.2
