# typo3 -- multiple vulnerabilities
TYPO3 Security Team reports:

> It has been discovered that TYPO3 CMS is susceptible to Cross-Site
> Scripting and Cross-Site Flashing.


## Affected packages
 - **typo3**
    - **Fixed:** 7.6.1
 - **typo3-lts**
    - **Fixed:** 6.2.16
