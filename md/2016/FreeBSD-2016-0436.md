# Remote-Code-Execution vulnerability in mysql and its variants CVE 2016-6662
LegalHackers\' reports:

> RCE Bugs discovered in MySQL and its variants like MariaDB. It works
> by manipulating my.cnf files and using \--malloc-lib. The bug seems
> fixed in MySQL 5.7.15 by Oracle


## Affected packages
 - **mysql57-client**
    - **Fixed:** 5.7.15
 - **mysql57-server**
    - **Fixed:** 5.7.15
 - **mysql56-client**
    - **Fixed:** 5.6.33
 - **mysql56-server**
    - **Fixed:** 5.6.33
 - **mysql55-client**
    - **Fixed:** 5.5.52
 - **mysql55-server**
    - **Fixed:** 5.5.52
