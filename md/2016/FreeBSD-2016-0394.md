# openoffice -- information disclosure vulnerability
Apache reports:

> The exposure exploits the way OLE previews are generated to embed
> arbitrary file data into a specially crafted document when it is
> opened. Data exposure is possible if the updated document is
> distributed to other parties.


## Affected packages
 - **apache-openoffice**
    - **Fixed:** 4.1.1
 - **apache-openoffice-devel**
    - **Fixed:** 4.1.1
