# FreeBSD -- Multiple libarchive vulnerabilities
# Problem Description:

Flaws in libarchive\'s handling of symlinks and hard links allow
overwriting files outside the extraction directory, or permission
changes to a directory outside the extraction directory.

# Impact:

An attacker who can control freebsd-update\'s or portsnap\'s input to
tar(1) can change file content or permissions on files outside of the
update tool\'s working sandbox.
