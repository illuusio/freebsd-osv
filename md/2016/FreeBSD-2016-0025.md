# tiff -- out-of-bounds read in CIE Lab image format
zzf of Alibaba discovered an out-of-bounds vulnerability in the code
processing the LogLUV and CIE Lab image format files. An attacker could
create a specially-crafted TIFF file that could cause libtiff to crash.


## Affected packages
 - **tiff**
    - **Fixed:** 4.0.6_1
 - **linux-c6-tiff**
    - **Fixed:** 3.9.4_2
 - **linux-f10-tiff**
