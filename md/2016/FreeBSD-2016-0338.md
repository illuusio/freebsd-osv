# FreeBSD -- rpcbind(8) remote denial of service [REVISED]
# Problem Description:

In rpcbind(8), netbuf structures are copied directly, which would result
in two netbuf structures that reference to one shared address buffer.
When one of the two netbuf structures is freed, access to the other
netbuf structure would result in an undefined result that may crash the
rpcbind(8) daemon.

# Impact:

A remote attacker who can send specifically crafted packets to the
rpcbind(8) daemon can cause it to crash, resulting in a denial of
service condition.
