# ricochet -- information disclosure
special reports:

> By sending a nickname with some HTML tags in a contact request, an
> attacker could cause Ricochet to make network requests without Tor
> after the request is accepted, which would reveal the user\'s IP
> address.


## Affected packages
 - **ricochet**
    - **Fixed:** 1.1.2
