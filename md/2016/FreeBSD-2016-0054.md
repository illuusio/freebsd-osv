# sudo -- potential privilege escalation via symlink misconfiguration
MITRE reports:

> sudoedit in Sudo before 1.8.15 allows local users to gain privileges
> via a symlink attack on a file whose full path is defined using
> multiple wildcards in /etc/sudoers, as demonstrated by
> \"/home/\*/\*/file.txt.\"


## Affected packages
 - **sudo**
    - **Fixed:** 1.8.15
