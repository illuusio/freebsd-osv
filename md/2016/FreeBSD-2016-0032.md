# ntp -- denial of service vulnerability
Network Time Foundation reports:

> NTF\'s NTP Project has been notified of the following 1
> medium-severity vulnerability that is fixed in ntp-4.2.8p5, released
> on Thursday, 7 January 2016:
>
> NtpBug2956: Small-step/Big-step CVE-2015-5300


## Affected packages
 - **ntp**
    - **Fixed:** 4.2.8p5
 - **ntp-devel**
    - **Fixed:** 4.3.78
 - **FreeBSD**
    - **Introduced:** 10.2
    - **Fixed:** 10.2_9
    - **Introduced:** 10.1
    - **Fixed:** 10.1_26
    - **Introduced:** 9.3
    - **Fixed:** 9.3_33
