# libreoffice -- use-after-free vulnerability
Talos reports:

> An exploitable Use After Free vulnerability exists in the RTF parser
> LibreOffice. A specially crafted file can cause a use after free
> resulting in a possible arbitrary code execution. To exploit the
> vulnerability a malicious file needs to be opened by the user via
> vulnerable application.


## Affected packages
 - **libreoffice**
    - **Fixed:** 5.1.4
