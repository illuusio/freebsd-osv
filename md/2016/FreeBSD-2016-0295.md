# gd -- multiple vulnerabilities
Pierre Joye reports:

> - fix php bug 72339, Integer Overflow in \_gd2GetHeader
>   (CVE-2016-5766)
>
> - gd: Buffer over-read issue when parsing crafted TGA file
>   (CVE-2016-6132)
>
> - Integer overflow error within \_gdContributionsAlloc()
>   (CVE-2016-6207)
>
> - fix php bug 72494, invalid color index not handled, can lead to
>   crash ( CVE-2016-6128)


## Affected packages
 - **gd**
    - **Fixed:** 2.2.3,1
