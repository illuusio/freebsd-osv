# cacti -- multiple vulnerabilities
The Cacti Group, Inc. reports:

> Changelog
>
> - bug:0002652: CVE-2015-8604: SQL injection in graphs_new.php
> - bug:0002655: CVE-2015-8377: SQL injection vulnerability in the
>   host_new_graphs_save function in graphs_new.php
> - bug:0002656: Authentication using web authentication as a user not
>   in the cacti database allows complete access


## Affected packages
 - **cacti**
    - **Fixed:** 0.8.8g
