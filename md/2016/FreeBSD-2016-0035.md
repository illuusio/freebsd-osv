# pygments -- shell injection vulnerability
NVD reports:

> The FontManager.\_get_nix_font_path function in formatters/img.py in
> Pygments 1.2.2 through 2.0.2 allows remote attackers to execute
> arbitrary commands via shell metacharacters in a font name.


## Affected packages
 - **py27-pygments**
    - **Fixed:** 2.0.2_1
 - **py32-pygments**
    - **Fixed:** 2.0.2_1
 - **py33-pygments**
    - **Fixed:** 2.0.2_1
 - **py34-pygments**
    - **Fixed:** 2.0.2_1
 - **py35-pygments**
    - **Fixed:** 2.0.2_1
