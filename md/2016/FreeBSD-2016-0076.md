# nginx -- multiple vulnerabilities
Maxim Dounin reports:

> Several problems in nginx resolver were identified, which might allow
> an attacker to cause worker process crash, or might have potential
> other impact if the \"resolver\" directive is used in a configuration
> file.


## Affected packages
 - **nginx**
    - **Fixed:** 1.8.1,2
 - **nginx-devel**
    - **Fixed:** 1.9.10
