# rails -- multiple vulnerabilities
Ruby on Rails blog:

> Rails 5.0.0.beta1.1, 4.2.5.1, 4.1.14.1, and 3.2.22.1 have been
> released! These contain important security fixes, and it is
> recommended that users upgrade as soon as possible.


## Affected packages
 - **rubygem-actionpack**
    - **Fixed:** 3.2.22.1
 - **rubygem-actionpack4**
    - **Fixed:** 4.2.5.1
 - **rubygem-actionview**
    - **Fixed:** 4.2.5.1
 - **rubygem-activemodel4**
    - **Fixed:** 4.2.5.1
 - **rubygem-activerecord**
    - **Fixed:** 3.2.22.1
 - **rubygem-activerecord4**
    - **Fixed:** 4.2.5.1
 - **rubygem-rails**
    - **Fixed:** 3.2.22.1
 - **rubygem-rails-html-sanitizer**
    - **Fixed:** 1.0.3
 - **rubygem-rails4**
    - **Fixed:** 4.2.5.1
