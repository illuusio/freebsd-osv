# privoxy -- multiple vulnerabilities
Privoxy Developers reports:

> Prevent invalid reads in case of corrupt chunk-encoded content.
> CVE-2016-1982. Bug discovered with afl-fuzz and AddressSanitizer.
>
> Remove empty Host headers in client requests. Previously they would
> result in invalid reads. CVE-2016-1983. Bug discovered with afl-fuzz
> and AddressSanitizer.


## Affected packages
 - **privoxy**
    - **Fixed:** 3.0.24
