# file-roller -- path traversal vulnerability
reports:

> File Roller 3.5.4 through 3.20.2 was affected by a path traversal bug
> that could result in deleted files if a user were tricked into opening
> a malicious archive.


## Affected packages
 - **file-roller**
    - **Introduced:** 3.5.4,1
    - **Fixed:** 3.20.2,1
