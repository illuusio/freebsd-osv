# kdelibs -- directory traversal vulnerability
David Faure reports:

> A maliciously crafted archive (.zip or .tar.bz2) with \"../\" in the
> file paths could be offered for download via the KNewStuff framework
> (e.g. on www.kde-look.org), and upon extraction would install files
> anywhere in the user\'s home directory.


## Affected packages
 - **kdelibs**
    - **Fixed:** 4.14.10_7
