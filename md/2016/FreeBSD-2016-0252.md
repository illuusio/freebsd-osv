# Python -- HTTP Header Injection in Python urllib
Guido Vranken reports:

> HTTP header injection in urrlib2/urllib/httplib/http.client with
> newlines in header values, where newlines have a semantic consequence
> of denoting the start of an additional header line.


## Affected packages
 - **python27**
    - **Fixed:** 2.7.10
 - **python33**
 - **python34**
    - **Fixed:** 3.4.4
 - **python35**
    - **Fixed:** 3.5.0
