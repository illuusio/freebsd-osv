# FreeBSD -- bsnmpd remote denial of service vulnerability
Problem Description:

The bsnmpd(8) daemon is prone to a stack-based buffer-overflow when it
has received a specifically crafted GETBULK PDU request.

Impact:

This issue could be exploited to execute arbitrary code in the context
of the service daemon, or crash the service daemon, causing a
denial-of-service.
