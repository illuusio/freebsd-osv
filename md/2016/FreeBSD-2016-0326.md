# FreeBSD -- SCTP SCTP_SS_VALUE kernel memory corruption and disclosure
# Problem Description:

Due to insufficient validation of the SCTP stream ID, which serves as an
array index, a local unprivileged attacker can read or write 16-bits of
kernel memory.

# Impact:

An unprivileged process can read or modify 16-bits of memory which
belongs to the kernel. This may lead to exposure of sensitive
information or allow privilege escalation.
