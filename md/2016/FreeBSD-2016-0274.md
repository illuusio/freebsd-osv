# p7zip -- heap overflow vulnerability
Cisco Talos reports:

> An exploitable heap overflow vulnerability exists in the
> NArchive::NHfs::CHandler::ExtractZlibFile method functionality of 7zip
> that can lead to arbitrary code execution.


## Affected packages
 - **p7zip**
    - **Fixed:** 15.14_1
