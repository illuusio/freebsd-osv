# hadoop2 -- unauthorized disclosure of data vulnerability
Arun Suresh reports:

> RPC traffic from clients, potentially including authentication
> credentials, may be intercepted by a malicious user with access to run
> tasks or containers on a cluster.


## Affected packages
 - **hadoop2**
    - **Introduced:** 2.6
    - **Fixed:** 2.7
