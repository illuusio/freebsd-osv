# gnutls -- file overwrite by setuid programs
gnutls.org reports:

> Setuid programs using GnuTLS 3.4.12 could potentially allow an
> attacker to overwrite and corrupt arbitrary files in the filesystem.


## Affected packages
 - **gnutls**
    - **Introduced:** 3.4.12
    - **Fixed:** 3.4.13
