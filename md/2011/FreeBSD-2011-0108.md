# mozilla -- multiple vulnerabilities
The Mozilla Project reports:

> MFSA 2011-29 Security issues addressed in Firefox 6
>
> MFSA 2011-28 Security issues addressed in Firefox 3.6.20


## Affected packages
 - **firefox**
    - **Introduced:** 3.6.*,1,1
    - **Fixed:** 3.6.20,1
    - **Introduced:** 5.0.*,1,1
    - **Fixed:** 6.0,1
 - **seamonkey**
    - **Fixed:** 2.3
 - **linux-firefox**
    - **Fixed:** 3.6.20,1
 - **thunderbird**
    - **Fixed:** 3.1.12
 - **linux-thunderbird**
    - **Fixed:** 3.1.12
