# isc-dhcp-server -- server halt upon processing certain packets
ISC reports:

> A pair of defects cause the server to halt upon processing certain
> packets. The patch is to properly discard or process those packets.


## Affected packages
 - **isc-dhcp31-server**
    - **Fixed:** 3.1.ESV_1,1
 - **isc-dhcp41-server**
    - **Fixed:** 4.1.e_2,2
 - **isc-dhcp42-server**
    - **Fixed:** 4.2.2
