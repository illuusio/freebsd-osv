# BIND -- Large RRSIG RRsets and Negative Caching DoS
ISC reports:

> A BIND 9 DNS server set up to be a caching resolver is vulnerable to a
> user querying a domain with very large resource record sets (RRSets)
> when trying to negatively cache a response. This can cause the BIND 9
> DNS server (named process) to crash.
