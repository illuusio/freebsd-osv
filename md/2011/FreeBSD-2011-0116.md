# apache -- Range header DoS vulnerability
Apache HTTP server project reports:

> A denial of service vulnerability has been found in the way the
> multiple overlapping ranges are handled by Apache HTTPD server.


## Affected packages
 - **apache**
    - **Introduced:** 2.*,1
    - **Fixed:** 2.2.20
 - **apache-event**
    - **Introduced:** 2.*,1
    - **Fixed:** 2.2.20
 - **apache-itk**
    - **Introduced:** 2.*,1
    - **Fixed:** 2.2.20
 - **apache-peruser**
    - **Introduced:** 2.*,1
    - **Fixed:** 2.2.20
 - **apache-worker**
    - **Introduced:** 2.*,1
    - **Fixed:** 2.2.20
