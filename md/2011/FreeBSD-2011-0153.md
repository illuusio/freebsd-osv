# hiawatha -- memory leak in PreventSQLi routine
Hugo Leisink reports via private mail to maintainer:

> The memory leak was introduced in version 7.6. It is in the routing
> that checks for SQL injections. So, if you have set PreventSQLi to
> \'no\', there is no problem.


## Affected packages
 - **hiawatha**
    - **Introduced:** 7.6
    - **Fixed:** 7.8.2
