# VLC -- Heap corruption in MP4 demultiplexer
VideoLAN project reports:

> When parsing some MP4 (MPEG-4 Part 14) files, insufficient buffer size
> might lead to corruption of the heap.


## Affected packages
 - **vlc**
    - **Introduced:** 1.0.0
    - **Fixed:** 1.1.9
