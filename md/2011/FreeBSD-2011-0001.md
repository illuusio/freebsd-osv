# php -- corruption of $GLOBALS and $this variables via extract() method
Off-by-one error in the sanity validator for the extract() method
allowed attackers to replace the values of \$GLOBALS and \$this when
mode EXTR_OVERWRITE was used.


## Affected packages
 - **php5**
    - **Fixed:** 5.3.4
 - **php52**
    - **Fixed:** 5.2.15
