# asterisk -- remote crash vulnerability in SIP channel driver
Asterisk project reports:

> A remote authenticated user can cause a crash with a malformed request
> due to an unitialized variable.


## Affected packages
 - **asterisk18**
    - **Introduced:** 1.8.*,1
    - **Fixed:** 1.8.7.1
 - **asterisk**
    - **Introduced:** 10.0.0.*,1
    - **Fixed:** 10.0.0.r1
