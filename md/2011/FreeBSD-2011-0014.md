# MoinMoin -- cross-site scripting vulnerabilities
The MoinMoin developers reports:

> Fix XSS in Despam action (CVE-2010-0828)

> Fix XSS issues
>
> - by escaping template name in messages
> - by fixing other places that had similar issues
