# libxml -- Stack consumption vulnerability
Stack consumption vulnerability allows context-dependent attackers to
cause a denial of service (application crash) via a large depth of
element declarations in a DTD.


## Affected packages
 - **libxml**
    - **Fixed:** 1.8.17_5
