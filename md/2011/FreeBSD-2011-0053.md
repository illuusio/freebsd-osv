# tinyproxy -- ACL lists ineffective when range is configured
When including a line to allow a network of IP addresses, the access to
tinyproxy 56 is actually allowed for all IP addresses.


## Affected packages
 - **tinyproxy**
    - **Fixed:** 1.8.2_2,1
