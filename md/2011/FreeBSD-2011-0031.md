# rubygem-mail -- Remote Arbitrary Shell Command Injection Vulnerability
Secunia reports:

> Input passed via an email from address is not properly sanitised in
> the \"deliver()\" function
> (lib/mail/network/delivery_methods/sendmail.rb) before being used as a
> command line argument. This can be exploited to inject arbitrary shell
> commands.


## Affected packages
 - **rubygem-mail**
    - **Fixed:** 2.2.15
