# django -- multiple vulnerabilities
Django project reports:

> Today the Django team is issuing multiple releases \-- Django 1.2.5
> and Django 1.1.4 \-- to remedy three security issues reported to us.
> All users of affected versions of Django are urged to upgrade
> immediately.
