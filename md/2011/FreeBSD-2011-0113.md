# PHP -- crypt() returns only the salt for MD5
PHP development team reports:

> If crypt() is executed with MD5 salts, the return value consists of
> the salt only. DES and BLOWFISH salts work as expected.


## Affected packages
 - **php5**
    - **Introduced:** 5.3.7
    - **Fixed:** 5.3.7_2
