# OTRS -- Several XSS attacks possible
OTRS Security Advisory reports:

> - Several XSS attacks possible: An attacker could trick a logged in
>   user to following a prepared URL inside of the OTRS system which
>   causes a page to be shown that possibly includes malicious
>   !JavaScript code because of incorrect escaping during the generation
>   of the HTML page.


## Affected packages
 - **otrs**
    - **Introduced:** 2.3.*,1
    - **Fixed:** 3.0.7
