# phpmyfaq -- Remote PHP Code Injection Vulnerability
The phpMyFAQ project reports:

> The phpMyFAQ Team has learned of a serious security issue that has
> been discovered in our bundled ImageManager library we use in phpMyFAQ
> 2.6 and 2.7. The bundled ImageManager library allows injection of
> arbitrary PHP code via POST requests.


## Affected packages
 - **phpmyfaq**
    - **Fixed:** 2.6.19
