# zabbix-frontend -- multiple XSS vulnerabilities
Martina Matari reports:

> These URLs (hostgroups.php, usergrps.php) are vulnerable to persistent
> XSS attacks due to improper sanitation of gname variable when creating
> user and host groups.


## Affected packages
 - **zabbix-frontend**
    - **Fixed:** 1.8.10,2
