# subversion -- remote HTTP DoS vulnerability
Subversion project reports:

> Subversion HTTP servers up to 1.5.9 (inclusive) or 1.6.15 (inclusive)
> are vulnerable to a remotely triggerable NULL-pointer dereference.


## Affected packages
 - **subversion**
    - **Introduced:** 1.6
    - **Fixed:** 1.6.15
    - **Introduced:** 1.5
    - **Fixed:** 1.6.9
 - **subversion-freebsd**
    - **Introduced:** 1.6
    - **Fixed:** 1.6.15
    - **Introduced:** 1.5
    - **Fixed:** 1.6.9
