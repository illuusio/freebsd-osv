# BIND -- Remote DoS against authoritative and recursive servers
ISC reports:

> A defect in the affected BIND 9 versions allows an attacker to
> remotely cause the \"named\" process to exit using a specially crafted
> packet.
>
> This defect affects both recursive and authoritative servers.


## Affected packages
 - **bind96**
    - **Fixed:** 9.6.3.1.ESV.R4.3
 - **bind97**
    - **Fixed:** 9.7.3.3
 - **bind98**
    - **Fixed:** 9.8.0.4
