# OTRS -- Vulnerabilities in OTRS-Core allows read access to any file on local file system
OTRS Security Advisory reports:

> - An attacker with valid session and admin permissions could get read
>   access to any file on the servers local operating system. For this
>   it would be needed minimum one installed OTRS package.


## Affected packages
 - **otrs**
    - **Introduced:** 2.1.*,1
    - **Fixed:** 3.0.10
