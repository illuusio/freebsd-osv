# postfix -- plaintext command injection with SMTP over TLS
Wietse Venema has discovered a software flaw that allows an attacker to
inject client commands into an SMTP session during the unprotected
plaintext SMTP protocol phase, such that the server will execute those
commands during the SMTP- over-TLS protocol phase when all communication
is supposed to be protected.
