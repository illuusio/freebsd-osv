# mupdf -- Remote System Access
Secunia reports:

> The vulnerability is caused due to an error within the \"closedctd()\"
> function in fitz/filt_dctd.c when processing PDF files containing
> certain malformed JPEG images. This can be exploited to cause a stack
> corruption by e.g. tricking a user into opening a specially crafted
> PDF file.


## Affected packages
 - **mupdf**
    - **Fixed:** 0.8
