# libxml -- Multiple use-after-free vulnerabilities
Multiple use-after-free vulnerabilities in libxml 1.8.17 that allow
context-dependent attackers to cause a denial of service (application
crash) via crafted (1) Notation or (2) Enumeration attribute types in an
XML file.


## Affected packages
 - **libxml**
    - **Fixed:** 1.8.17_5
