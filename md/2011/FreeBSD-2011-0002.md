# php -- open_basedir bypass
MITRE reports:

> fopen_wrappers.c in PHP 5.3.x through 5.3.3 might allow remote
> attackers to bypass open_basedir restrictions via vectors related to
> the length of a filename.


## Affected packages
 - **php5**
    - **Fixed:** 5.3.4
 - **php52**
    - **Fixed:** 5.2.15
