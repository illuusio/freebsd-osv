# phpmyadmin -- Local file inclusion
Jan Lieskovsky reports:

> Importing a specially-crafted XML file which contains an XML entity
> injection permits to retrieve a local file (limited by the privileges
> of the user running the web server).


## Affected packages
 - **phpMyAdmin**
    - **Introduced:** 3.4,1
    - **Fixed:** 3.4.7.1
    - **Fixed:** 3.3.10.5
