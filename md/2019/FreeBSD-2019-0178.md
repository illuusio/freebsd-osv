# NGINX -- Multiple vulnerabilities
NGINX Team reports:

> Several security issues were identified in nginx HTTP/2 implementation
> which might cause excessive memory consumption and CPU usage
> (CVE-2019-9511, CVE-2019-9513, CVE-2019-9516). The issues affect nginx
> compiled with the ngx_http_v2_module (not compiled by default) if the
> http2 option of the listen directive is used in a configuration file.


## Affected packages
 - **nginx**
    - **Fixed:** 1.16.1,2
 - **nginx-devel**
    - **Fixed:** 1.17.3
