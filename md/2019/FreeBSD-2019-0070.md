# Kubectl -- Potential directory traversal
Kubernetes.io reports:

> A security issue was discovered with the Kubernetes kubectl cp command
> that could enable a directory traversal replacing or deleting files on
> a user's workstation.


## Affected packages
 - **kubectl**
    - **Fixed:** 1.11.9
    - **Introduced:** 1.12.0
    - **Fixed:** 1.12.7
    - **Introduced:** 1.13.0
    - **Fixed:** 1.13.5
