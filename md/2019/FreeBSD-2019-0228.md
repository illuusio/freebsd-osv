# Xpdf -- Multiple Vulnerabilities
Xpdf 4.02 fixes two vulnerabilities. Both fixes have been backported to
3.04.

> An invalid memory access vulnerability in TextPage::findGaps() in Xpdf
> 4.01 through a crafted PDF document can cause a segfault.

> An out of bounds write exists in TextPage::findGaps() of Xpdf 4.01.01


## Affected packages
 - **xpdf**
    - **Fixed:** 4.02,1
 - **xpdf4**
    - **Fixed:** 4.02,1
 - **xpdf3**
    - **Fixed:** 3.04_11
