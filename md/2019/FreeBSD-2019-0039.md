# msmtp -- certificate-verification issue
msmtp developers report:

> In msmtp 1.8.2, when tls_trust_file has its default configuration,
> certificate-verification results are not properly checked.


## Affected packages
 - **msmtp**
    - **Introduced:** 1.8.1,1
    - **Fixed:** 1.8.3
