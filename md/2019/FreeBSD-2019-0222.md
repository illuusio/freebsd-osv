# Exim -- heap-based buffer overflow in string_vformat leading to RCE
Exim developers team report:

> There is a heap overflow in string_vformat().Using a EHLO message,
> remote code execution seems to be possible.


## Affected packages
 - **exim**
    - **Introduced:** 4.92
    - **Fixed:** 4.92.3
