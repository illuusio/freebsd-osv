# joomla3 -- vulnerabilitiesw
JSST reports:

> Inadequate escaping in mod_banners leads to a stored XSS
> vulnerability.

> Inadequate escaping in com_contact leads to a stored XSS vulnerability

> Inadequate checks at the Global Configuration Text Filter settings
> allowed a stored XSS.

> Inadequate checks at the Global Configuration helpurl settings allowed
> a stored XSS.


## Affected packages
 - **joomla3**
    - **Fixed:** 3.9.2
