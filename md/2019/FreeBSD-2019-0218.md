# ISC KEA -- Multiple vulnerabilities
Internet Systems Consortium, Inc. reports:

> A packet containing a malformed DUID can cause the kea-dhcp6 server to
> terminate (CVE-2019-6472) \[Medium\]
>
> An invalid hostname option can cause the kea-dhcp4 server to terminate
> (CVE-2019-6473) \[Medium\]
>
> An oversight when validating incoming client requests can lead to a
> situation where the Kea server\
> will exit when trying to restart (CVE-2019-6474) \[Medium\]


## Affected packages
 - **kea**
    - **Fixed:** 1.6.0
