# unbound -- parsing vulnerability
Unbound Security Advisories:

> Due to an error in parsing NOTIFY queries, it is possible for Unbound
> to continue processing malformed queries and may ultimately result in
> a pointer dereference in uninitialized memory. This results in a crash
> of the Unbound daemon.


## Affected packages
 - **unbound**
    - **Introduced:** 1.7.1
    - **Fixed:** 1.9.4
