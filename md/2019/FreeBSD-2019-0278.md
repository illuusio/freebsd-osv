# Gitlab -- Multiple Vulnerabilities
Gitlab reports:

> Path traversal with potential remote code execution
>
> Disclosure of private code via Elasticsearch integration
>
> Update Git dependency


## Affected packages
 - **gitlab-ce**
    - **Introduced:** 12.5.0
    - **Fixed:** 12.5.4
    - **Introduced:** 12.4.0
    - **Fixed:** 12.4.6
    - **Introduced:** 10.5.0
    - **Fixed:** 12.3.9
