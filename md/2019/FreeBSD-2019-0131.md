# znc -- privilege escalation
Mitre reports:

> Modules.cpp in ZNC before 1.7.4-rc1 allows remote authenticated
> non-admin users to escalate privileges and execute arbitrary code by
> loading a module with a crafted name.


## Affected packages
 - **znc**
    - **Fixed:** 1.7.4
