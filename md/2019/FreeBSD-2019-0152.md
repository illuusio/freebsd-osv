# drupal -- Drupal core - Access bypass
Drupal Security Team reports:

> In Drupal 8.7.4, when the experimental Workspaces module is enabled,
> an access bypass condition is created.
>
> This can be mitigated by disabling the Workspaces module. It does not
> affect any release other than Drupal 8.7.4.


## Affected packages
 - **drupal8**
    - **Fixed:** 8.7.5
