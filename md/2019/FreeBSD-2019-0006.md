# uriparser -- Out-of-bounds read
Upstream project reports:

> Out-of-bounds read in uriParse\*Ex\* for incomplete URIs with IPv6
> addresses with embedded IPv4 address, e.g. \"//\[::44.1\"; mitigated
> if passed parameter afterLast points to readable memory containing a
> \'\\0\' byte.


## Affected packages
 - **uriparser**
    - **Fixed:** 0.9.1
