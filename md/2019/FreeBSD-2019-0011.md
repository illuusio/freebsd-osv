# Helm -- client unpacking chart that contains malicious content
Helm security notice

> A specially crafted chart may be able to unpack content into locations
> on the filesystem outside of the chart\'s path, potentially
> overwriting existing files.


## Affected packages
 - **helm**
    - **Introduced:** 2.0.0
    - **Fixed:** 2.12.2
