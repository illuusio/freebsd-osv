# Gitlab -- Multiple Vulnerabilities
Gitlab reports:

> Insecure Authentication Methods Disabled for Grafana By Default
>
> Multiple Command-Line Flag Injection Vulnerabilities
>
> Insecure Cookie Handling on GitLab Pages


## Affected packages
 - **gitlab-ce**
    - **Introduced:** 12.1.0
    - **Fixed:** 12.1.6
    - **Introduced:** 12.0.0
    - **Fixed:** 12.0.6
    - **Introduced:** 10.0.0
    - **Fixed:** 11.11.8
