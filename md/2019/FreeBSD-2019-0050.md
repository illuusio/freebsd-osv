# rt -- XSS via jQuery
BestPractical reports:

> The version of jQuery used in RT 4.2 and 4.4 has a Cross-site
> Scripting (XSS) vulnerability when using cross-domain Ajax requests.
> This vulnerability is assigned
> [CVE-2015-9251](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-9251).
> RT does not use this jQuery feature so it is not directly vulnerable.
> jQuery version 1.12 no longer receives official updates, however a fix
> was posted with recommendations for applications to patch locally, so
> RT will follow this recommendation and ship with a patched version.


## Affected packages
 - **rt42**
    - **Introduced:** 4.2.0
    - **Fixed:** 4.2.16
 - **rt44**
    - **Introduced:** 4.4.0
    - **Fixed:** 4.4.4
