# jenkins -- multiple vulnerabilities
Jenkins Security Advisory:

> # Description
>
> ##### (High) SECURITY-868
>
> Administrators could persist access to Jenkins using crafted
> \'Remember me\' cookie
>
> ##### (Medium) SECURITY-901
>
> Deleting a user in an external security realm did not invalidate their
> session or \'Remember me\' cookie


## Affected packages
 - **jenkins**
    - **Fixed:** 2.160
 - **jenkins-lts**
    - **Fixed:** 2.150.2
