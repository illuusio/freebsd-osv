# FreeBSD -- Microarchitectural Data Sampling (MDS)
# Problem Description:

On some Intel processors utilizing speculative execution a local process
may be able to infer stale information from microarchitectural buffers
to obtain a memory disclosure.

# Impact:

An attacker may be able to read secret data from the kernel or from a
process when executing untrusted code (for example, in a web browser).


## Affected packages
 - **FreeBSD-kernel**
    - **Introduced:** 12.0
    - **Fixed:** 12.0_5
    - **Introduced:** 11.2
    - **Fixed:** 11.2_10
