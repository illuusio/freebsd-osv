# py-yaml -- arbitrary code execution
pyyaml reports:

> the PyYAML.load function could be easily exploited to call any Python
> function. That means it could call any system command using
> os.system()


## Affected packages
 - **py27-yaml**
    - **Fixed:** 4.1
 - **py35-yaml**
    - **Fixed:** 4.1
 - **py36-yaml**
    - **Fixed:** 4.1
 - **py37-yaml**
    - **Fixed:** 4.1
