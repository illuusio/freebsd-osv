# gitea -- information disclosure
The Gitea Team reports:

> When a comment in an issue or PR mentions a user using \@username, the
> mentioned user receives a mail notification even if they don\'t have
> permission to see the originating repository.


## Affected packages
 - **gitea**
    - **Fixed:** 1.9.5
