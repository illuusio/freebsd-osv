# CUPS -- multiple vulnerabilities
Apple reports:

> - CVE-2019-8696 and CVE-2019-8675: SNMP buffer overflows.
> - IPP buffer overflow.
> - Memory disclosure in the scheduler.
> - DoS issues in the scheduler.


## Affected packages
 - **cups**
    - **Fixed:** 2.2.12
