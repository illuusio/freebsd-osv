# FreeBSD -- iconv buffer overflow
# Problem Description:

With certain inputs, iconv may write beyond the end of the output
buffer.

# Impact:

Depending on the way in which iconv is used, an attacker may be able to
create a denial of service, provoke incorrect program behavior, or
induce a remote code execution. iconv is a libc library function and the
nature of possible attacks will depend on the way in which iconv is used
by applications or daemons.


## Affected packages
 - **FreeBSD**
    - **Introduced:** 12.0
    - **Fixed:** 12.0_7
    - **Introduced:** 11.2
    - **Fixed:** 11.2_11
