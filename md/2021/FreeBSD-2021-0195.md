# Prosody -- Remote Information Disclosure
A Prosody XMPP server advisory reports:

> It was discovered that Prosody allows any entity to access the list of
> admins, members, owners and banned entities of any federated XMPP
> group chat of which they know the address.


## Affected packages
 - **prosody**
    - **Fixed:** 0.11.10
