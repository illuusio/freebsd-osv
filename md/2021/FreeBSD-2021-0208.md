# OpenSSL -- multiple vulnerabilities
The OpenSSL project reports:

> SM2 Decryption Buffer Overflow (CVE-2021-3711: High)
>
> Read buffer overruns processing ASN.1 strings (CVE-2021-3712:
> Moderate)


## Affected packages
 - **openssl**
    - **Fixed:** 1.1.1l,1
 - **openssl-devel**
    - **Fixed:** 3.0.0.b3
 - **FreeBSD**
    - **Introduced:** 13.0
    - **Fixed:** 13.0_4
    - **Introduced:** 12.2
    - **Fixed:** 12.2_10
