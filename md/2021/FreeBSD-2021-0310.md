# minio -- User privilege escalation
minio developers report:

> AddUser() API endpoint was exposed to a legacy behavior. i.e it
> accepts a \"policy\" field
>
> This API is mainly used to create a user or update a user\'s password.
>
> However, a malicious client can hand-craft an HTTP API call that
> allows for updating Policy for a user and gaining higher privileges.


## Affected packages
 - **minio**
    - **Fixed:** 2021.12.27.07.23.18
