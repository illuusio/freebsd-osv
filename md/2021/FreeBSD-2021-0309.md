# Pillow -- Regular Expression Denial of Service (ReDoS)
GitHub Advisory Database reports:

> Uncontrolled Resource Consumption in pillow.
>
> The package pillow from 0 and before 8.3.2 are vulnerable to Regular
> Expression Denial of Service (ReDoS) via the getrgb function.
>
