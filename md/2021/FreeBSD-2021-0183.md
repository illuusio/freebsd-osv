# asterisk -- Remote crash when using IAX2 channel driver
The Asterisk project reports:

> If the IAX2 channel driver receives a packet that contains an
> unsupported media format it can cause a crash to occur in Asterisk.


## Affected packages
 - **asterisk13**
    - **Fixed:** 13.38.3
 - **asterisk16**
    - **Fixed:** 16.19.1
 - **asterisk18**
    - **Fixed:** 18.5.1
