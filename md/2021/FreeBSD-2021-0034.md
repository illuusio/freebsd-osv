# oauth2-proxy -- domain whitelist could be used as redirect
The oauth2-proxy Team reports:

> In OAuth2 Proxy before version 7.0.0, for users that use the whitelist
> domain feature, a domain that ended in a similar way to the intended
> domain could have been allowed as a redirect.


## Affected packages
 - **oauth2-proxy**
    - **Fixed:** 7.0.0
