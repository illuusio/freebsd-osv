# redis -- integer overflow
Redis development team reports:

> An integer overflow bug in Redis version 6.0 or newer can be exploited
> using the STRALGO LCS command to corrupt the heap and potentially
> result with remote code execution. This is a result of an incomplete
> fix by CVE-2021-29477.


## Affected packages
 - **redis**
    - **Introduced:** 6.0.0
    - **Fixed:** 6.0.14
 - **redis-devel**
    - **Introduced:** 6.2.0
    - **Fixed:** 6.2.4
