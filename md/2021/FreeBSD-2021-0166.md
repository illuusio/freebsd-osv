# PuppetDB -- SQL Injection
Puppet reports:

> Fixed an issue where someone with the ability to query PuppetDB could
> arbitrarily write, update, or delete data CVE-2021-27021 PDB-5138.


## Affected packages
 - **puppetdb6**
    - **Fixed:** 6.17.0
 - **puppetdb7**
    - **Fixed:** 7.4.1
