# NGINX -- 1-byte memory overwrite in resolver
NGINX team reports:

> 1-byte memory overwrite might occur during DNS server response
> processing if the \"resolver\" directive was used, allowing an
> attacker who is able to forge UDP packets from the DNS server to cause
> worker process crash or, potentially, arbitrary code execution.


## Affected packages
 - **nginx**
    - **Fixed:** 1.20.1,2
 - **nginx-devel**
    - **Fixed:** 1.21.0
