# samba -- negative idmap cache entries vulnerability
The Samba Team reports:

> - CVE-2021-20254: Negative idmap cache entries can cause incorrect
>   group entries in the Samba file server process token.


## Affected packages
 - **samba412**
    - **Fixed:** 4.12.15
 - **samba413**
    - **Fixed:** 4.13.8
 - **samba414**
    - **Fixed:** 4.14.4
