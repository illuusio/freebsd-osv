# RabbitMQ -- Denial of Service in AMQP1.0 plugin
Pivotal.io reports:

> All versions prior to 3.8.16 are prone to a denial of service
> vulnerability due to improper input validation in AMQP 1.0 client
> connection endpoint.


## Affected packages
 - **rabbitmq**
    - **Fixed:** 3.8.16
