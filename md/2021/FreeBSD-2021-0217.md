# py-matrix-synapse -- several vulnerabilities
Matrix developers report:

> This release patches two moderate severity issues which could reveal
> metadata about private rooms:
>
> - CVE-2021-39164: Enumerating a private room\'s list of members and
>   their display names.
> - CVE-2021-39163: Disclosing a private room\'s name, avatar, topic,
>   and number of members.


## Affected packages
 - **py36-matrix-synapse**
    - **Fixed:** 1.41.1
 - **py37-matrix-synapse**
    - **Fixed:** 1.41.1
 - **py38-matrix-synapse**
    - **Fixed:** 1.41.1
 - **py39-matrix-synapse**
    - **Fixed:** 1.41.1
 - **py310-matrix-synapse**
    - **Fixed:** 1.41.1
