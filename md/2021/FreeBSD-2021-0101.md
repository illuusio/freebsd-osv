# MySQL -- Multiple vulnerabilities
Oracle reports:

> This Critical Patch Update contains 49 new security patches for Oracle
> MySQL. 10 of these vulnerabilities may be remotely exploitable without
> authentication, i.e., may be exploited over a network without
> requiring user credentials.\
> The highest CVSS v3.1 Base Score of vulnerabilities affecting Oracle
> MySQL is 9.8.
>
> MariaDB is affected by CVE-2021-2166 and CVE-2021-2154 only


## Affected packages
 - **mariadb103-server**
    - **Fixed:** 10.3.29
 - **mariadb104-server**
    - **Fixed:** 10.4.19
 - **mariadb105-server**
    - **Fixed:** 10.5.10
 - **mysql56-server**
    - **Fixed:** 5.6.52
 - **mysql57-server**
    - **Fixed:** 5.7.34
 - **mysql80-server**
    - **Fixed:** 8.0.24
