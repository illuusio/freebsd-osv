# Grafana -- Incorrect Access Control
Grafana Labs reports:

> When the fine-grained access control beta feature is enabled and there
> is more than one organization in the Grafana instance, Grafana 8.0
> introduced a mechanism which allowed users with the Organization Admin
> role to list, add, remove, and update users' roles in other
> organizations in which they are not an admin.


## Affected packages
 - **grafana8**
    - **Introduced:** 8.0.0
    - **Fixed:** 8.2.4
 - **grafana**
    - **Introduced:** 8.0.0
    - **Fixed:** 8.2.4
