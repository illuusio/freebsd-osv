# dino -- Path traversal in Dino file transfers
Dino team reports:

> It was discovered that when a user receives and downloads a file in
> Dino, URI-encoded path separators in the file name will be decoded,
> allowing an attacker to traverse directories and create arbitrary
> files in the context of the user.


## Affected packages
 - **dino**
    - **Fixed:** 0.2.1
