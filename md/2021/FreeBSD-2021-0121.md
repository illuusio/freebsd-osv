# cyrus-imapd -- Remote authenticated users could bypass intended access restrictions on certain server annotations.
Cyrus IMAP 3.4.1 Release Notes states:

> Fixed CVE-2021-32056: Remote authenticated users could bypass intended
> access restrictions on certain server annotations. Additionally, a
> long-standing bug in replication did not allow server annotations to
> be replicated. Combining these two bugs, a remote authenticated user
> could stall replication, requiring administrator intervention.


## Affected packages
 - **cyrus-imapd34**
    - **Introduced:** 3.4.0
    - **Fixed:** 3.4.1
 - **cyrus-imapd32**
    - **Introduced:** 3.2.0
    - **Fixed:** 3.2.7
