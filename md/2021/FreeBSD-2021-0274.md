# mailman -- 2.1.37 fixes XSS via user options, and moderator offline brute-force vuln against list admin password
Mark Sapiro reports:

> A potential XSS attack via the user options page has been reported by
> Harsh Jaiswal. This is fixed. CVE-2021-43331 (LP: #1949401).
>
> A potential for for a list moderator to carry out an off-line brute
> force attack to obtain the list admin password has been reported by
> Andre Protas, Richard Cloke and Andy Nuttall of Apple. This is fixed.
> CVE-2021-43332 (LP: #1949403)


## Affected packages
 - **mailman**
    - **Fixed:** 2.1.37
