# samba -- Multiple Vulnerabilities
The Samba Team reports:

> - CVE-2020-27840: An anonymous attacker can crash the Samba AD DC LDAP
>   server by sending easily crafted DNs as part of a bind request. More
>   serious heap corruption is likely also possible.
> - CVE-2021-20277: User-controlled LDAP filter strings against the AD
>   DC LDAP server may crash the LDAP server.


## Affected packages
 - **samba411**
    - **Fixed:** 4.11.15
 - **samba412**
    - **Fixed:** 4.12.14
 - **samba413**
    - **Fixed:** 4.13.7
 - **samba414**
    - **Fixed:** 4.14.2
