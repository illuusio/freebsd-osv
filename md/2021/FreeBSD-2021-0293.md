# Grafana -- Directory Traversal
GitHub Security Labs reports:

> A vulnerability through which authenticated users could read out fully
> lowercase or fully uppercase `.md` files through directory traversal.
> Doing our own follow-up investigation we found a related vulnerability
> through which authenticated users could read out arbitrary `.csv`
> files through directory traversal. Thanks to our defense-in-depth
> approach, at no time has [Grafana Cloud](https://grafana.com/cloud)
> been vulnerable.
>
> **The vulnerable URL path is:** `/api/ds/query`


## Affected packages
 - **grafana**
    - **Introduced:** 8.0.0
    - **Fixed:** 8.3.2
 - **grafana8**
    - **Introduced:** 8.0.0
    - **Fixed:** 8.3.2
