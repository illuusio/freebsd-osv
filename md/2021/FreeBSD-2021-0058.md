# Gitlab -- Multiple vulnerabilities
Gitlab reports:

> JWT token leak via Workhorse
>
> Stored XSS in wiki pages
>
> Group Maintainers are able to use the Group CI/CD Variables API
>
> Insecure storage of GitLab session keys


## Affected packages
 - **gitlab-ce**
    - **Introduced:** 13.9.0
    - **Fixed:** 13.9.2
    - **Introduced:** 13.8.0
    - **Fixed:** 13.8.5
    - **Fixed:** 13.7.8
