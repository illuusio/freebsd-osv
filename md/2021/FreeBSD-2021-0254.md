# Ansible -- Ansible user credentials disclosure in ansible-connection module
Red Hat reports:

> A flaw was found in Ansible Engine\'s ansible-connection module, where
> sensitive information such as the Ansible user credentials is
> disclosed by default in the traceback error message. The highest
> threat from this vulnerability is to confidentiality.
