# syncthing -- crash due to malformed relay protocol message
syncthing developers report:

> syncthing can be caused to crash and exit if sent a malformed relay
> protocol message message with a negative length field.
>
> The relay server strelaysrv can be caused to crash and exit if sent a
> malformed relay protocol message with a negative length field.


## Affected packages
 - **syncthing**
    - **Fixed:** 1.15.0
