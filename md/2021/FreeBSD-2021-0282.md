# mailman < 2.1.38 -- CSRF vulnerability of list mod or member against list admin page
Mark Sapiro reports:

> A list moderator or list member can potentially carry out a CSRF
> attack by getting a list admin to visit a crafted web page.


## Affected packages
 - **mailman**
    - **Fixed:** 2.1.38
 - **mailman-exim4**
    - **Fixed:** 2.1.38
 - **mailman-exim4-with-htdig**
    - **Fixed:** 2.1.38
 - **mailman-postfix**
    - **Fixed:** 2.1.38
 - **mailman-postfix-with-htdig**
    - **Fixed:** 2.1.38
 - **mailman-with-htdig**
    - **Fixed:** 2.1.38
