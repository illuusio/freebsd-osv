# mediawiki -- multiple vulnerabilities
Mediawiki reports:

> (T292763. CVE-2021-44854) REST API incorrectly publicly caches
> autocomplete search results from private wikis.
>
> (T271037, CVE-2021-44856) Title blocked in AbuseFilter can be created
> via Special:ChangeContentModel.
>
> (T297322, CVE-2021-44857) Unauthorized users can use action=mcrundo to
> replace the content of arbitrary pages.
>
> (T297322, CVE-2021-44858) Unauthorized users can view contents of
> private wikis using various actions.
>
> (T297574, CVE-2021-45038) Unauthorized users can access private wiki
> contents using rollback action
>
> (T293589, CVE-2021-44855) Blind Stored XSS in VisualEditor media
> dialog.
>
> (T294686) Special:Nuke doesn\'t actually delete pages.


## Affected packages
 - **mediawiki135**
    - **Fixed:** 1.35.5
 - **mediawiki136**
    - **Fixed:** 1.36.3
 - **mediawiki137**
    - **Fixed:** 1.37.1
