# MySQL -- Multiple vulnerabilities
Oracle reports:

> This Critical Patch Update contains 34 new security patches for Oracle
> MySQL Server and 4 for MySQL Client.
>
> The highest CVSS v3.1 Base Score of vulnerabilities affecting Oracle
> MySQL is 6.8.


## Affected packages
 - **mysql56-client**
    - **Fixed:** 5.6.51
 - **mysql57-client**
    - **Fixed:** 5.7.33
 - **mysql80-client**
    - **Fixed:** 8.0.23
 - **mysql56-server**
    - **Fixed:** 5.6.51
 - **mysql57-server**
    - **Fixed:** 5.7.33
 - **mysql80-server**
    - **Fixed:** 8.0.23
