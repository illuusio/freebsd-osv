# asterisk -- pjproject/pjsip: crash when SSL socket destroyed during handshake
The Asterisk project reports:

> Depending on the timing, it\'s possible for Asterisk to crash when
> using a TLS connection if the underlying socket parent/listener gets
> destroyed during the handshake.


## Affected packages
 - **asterisk13**
    - **Fixed:** 13.38.3
 - **asterisk16**
    - **Fixed:** 16.19.1
 - **asterisk18**
    - **Fixed:** 18.5.1
