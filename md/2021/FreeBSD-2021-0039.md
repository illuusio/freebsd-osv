# Rails -- multiple vulnerabilities
Ruby on Rails blog:

> Rails version 5.2.4.5, 6.0.3.5 and 6.1.2.1 have been released! Those
> version are security releases and addresses two issues:
>
> CVE-2021-22880: Possible DoS Vulnerability in Active Record PostgreSQL
> adapter.
>
> CVE-2021-22881: Possible Open Redirect in Host Authorization
> Middleware.


## Affected packages
 - **rubygem-activerecord52**
    - **Fixed:** 5.2.4.5
 - **rubygem-actionpack60**
    - **Fixed:** 6.0.3.5
 - **rubygem-activerecord60**
    - **Fixed:** 6.0.3.5
 - **rubygem-actionpack61**
    - **Fixed:** 6.1.2.1
 - **rubygem-activerecord61**
    - **Fixed:** 6.1.2.1
