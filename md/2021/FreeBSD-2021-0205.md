# bouncycastle15 -- bcrypt password checking vulnerability
The Bouncy Castle team reports:

> The OpenBSDBCrypt.checkPassword utility method compared incorrect data
> when checking the password, allowing incorrect passwords to indicate
> they were matching with previously hashed ones that were different.


## Affected packages
 - **bouncycastle15**
    - **Introduced:** 1.65
    - **Fixed:** 1.67
