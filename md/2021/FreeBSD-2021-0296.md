# openhab -- log4j remote code injection
Openhab reports:

> Any openHAB instance that is publicly available or which consumes
> untrusted content from remote servers is potentially a target of this
> attack.


## Affected packages
 - **openhab2**
    - **Fixed:** 2.5.12
    - **Fixed:** 3.1.1
 - **openhab**
    - **Fixed:** 2.5.12
    - **Fixed:** 3.1.1
