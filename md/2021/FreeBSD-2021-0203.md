# binutils -- excessive debug section size can cause excessive memory consumption in bfd's dwarf2.c read_section()
Hao Wang reports:

> There\'s a flaw in the BFD library of binutils in versions before
> 2.36. An attacker who supplies a crafted file to an application linked
> with BFD, and using the DWARF functionality, could cause an impact to
> system availability by way of excessive memory consumption.


## Affected packages
 - **binutils**
    - **Fixed:** 2.33.1_5
