# nokogiri -- Security vulnerability
Nokogiri reports:

> In Nokogiri versions \<= 1.11.0.rc3, XML Schemas parsed by
> Nokogiri::XML::Schema were trusted by default, allowing external
> resources to be accessed over the network, potentially enabling XXE or
> SSRF attacks.


## Affected packages
 - **rubygem-nokogiri**
    - **Fixed:** 1.11.0.rc3
 - **rubygem-nokogiri18**
    - **Fixed:** 1.11.0.rc3
